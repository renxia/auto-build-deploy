Guide de Dépannage

Solutions aux problèmes courants et questions fréquemment posées sur Gemini CLI. Trouvez des corrections rapides et des conseils détaillés pour divers problèmes.

Commandes de Diagnostic Rapide

Commencez par ces commandes pour diagnostiquer les problèmes courants

gemini --version

Vérifier la version CLI et les informations de build

gemini config list

Afficher les paramètres de configuration actuels

gemini config reset

Réinitialiser la configuration aux valeurs par défaut

gemini logs

Voir les journaux d'erreurs récents et les informations de débogage

gemini doctor

Exécuter les vérifications de diagnostic et de santé

gemini cache clear

Effacer les données en cache et les fichiers temporaires

Problèmes Courants et Solutions

Parcourez par catégorie pour trouver des solutions à des problèmes spécifiques

Problèmes d'Installation

Problèmes lors de l'installation et de la configuration

npm install échoue avec des erreurs de permission

Utilisez npm avec le flag --global ou configurez npm pour utiliser un répertoire différent. Sur macOS/Linux, vous devrez peut-être utiliser sudo ou configurer npm correctement.

Solution:
npm install -g @google/generative-ai-cli

Commande introuvable après l'installation

Assurez-vous que le répertoire bin global npm est dans votre PATH. Vérifiez avec `npm config get prefix` et ajoutez le répertoire bin à votre PATH.

Solution:
export PATH=$PATH:$(npm config get prefix)/bin

Problèmes de compatibilité de version Node.js

Gemini CLI nécessite Node.js 16 ou supérieur. Mettez à jour Node.js en utilisant nvm ou téléchargez depuis nodejs.org.

Solution:
node --version  # Should be 16.0.0 or higher

Problèmes d'Authentification et d'API

Problèmes avec les clés API et l'authentification

Erreur de clé API invalide

Vérifiez que votre clé API est correcte et dispose des autorisations appropriées. Obtenez une nouvelle clé depuis Google AI Studio si nécessaire.

Solution:
gemini config set api-key YOUR_API_KEY

Quota API dépassé

Vérifiez vos limites d'utilisation dans Google AI Studio. Envisagez de mettre à niveau votre plan ou d'implémenter une limitation de débit.

Solution:
gemini config set rate-limit 10  # Requests per minute

Problèmes de connectivité réseau

Vérifiez la connexion Internet et les paramètres du pare-feu. Vérifiez la configuration du proxy si vous êtes derrière un pare-feu d'entreprise.

Solution:
gemini config set proxy http://proxy.company.com:8080

Configuration Problems

Issues with CLI configuration and settings

Fichier de configuration corrompu ou manquant

Réinitialisez la configuration aux valeurs par défaut et reconfigurez. Sauvegardez d'abord les paramètres importants.

Solution:
gemini config reset
gemini config set api-key YOUR_API_KEY

Modèle non disponible ou obsolète

Vérifiez les modèles disponibles et mettez à jour la configuration. Certains modèles peuvent avoir des restrictions régionales.

Solution:
gemini models list
gemini config set model gemini-pro

Variables d'environnement non reconnues

Assurez-vous que les variables d'environnement sont correctement définies et exportées. Vérifiez les noms et valeurs des variables.

Solution:
export GEMINI_API_KEY=your_key_here
export GEMINI_MODEL=gemini-pro

Problèmes de Performance

Réponses lentes et problèmes de performance

Temps de réponse lents

Vérifiez la connexion réseau, réduisez la taille de la requête ou essayez un modèle différent. Envisagez d'utiliser le streaming pour les longues réponses.

Solution:
gemini config set stream true
gemini config set max-tokens 1000

Utilisation mémoire élevée

Limitez l'historique des conversations, réduisez les tailles de lot ou redémarrez périodiquement la session CLI.

Solution:
gemini config set history-limit 10
gemini session clear

Erreurs de délai d'attente

Augmentez les paramètres de délai d'attente ou divisez les grandes requêtes en plus petits morceaux.

Solution:
gemini config set timeout 60000  # 60 seconds

Problèmes d'Outils et de Plugins

Problèmes avec les fonctionnalités des outils et plugins

Échec d'exécution de l'outil

Vérifiez les permissions de l'outil, les chemins de fichiers et les dépendances. Vérifiez que l'outil est correctement installé et configuré.

Solution:
gemini tools list
gemini tools test tool-name

Plugin ne se charge pas

Vérifiez le chemin du plugin, recherchez les erreurs de syntaxe et assurez-vous que toutes les dépendances sont installées.

Solution:
gemini plugins list
gemini plugins reload plugin-name

Problèmes de connexion au serveur MCP

Vérifiez l'état du serveur, la connectivité réseau et la compatibilité des versions de protocole.

Solution:
gemini mcp status
gemini mcp reconnect server-name
