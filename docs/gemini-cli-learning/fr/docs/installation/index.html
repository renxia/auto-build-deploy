Guide d'Installation

Installez et configurez Gemini CLI sur votre système

Configuration Système Requise

Assurez-vous que votre système répond à ces exigences

Windows

  • Windows 10 ou version ultérieure
  • Node.js 18.0.0 ou supérieur
  • PowerShell 5.1 ou PowerShell Core 7+
  • Windows Terminal (recommandé)

macOS

  • macOS 10.15 (Catalina) ou version ultérieure
  • Node.js 18.0.0 ou supérieur
  • Outils de ligne de commande Xcode
  • Terminal ou iTerm2

Linux

  • Ubuntu 18.04+ / CentOS 8+ / Debian 10+
  • Node.js 18.0.0 ou supérieur
  • curl ou wget
  • shell bash ou zsh

Méthodes d'Installation

Choisissez la méthode d'installation qui convient le mieux à vos besoins

Installation NPM

Méthode recommandée pour la plupart des utilisateurs

Étape 1: Installer globalement via npm

Ceci installe Gemini CLI globalement sur votre système

npm install -g @google/generative-ai-cli

Étape 2: Vérifier l'installation

Vérifiez que l'installation a réussi

gemini --version

Étape 3: Configurer la clé API

Configurez votre clé API Google AI

gemini config set api-key YOUR_API_KEY

Installation Docker

Environnement conteneurisé pour un déploiement cohérent

Étape 1: Télécharger l'image Docker

Téléchargez l'image Docker officielle

docker pull google/gemini-cli:latest

Étape 2: Exécuter avec la clé API

Démarrez un conteneur avec votre clé API

docker run -e GEMINI_API_KEY=your_key google/gemini-cli:latest

Étape 3: Mode interactif

Démarrez une session de chat interactive

docker run -it -e GEMINI_API_KEY=your_key google/gemini-cli:latest chat

Depuis les Sources

Construire à partir du code source pour le développement

Étape 1: Cloner le dépôt

Obtenez le code source depuis GitHub

git clone https://github.com/google-gemini/gemini-cli.git

Étape 2: Installer les dépendances

Installez toutes les dépendances requises

cd gemini-cli && npm install

Étape 3: Construire et lier

Construisez le projet et créez un lien symbolique global

npm run build && npm link

Configuration de la Clé API

Configurez votre clé API Google AI pour commencer à utiliser Gemini CLI

Obtenez Votre Clé API

  1. 1Visitez Google AI Studio
  2. 2Connectez-vous avec votre compte Google
  3. 3Cliquez sur "Créer une clé API" et copiez la clé

Configurer la Clé

En utilisant la commande CLI:

gemini config set api-key YOUR_API_KEY

En utilisant une variable d'environnement:

export GEMINI_API_KEY=YOUR_API_KEY

Vérifier l'Installation

Testez votre installation pour vous assurer que tout fonctionne

Vérifier la Version

gemini --version

Devrait afficher le numéro de version installé

Tester la Connexion API

gemini ask "Bonjour, pouvez-vous m'entendre ?"

Devrait retourner une réponse de Gemini

Vérifier la Configuration

gemini config list

Devrait afficher vos paramètres de configuration actuels

Problèmes Courants

Solutions aux problèmes d'installation courants

Commande introuvable

Si vous obtenez "commande introuvable" après l'installation, votre PATH pourrait ne pas inclure le répertoire bin global de npm.

npm config get prefix

Ajoutez le répertoire bin à votre PATH dans votre profil shell (.bashrc, .zshrc, etc.)

Permission refusée

Sur macOS/Linux, vous pourriez avoir besoin d'utiliser sudo ou configurer npm pour utiliser un répertoire différent.

npm config set prefix ~/.npm-global

Puis ajoutez ~/.npm-global/bin à votre PATH
