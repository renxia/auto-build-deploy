トラブルシューティングガイド

Gemini CLIの一般的な問題とよくある質問の解決策。さまざまな問題に対する迅速な修正と詳細なガイダンスを見つけてください。

クイック診断コマンド

一般的な問題を診断するためにこれらのコマンドから始めてください

gemini --version

CLIバージョンとビルド情報を確認

gemini config list

現在の設定を表示

gemini config reset

設定をデフォルト値にリセット

gemini logs

最近のエラーログとデバッグ情報を表示

gemini doctor

診断チェックとヘルス検証を実行

gemini cache clear

キャッシュデータと一時ファイルをクリア

一般的な問題と解決策

カテゴリ別に閲覧して特定の問題の解決策を見つけてください

インストールの問題

インストールとセットアップ中の問題

npm install が権限エラーで失敗する

--global フラグ付きで npm を使用するか、npm を別のディレクトリを使用するように設定します。macOS/Linux では、sudo を使用するか npm を適切に設定する必要があります。

解決策:
npm install -g @google/generative-ai-cli

インストール後にコマンドが見つからない

npm グローバル bin ディレクトリが PATH に含まれていることを確認してください。`npm config get prefix` で確認し、bin ディレクトリを PATH に追加してください。

解決策:
export PATH=$PATH:$(npm config get prefix)/bin

Node.js バージョン互換性の問題

Gemini CLI には Node.js 16 以上が必要です。nvm を使用して Node.js を更新するか、nodejs.org からダウンロードしてください。

解決策:
node --version  # Should be 16.0.0 or higher

認証とAPIの問題

APIキーと認証の問題

無効なAPIキーエラー

APIキーが正しく、適切な権限を持っていることを確認してください。必要に応じてGoogle AI Studioから新しいキーを取得してください。

解決策:
gemini config set api-key YOUR_API_KEY

APIクォータ超過

Google AI Studioで使用制限を確認してください。プランのアップグレードまたはレート制限の実装を検討してください。

解決策:
gemini config set rate-limit 10  # Requests per minute

ネットワーク接続の問題

インターネット接続とファイアウォール設定を確認してください。企業ファイアウォールの背後にいる場合は、プロキシ設定を確認してください。

解決策:
gemini config set proxy http://proxy.company.com:8080

Configuration Problems

Issues with CLI configuration and settings

設定ファイルが破損または欠落

設定をデフォルトにリセットして再設定してください。まず重要な設定をバックアップしてください。

解決策:
gemini config reset
gemini config set api-key YOUR_API_KEY

モデルが利用できないか廃止予定

利用可能なモデルを確認し、設定を更新してください。一部のモデルには地域制限がある場合があります。

解決策:
gemini models list
gemini config set model gemini-pro

環境変数が認識されない

環境変数が適切に設定およびエクスポートされていることを確認してください。変数名と値を確認してください。

解決策:
export GEMINI_API_KEY=your_key_here
export GEMINI_MODEL=gemini-pro

パフォーマンスの問題

応答の遅延とパフォーマンスの問題

応答時間が遅い

ネットワーク接続を確認し、リクエストサイズを減らすか、別のモデルを試してください。長い応答にはストリーミングの使用を検討してください。

解決策:
gemini config set stream true
gemini config set max-tokens 1000

メモリ使用量が多い

会話履歴を制限し、バッチサイズを減らすか、CLIセッションを定期的に再起動してください。

解決策:
gemini config set history-limit 10
gemini session clear

タイムアウトエラー

タイムアウト設定を増やすか、大きなリクエストを小さなチャンクに分割してください。

解決策:
gemini config set timeout 60000  # 60 seconds

ツールとプラグインの問題

ツールとプラグイン機能の問題

ツールの実行が失敗

ツールの権限、ファイルパス、依存関係を確認してください。ツールが適切にインストールおよび設定されていることを確認してください。

解決策:
gemini tools list
gemini tools test tool-name

プラグインが読み込まれない

プラグインパスを確認し、構文エラーをチェックし、すべての依存関係がインストールされていることを確認してください。

解決策:
gemini plugins list
gemini plugins reload plugin-name

MCPサーバー接続の問題

サーバーステータス、ネットワーク接続、プロトコルバージョンの互換性を確認してください。

解決策:
gemini mcp status
gemini mcp reconnect server-name
