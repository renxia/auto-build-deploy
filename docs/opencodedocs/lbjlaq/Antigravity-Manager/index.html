Antigravity Manager
Professional AI Account Manager & API Gateway

A full-featured desktop application for AI developers. Manage multiple Google/Anthropic accounts, convert Web Sessions to standard APIs, and enjoy intelligent request routing with automatic failover. Built with Tauri v2 + React (Rust).

Get Started GitHub
Why Antigravity Manager?

Your personal high-performance AI dispatch gateway. Not just account management, but the ultimate solution for breaking API call barriers.

Smart Dashboard

Real-time monitoring of all account health status, including quota for Gemini Pro, Flash, Claude, and Imagen. One-click switch to the best account.

Multi-Protocol Support

Full protocol adaptation: OpenAI /v1/chat/completions, Anthropic /v1/messages, and native Gemini SDK. Compatible with 99% of AI applications.

Intelligent Routing

Smart model routing with tiered priority based on account type (Ultra/Pro/Free). Automatic failover on 429/401 errors with millisecond-level retry.

OAuth 2.0 Authentication

Seamless OAuth flow with pre-generated authorization links. Support for batch import and automatic migration from legacy versions.

Imagen 3 Support

Advanced image generation with quality control. Support for custom size parameters and up to 100MB payload for 4K image recognition.

Cross-Platform

Native desktop app for macOS, Windows, and Linux. Docker deployment available for NAS/servers with noVNC web access.
