  • 🎯 최신 기능 빠르게 이해: 최신 버전(예: v3.0)의 changelog를 직접 읽으세요
  • 📚 버전 업그레이드 가이드: 현재 버전부터 역순으로 읽으며 업그레이드 경로를 이해하세요
  • 🔍 새로운 기능 심층 탐색: 관심 있는 버전의 상세 문서와 해당 고급 튜토리얼 장을 함께 읽으세요
