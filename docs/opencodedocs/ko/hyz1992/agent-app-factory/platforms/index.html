플랫폼 및 통합 ​

이 섹션은 AI 개발 환경을 구성하고 파이프라인이 정상적으로 실행될 수 있도록 필수 플러그인을 설치하는 데 도움이 됩니다.

학습 경로 ​

1. Claude Code 통합 가이드 ​

Claude Code를 사용하여 파이프라인을 실행하는 방법, 권한 구성 및 모범 사례에 대해 알아봅니다.

  • Claude Code 권한 구성
  • 파일 액세스 제한 이해
  • 파이프라인 실행 단계 습득

2. OpenCode 및 기타 AI 어시스턴트 ​

OpenCode, Cursor 등의 AI 어시스턴트를 사용하여 파이프라인을 실행하는 방법을 알아봅니다.

  • OpenCode CLI 구성
  • Cursor 프로젝트 설정
  • 호환성 설명

3. 필수 플러그인 설치 ​

superpowers 및 ui-ux-pro-max 플러그인을 설치하여 파이프라인이 정상적으로 실행되도록 합니다.

  • superpowers 플러그인 설치
  • ui-ux-pro-max 플러그인 설치
  • 플러그인 사용 가능 여부 확인
