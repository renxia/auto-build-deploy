부록 ​

이 장에는 참조 문서와 추가 자료가 포함되어 있어 AI App Factory의 기술적 세부사항을 더 깊이 이해할 수 있습니다.

하위 페이지 내비게이션 ​

CLI 명령어 참조 ​

전체 CLI 명령어 목록, 매개변수 설명 및 사용 예제. init, run, continue, status, list 및 reset 등의 명령어 포함.

디렉토리 구조 상세 설명 ​

Factory 프로젝트의 완전한 디렉토리 구조와 각 파일의 용도 이해. agents/, skills/, cli/ 등 핵심 디렉토리의 역할 포함.

코드 스타일 ​

생성된 코드가 따라야 할 TypeScript 코딩 스타일, 파일 구조, 주석 요구사항 및 커밋 메시지 스타일 이해.

기술 스택 상세 설명 ​

생성된 애플리케이션에서 사용하는 기술 스택 이해: Node.js + Express + Prisma + React Native, 각 부분의 기술 선정 이유 포함.

학습 경로 제안 ​

이 장의 문서는 참조 자료이므로 순서대로 읽을 필요가 없습니다. 주 튜토리얼을 학습하며 해당 장을 필요에 따라 참조하는 것이 좋습니다.

  • 설치 및 구성 학습 시, CLI 명령어 참조를 참조할 수 있음
  • Factory 프로젝트 초기화 학습 시, 디렉토리 구조 상세 설명를 참조할 수 있음
  • 파이프라인 각 단계 학습 시, 기술 스택 상세 설명를 참조할 수 있음
  • 생성된 코드의 품질 기준을 이해하려면 코드 스타일를 참조할 수 있음

전제 조건 ​

전제 요구사항 없음. 하지만 이 장의 내용을 더 잘 이해하려면 먼저 다음 튜토리얼을 완료하는 것이 좋습니다.

  • ✅ 빠른 시작: 아이디어에서 애플리케이션까지
  • ✅ 설치 및 구성
  • ✅ Factory 프로젝트 초기화

다음 단계 안내 ​

주 튜토리얼 학습을 완료한 후 계속 탐색할 수 있습니다.

  • 변경 로그에서 최신 기능 확인
  • 모범 사례를 검토하여 사용 효율 향상
  • 고급 주제를 깊이 있게 학습하여 시스템 설계 원리 이해
