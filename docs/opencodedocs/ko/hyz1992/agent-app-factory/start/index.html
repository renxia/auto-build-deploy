시작하기 ​

이 장에서는 AI App Factory를 빠르게 시작할 수 있도록 핵심 가치를 이해하고, 도구를 설치 및 구성하며, 7단계 파이프라인의 작동 원리를 파악하여 첫 번째 앱을 생성할 준비를 합니다.

학습 경로 ​

이 장의 내용을 다음 순서대로 학습하는 것을 권장합니다:

1. 빠른 시작: 아이디어부터 앱까지 ​

AI App Factory의 핵심 가치와 전제 요구사항을 이해하고, 전체 사례를 통해 아이디어부터 앱까지의 엔드투엔드 프로세스를 경험해 보세요.

  • 체크포인트 기반의 다중 에이전트 협업 메커니즘 이해
  • 7단계 파이프라인의 전체 작업 흐름 파악
  • 명확한 제품 설명을 제공하는 기술 습득

2. 설치 및 구성 ​

CLI 도구를 전역 설치하고, AI 도우미(Claude Code/OpenCode) 및 관련 환경을 구성하세요.

  • npm을 사용하여 agent-app-factory 전역 설치
  • Claude Code 또는 OpenCode 구성
  • 설치 성공 여부 확인

3. Factory 프로젝트 초기화 ​

factory init 명령을 사용하여 디렉터리를 초기화하고, 생성된 프로젝트 구조와 구성 파일을 이해하세요.

  • 프로젝트 초기화 명령 실행
  • 프로젝트 디렉터리 구조와 구성 파일 이해
  • 제품 아이디어 입력 준비

4. 7단계 파이프라인 개요 ​

Bootstrap부터 Preview까지 전체 파이프라인 프로세스, 각 단계의 입력/출력 및 체크포인트 메커니즘을 심층적으로 이해하세요.

  • 7단계의 목표와 산출물 이해
  • 체크포인트 메커니즘 및 수동 확인 프로세스 마스터
  • 파이프라인 실행 제어 방법 학습

전제 조건 ​

이 장을 학습하기 전에 다음이 필요합니다:

전제 요구사항

  1. Node.js 환경: Node.js v18 이상이 설치되어 있어야 합니다.
  2. AI 프로그래밍 도우미: 다음 도구 중 하나를 구성해야 합니다.
    • Claude Code (추천) ⭐
    • OpenCode 또는 에이전트 모드를 지원하는 다른 AI 도우미
  3. 기본 명령줄 작업: 터미널에서 명령을 실행할 수 있어야 합니다.

다음 단계 ​

이 장의 학습을 완료하면 다음을 마스터하게 됩니다:

  • ✅ AI App Factory의 핵심 작동 원리
  • ✅ CLI 도구의 설치 및 구성
  • ✅ 프로젝트 초기화 및 구조 이해
  • ✅ 7단계 파이프라인의 전체 프로세스

다음으로 플랫폼 및 통합 장을 학습하여 다양한 AI 도우미에서 파이프라인을 실행하는 방법을 알아보세요.
