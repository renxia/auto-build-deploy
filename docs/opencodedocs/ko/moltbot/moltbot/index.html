테마

Moltbot
당신의 개인 AI 어시스턴트, 어디서나 사용 가능 (구 Clawdbot)

자신의 기기에서 AI 어시스턴트를 실행하고, WhatsApp, Telegram, Slack 등 12개 이상의 채널과 원활하게 연결하세요. 브라우저 자동화, 멀티 디바이스 협업, 음성 상호작용을 지원하며, 데이터는 완전히 로컬로 처리되어 프라이버시가 보장됩니다.

빠른 시작 GitHub
왜 Moltbot을 선택해야 할까요?

강력하고, 프라이버시가 안전하며, 배포하기 쉬운 개인 AI 어시스턴트

🌐 멀티 채널 통합

하나의 설정으로 WhatsApp, Telegram, Slack 등 12개 이상의 메시징 플랫폼 관리

🔒 로컬 우선

모든 데이터와 AI 런타임이 사용자의 기기에서 실행되어 제로 프라이버시 유출

🛠️ 강력한 툴체인

브라우저 자동화, 예약 작업, Webhook 등 확장 기능

📱 멀티 디바이스 협업

macOS/iOS/Android 노드가 연결되어 로컬 작업 수행 가능

🎤 음성 상호작용

Voice Wake, Talk Mode 및 다양한 TTS 엔진 지원

🧠 메모리 시스템

벡터화 저장 및 의미 검색으로 AI가 모든 대화를 기억

🔐 보안 격리

Sandbox 격리, DM 페어링 보호, 툴 권한 제어
