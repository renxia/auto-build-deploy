고급 기능 ​

개요 ​

이 장에서는 Clawdbot의 고급 기능을 심층적으로 소개하며, AI 어시스턴트의 강력한 기능을 최대한 활용할 수 있도록 도와드립니다. AI 모델 구성 및 다중 Agent 협업부터 브라우저 자동화, 메모리 시스템 및 음성 기능까지, 필요에 따라 학습 내용을 선택할 수 있습니다.

사전 요구사항

이 장을 학습하기 전에 다음 내용을 완료하세요:

  • 빠른 시작
  • Gateway 시작

학습 경로 ​

요구사항에 따라 다른 학습 경로를 선택할 수 있습니다:

🚀 빠른 시작 경로 (신규 사용자 추천) ​

  1. AI 모델 및 인증 구성 - 선호하는 AI 모델 구성
  2. 명령 실행 도구 및 승인 - AI가 명령을 안전하게 실행하도록 설정
  3. 웹 검색 및 스크래핑 도구 - AI의 지식 습득 능력 확장

🤖 AI 능력 확장 경로 ​

  1. 세션 관리 및 다중 Agent - AI 협업 메커니즘 이해
  2. 메모리 시스템 및 벡터 검색 - AI가 중요한 정보를 기억하도록 설정
  3. 스킬 플랫폼 및 ClawdHub - 스킬 팩 사용 및 공유

🔧 자동화 도구 경로 ​

  1. 브라우저 자동화 도구 - 웹페이지 작업 자동화
  2. Cron 예약 작업 및 Webhook - 예약 작업 및 이벤트 트리거
  3. 원격 Gateway 및 Tailscale - 원격 AI 어시스턴트 액세스

🎨 상호작용 경험 경로 ​

  1. Canvas 시각화 인터페이스 및 A2UI - 시각화 상호작용 인터페이스
  2. 음성 웨이크업 및 텍스트 음성 변환 - 음성 상호작용 기능

🔒 보안 및 배포 경로 ​

  1. 보안 및 샌드박스 격리 - 보안 메커니즘 심층 이해
  2. 원격 Gateway 및 Tailscale - 안전한 원격 액세스

다음 단계 ​

이 장의 학습을 완료한 후 다음을 수행할 수 있습니다:

  1. 심층 학습 - 문제 해결을 확인하여 발생한 문제 해결
  2. 배포 이해 - 배포 옵션을 확인하여 Clawdbot을 프로덕션 환경에 배포
  3. 확장 개발 - 개발 가이드를 확인하여 플러그인 개발 및 코드 기여 방법 학습
  4. 구성 확인 - 전체 구성 참조를 참조하여 모든 구성 옵션 확인

학습 제안
