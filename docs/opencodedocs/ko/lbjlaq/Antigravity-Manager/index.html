Antigravity-Manager
로컬 AI 계정 관리 및 프로토콜 리버스 프록시 시스템

여러 AI 플랫폼의 Web/Session/OAuth 인증을 통합 API 게이트웨이로 변환하여 다중 계정 스케줄링, 지능형 회전, 할당량 모니터링 및 다중 프로토콜 호환 기능을 제공합니다. Tauri v2 + React + Rust 기반 데스크톱 애플리케이션입니다.

빠른 시작 GitHub
왜 Antigravity-Manager인가요?

로컬 고성능 AI 스케줄링 게이트웨이. 단순한 계정 관리를 넘어, API 호출 장벽을 깨는 최종 솔루션.

다중 프로토콜 호환 엔드포인트

OpenAI /v1/chat/completions, Anthropic /v1/messages, Gemini 네이티브 SDK 엔드포인트를 외부에 노출하여 기존 클라이언트와 SDK를 재사용할 수 있습니다.

지능형 계정 스케줄링

회전, 고정 계정, 스티키 세션 및 실패 재시도를 지원합니다. 429/401 발생 시 자동 회전, invalid_grant 시 자동 오프라인, 밀리초급 자동 복구.

OAuth 인증 추가

브라우저 OAuth 인증과 Refresh Token 듀얼 채널을 통한 계정 추가, 일괄 가져오기/내보내기, V1/DB 핫 마이그레이션을 지원합니다.

할당량 거버넌스 및 보호

각 모델의 남은 할당량을 실시간으로 모니터링하고 모델 수준 할당량 보호와 Smart Warmup을 지원하여 무감각 소모를 방지합니다.

트래픽 모니터링 및 통계

Proxy Monitor는 요청 로그를 실시간으로 기록하고 Token Stats는 시간/일/주 단위로 소모를 통계하며 모델 및 계정별 분석을 지원합니다.

크로스 플랫폼 배포

macOS, Windows, Linux 데스크톱을 네이티브로 지원합니다. Docker noVNC와 Headless Xvfb 솔루션을 제공하여 NAS/서버 배포에 적합합니다.
