빠른 시작 ​

이 장에서는 opencode-mystatus 플러그인을 5분 만에 설치부터 사용까지 안내합니다.

학습 경로 ​

다음 순서대로 학습하는 것을 권장합니다:

1. 빠른 시작 ​

5분 안에 플러그인 설치 및 설정을 완료하고, 첫 번째 AI 플랫폼 할당량을 조회합니다.

  • 세 가지 설치 방법: AI가 설치 도와주기, 수동 설치, 로컬 파일에서 설치
  • 슬래시 명령어 /mystatus 설정
  • 설치 성공 확인

2. mystatus 사용하기 ​

슬래시 명령어와 자연어 두 가지 사용 방법을 학습합니다.

  • 슬래시 명령어 /mystatus 사용법
  • 자연어 질문으로 mystatus 도구 트리거
  • 두 방법의 차이점과 사용 시나리오

3. 출력 해석하기 ​

프로그레스 바, 재설정 시간, 다중 계정 출력 형식을 이해합니다.

  • 프로그레스 바 의미: 채워진 블록 vs 빈 블록
  • 다른 플랫폼의 할당량 주기(3시간, 5시간, 월간)
  • 다중 계정 시나리오에서의 출력 형식

전제 조건 ​

학습을 시작하기 전에 다음을 확인하세요:

  • OpenCode가 설치되어 있어야 합니다
  • 적어도 하나의 AI 플랫폼 인증 정보(OpenAI, 지푸 AI, Z.ai, GitHub Copilot 또는 Google Cloud)가 설정되어 있어야 합니다

다음 단계 ​

이 장을 완료하면 플랫폼 기능을 계속 학습하여 각 플랫폼의 할당량 조회 세부 사항을深入了解할 수 있습니다.
