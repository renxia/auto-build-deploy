agent-skills
Vercel AI 코딩 에이전트 스킬 팩

Claude 등 AI 에이전트를 위한 확장 가능한 스킬 컬렉션. React 성능 최적화 규칙 57개, 웹 디자인 가이드 100+개 포함, Vercel 원클릭 배포 지원.

빠른 시작 GitHub
agent-skills를 선택해야 하는 이유?

Vercel 공식 제품, 엔지니어링 팀의 베스트 프랙티스를 재사용 가능한 스킬로 패키징하여 AI 에이전트가 자동으로 적용하도록 합니다.

React 성능 최적화

Vercel 엔지니어링 팀이 검증한 규칙 57개, CRITICAL/HIGH/MEDIUM 등급으로 분류, 워터폴 제거, 번들 최적화 등 8개 카테고리를 포함.

웹 디자인 가이드

UI 감사 규칙 100+개, 접근성, 애니메이션 성능, 다크 모드, 폼 검증 등 다차원을 포괄.

Vercel 원클릭 배포

40+개 프레임워크 자동 감지, 구성 없이 배포 및 미리보기 링크와 소유권 이전 링크 반환.

자동 트리거 메커니즘

설치 후 에이전트가 자동으로 작업을 인식하고 해당 스킬을 활성화, 수동 선택이나 명령어 기억 불필요.

표준화된 형식

Agent Skills 스펙 준수, 각 스킬은 SKILL.md, scripts/, references/ 포함, 확장 용이.

오픈 소스 자유 확장

MIT 라이선스, 스킬 자유롭게 확장 및 사용자 정의하거나 기존 규칙을 기반으로 자신만의 스킬 팩 구축 가능.
