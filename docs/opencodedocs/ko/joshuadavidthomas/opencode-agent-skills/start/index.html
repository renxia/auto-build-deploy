빠른 시작 ​

이 장에서는 OpenCode Agent Skills 플러그인을 처음부터 사용하는 방법을 안내합니다. 플러그인의 핵심 가치를 이해하고, 설치 및 구성을 완료한 후 첫 번째 스킬을 생성하게 됩니다.

이 장의 내용 ​

OpenCode Agent Skills란 무엇인가요?

플러그인의 핵심 가치와 기능 특성을 이해하세요. 동적 스킬 발견, 컨텍스트 주입, 압축 복원 등 메커니즘이 포함됩니다.

설치 가이드

플러그인을 설치하고 실행을 확인하세요. 기본 설치, 고정 버전 설치, 로컬 개발 설치 세 가지 방법을 지원합니다.

첫 번째 스킬 만들기

스킬 디렉토리 구조와 SKILL.md 형식 규칙을 마스터하고, 직접 간단한 스킬을 생성하고 테스트해보세요.

학습 경로 ​

다음 순서대로 학습하는 것을 권장합니다:

  1. OpenCode Agent Skills란 무엇인가요? — 플러그인이 무엇을 할 수 있는지 먼저 이해하고 전체적인 인식을 구축하세요
  2. 설치 가이드 — 플러그인을 설치하여 OpenCode에 스킬 관리 기능을 부여하세요
  3. 첫 번째 스킬 만들기 — 직접 실습하여 나만의 첫 번째 스킬을 만들어보세요

예상 소요 시간

이 장을 완료하는 데 약 30-45분이 소요됩니다.

다음 단계 ​

이 장을 완료한 후 **플랫폼 기능**을 계속 학습하여 스킬 발견 메커니즘, 스킬 로딩, 자동 매칭 등 고급 기능을 자세히 이해할 수 있습니다.
