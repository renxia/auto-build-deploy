opencode-agent-skills
OpenCode AI 스킬 동적 관리 플러그인

OpenCode에 동적 스킬 발견 및 관리 기능을 제공하는 플러그인 시스템입니다. AI 에이전트가 재사용 가능한 스킬 라이브러리를 로드하고 사용할 수 있도록 지원합니다. 프로젝트 및 세션 간 스킬 재사용을 구현하며, 의미적 유사도를 기반으로 관련 스킬을 자동 추천합니다.

빠른 시작 GitHub
왜 OpenCode Agent Skills를 선택해야 할까요?

OpenCode에 동적 스킬 발견 및 관리 기능을 제공하여, 프로젝트 및 세션 간 스킬 재사용을 구현합니다.

동적 스킬 발견

프로젝트, 사용자, 플러그인 디렉토리에서 자동으로 스킬을 발견합니다. 6개 위치를 스캔하고 우선순위에 따라 첫 번째 일치하는 스킬을 반환합니다.

Claude Code 호환

기존 Claude Code 스킬 및 플러그인 형식을 지원하여 마이그레이션 비용을 줄이고 기존 리소스를 재사용할 수 있습니다.

의미론적 매칭 추천

HuggingFace 임베딩 모델을 사용하여 유사도를 계산하고, 사용자 입력을 기반으로 관련 스킬을 자동으로 추천합니다.

컨텍스트 주입

스킬 내용이 XML 형식으로 세션 컨텍스트에 주입되어, AI가 스킬 가이드를 따라 고품질 결과를 출력합니다.

압축 후 복구

세션이 압축된 후 스킬 목록을 자동으로 다시 주입하여, 장시간 세션에서 스킬을 지속적으로 사용할 수 있습니다.

스크립트 실행 지원

스킬 디렉토리에 실행 가능한 스크립트를 포함할 수 있으며, 스킬 디렉토리 컨텍스트 내에서 자동화 작업을 실행할 수 있습니다.

입문 가이드 (Start)
OpenCode Agent Skills란 무엇인가요?

플러그인의 핵심 가치와 기능적 특성을 이해합니다.

OpenCode Agent Skills 설치

플러그인 설치를 완료하고 실행을 검증합니다.

첫 번째 스킬 만들기

스킬 디렉토리 구조와 SKILL.md 형식 사양을 마스터합니다.

핵심 기능 (Platforms)
스킬 발견 메커니즘 상세 설명

여러 위치에서 자동으로 스킬을 발견하는 방법을 이해합니다.

사용 가능한 스킬 쿼리 및 나열

get_available_skills 도구 및 네임스페이스를 사용하는 방법을 배웁니다.

세션 컨텍스트에 스킬 로드

use_skill 도구 사용과 XML 컨텐츠 주입 메커니즘을 마스터합니다.

자동 스킬 추천: 의미론적 매칭 원리

의미적 유사도를 기반으로 관련 스킬을 자동으로 추천하는 방법을 이해합니다.

스킬 스크립트 실행

run_skill_script 도구를 사용하여 스킬 디렉토리에서 자동화 스크립트를 실행하는 방법을 배웁니다.

스킬 파일 읽기

read_skill_file 도구를 사용하여 스킬의 문서, 구성 등 지원 파일에 액세스합니다.

고급 기능 (Advanced)
Claude Code 스킬 호환성

플러그인이 Claude Code의 스킬 및 플러그인 시스템과 호환되는 방식을 이해합니다.

Superpowers 워크플로우 통합

Superpowers 모드를 구성하고 사용하여 엄격한 워크플로우 가이드를 얻습니다.

네임스페이스 및 스킬 우선순위

스킬의 네임스페이스 시스템과 발견 우선순위 규칙을 이해합니다.

컨텍스트 압축 복구 메커니즘

긴 세션에서 스킬의 가용성을 유지하는 방법을 이해합니다.

자주 묻는 질문 (FAQ)
일반적인 문제 해결

플러그인 사용 중 발생하는 일반적인 문제를 해결합니다.

보안 고려사항

플러그인의 보안 메커니즘과 모범 사례를 이해합니다.
