opencode-md-table-formatter
AI 표 자동 정렬

OpenCode 플러그인, AI가 생성한 Markdown 표를 자동으로 포맷합니다. 콘텐츠 너비를 지능적으로 인식하고, 굵게, 기울임꼴, 코드 블록 등의 형식을 지원하여 표를 항상 깔끔하고 아름답게 만듭니다.

튜토리얼 보기 GitHub
핵심 기능

Markdown 구문 자동 인식, 표시 너비 지능적 계산

Markdown 형식 지능적 인식

굵게, 기울임꼴, 코드 블록 등 형식의 표시 너비를 정확하게 계산하여 표가 항상 완벽하게 정렬되도록 합니다

AI 생성 후 자동 포맷

experimental.text.complete 훅을 통해 자동으로 트리거되며, AI가 텍스트를 생성한 후 수동 작업 없이 즉시 포맷됩니다

코드 블록 지능적 보호

인라인 코드 블록 내의 Markdown 기호가 잘못 제거되지 않도록 보호하여 코드 내 구문이 원래대로 유지되도록 합니다

세 가지 정렬 방식 지원

왼쪽 정렬(:---), 가운데 정렬(:---:), 오른쪽 정렬(---:) 세 가지 형식을 지원합니다

중첩 Markdown 처리

다중 정규 표현식 알고리즘을 사용하여 굵게, 기울임꼴, 취소선 등 중첩된 Markdown 구문을 처리합니다

경계 상황 호환성

이모지, 유니코드 문자, 빈 셀, 긴 콘텐츠 등 경계 상황을 올바르게 처리합니다

조용한 실행

콘솔 로그 출력이 없으며, 오류가 워크플로우를 중단하지 않고 조용하게 포맷을 완료합니다

검증 피드백

유효하지 않은 표에 친절한 오류 주석을 추가하여 사용자가 문제를 빠르게 찾을 수 있도록 도와줍니다
