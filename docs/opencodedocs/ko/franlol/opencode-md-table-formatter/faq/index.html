자주 묻는 질문 ​

섹션 개요 ​

이 섹션에서는 opencode-md-table-formatter 플러그인의 자주 묻는 질문과 문제 해결 방법을 다루며, 테이블 포맷팅 과정에서 발생하는 다양한 문제를 빠르게 찾아내고 해결하는 데 도움을 줍니다.

하위 페이지 탐색 ​

자주 묻는 질문: 테이블이 포맷팅되지 않는 문제 해결 ​

플러그인 사용 중 발생하는 일반적인 문제와 해결 방법을 포괄적으로 소개합니다:

  • 테이블이 포맷팅되지 않음: 구성, 버전 요구사항, OpenCode 재시작 확인
  • 유효하지 않은 테이블 구조: "invalid structure" 오류 의미 이해, 올바른 테이블 구문 학습
  • 포맷팅 실패: "table formatting failed" 오류 및 예외 상황 처리
  • 지원되지 않는 테이블 유형: HTML 테이블, 다중 행 셀 등 제한된 시나리오 이해
  • 포맷팅 후 정렬되지 않음: 숨김 모드, 셀 초과 길이, 인라인 코드 등 문제 해결

학습 경로 제안 ​

기능 개요를 완료한 후 실제 문제가 발생했을 때 이 섹션을 조회하거나 문제 해결 참조 매뉴얼로 사용하는 것을 권장합니다.

다음 순서대로 학습할 수 있습니다:

  1. 문제가 발생했을 때: 증상에 따라 문제 유형(테이블이 포맷팅되지 않음, 오류 메시지 등)을 먼저 파악
  2. 단계별 문제 해결: 문제 해결 가이드의 확인 목록에 따라 하나씩 검증
  3. 심층 이해: 테이블 사양 및 숨김 모드 원리를 읽어 기술 세부사항 이해
  4. 알려진 제한 사항: 알려진 제한 사항를 확인하여 플러그인 경계 이해

전제 조건 ​

이 섹션을 학습하기 전에 다음을 먼저 완료하는 것이 좋습니다:

  • ✅ 1분 빠른 시작: 설치 및 구성 - 플러그인 설치 및 구성 방법 이해
  • ✅ 기능 개요: 자동 포맷팅의 마법 - 플러그인의 핵심 기능 및 작동 원리 이해

또한 다음을 갖추어야 합니다:

  • Markdown 테이블의 기본 구문에 익숙함
  • OpenCode 플러그인 구성 방식(opencode.jsonc) 이해

다음 단계 ​

이 섹션을 완료한 후 다음을 계속 학습할 수 있습니다:

  • 고급 지식:

    • 테이블 사양: 어떤 테이블이 포맷팅될 수 있는가 - 유효한 테이블의 구조 요구사항 파악
    • 숨김 모드 원리: 왜 너비 계산이 중요한가 - OpenCode 숨김 모드의 작동 원리 이해
    • 정렬 방식 상세: 왼쪽 정렬, 가운데 정렬, 오른쪽 정렬 - 세 가지 정렬 방식의 구문 및 효과 파악

  • 기술 세부사항:

    • 알려진 제한 사항: 플러그인의 경계는 어디까지인가 - 플러그인의 알려진 제한 사항 및 적용되지 않는 시나리오 이해
    • 기술 세부사항: 캐싱 메커니즘 및 성능 최적화 - 플러그인의 내부 캐싱 메커니즘 및 성능 최적화 전략 이해
