고급 기능: Markdown 표 서식의 기술적 세부사항 심층 이해 ​

섹션 개요 ​

이 섹션에서는 Markdown 표 서식의 기술적 세부사항을 심층적으로 다룹니다. OpenCode 숨김 모드의 작동 원리, 유효한 표의 구조 규격, 정렬 방식에 대한 상세 설명을 포함합니다. 이 내용을 학습하면 플러그인이 표 서식을 처리하는 방법과 일반적인 오류를 방지하는 방법을 완전히 이해할 수 있습니다.

학습 경로 ​

이 섹션의 내용은 다음 순서로 학습하는 것을 권장합니다:

학습 경로

  1. 숨김 모드 원리 → OpenCode의 숨김 모드를 특별하게 처리해야 하는 이유 이해
  2. 표 규격 → 올바르게 서식화될 수 있는 표의 조건 마스터
  3. 정렬 방식 상세 → 표의 정렬과 미적 품질을 제어하는 방법 학습

전제 조건 ​

이 섹션을 시작하기 전에 다음 사항을 확인하세요:

  • [x] 1분 빠르게 시작을 완료하고 플러그인을 성공적으로 설치 및 구성했는지 확인
  • [x] 기능 개요를 읽고 플러그인의 기본 기능을 이해했는지 확인

중요 알림

아직 기초 학습을 완료하지 않았다면 빠르게 시작로 돌아가서 시작하는 것을 권장합니다.

섹션 탐색 ​

숨김 모드 원리 ​

OpenCode 숨김 모드의 작동 원리와 플러그인이 표시 너비를 올바르게 계산하는 방법을 이해합니다. 다음 내용을 학습합니다:

  • 숨김 모드란 무엇이며 특별한 처리가 필요한 이유
  • Markdown 기호 제거 알고리즘이 작동하는 방식
  • 너비 계산에서 Bun.stringWidth()의 역할

예상 소요 시간: 8분 | 난이도: 중간 | 전제 조건: 기능 개요

표 규격 ​

유효한 Markdown 표의 구조 요구사항을 마스터하고 "유효하지 않은 표" 오류를 방지합니다. 다음 내용을 학습합니다:

  • 어떤 표 구조가 유효한지
  • 구분 행의 역할과 서식 요구사항
  • 열 수 일관성 검사의 원리
  • 표 구조 문제를 빠르게 식별하는 방법

예상 소요 시간: 6분 | 난이도: 입문 | 전제 조건: 숨김 모드 원리

정렬 방식 상세 ​

세 가지 정렬 방식의 구문과 효과를 마스터하여 표를 더 아름답게 만듭니다. 다음 내용을 학습합니다:

  • 왼쪽 정렬, 중앙 정렬, 오른쪽 정렬의 구문
  • 구분 행에서 정렬 방식을 지정하는 방법
  • 셀 내용 채우기 알고리즘
  • 정렬과 표시 너비의 관계

예상 소요 시간: 10분 | 난이도: 중간 | 전제 조건: 표 규격

섹션 요약 ​

이 섹션을 완료하면 다음을 수행할 수 있습니다:

  • ✅ OpenCode 숨김 모드의 작동 원리 이해
  • ✅ 유효한 표의 구조 요구사항 마스터
  • ✅ 유효하지 않은 표 식별 및 수정
  • ✅ 세 가지 정렬 방식 능숙하게 사용
  • ✅ 플러그인 내부의 기술적 구현 세부사항 이해

다음 단계 ​

이 섹션을 완료한 후 다음을 수행할 수 있습니다:

  1. 실제 문제 해결 → 일반적인 문제를 학습하여 문제를 빠르게 찾고 해결
  2. 기술적 경계 이해 → 알려진 제한 사항을 읽고 플러그인의 적용 가능한 시나리오 이해
  3. 구현 심층 이해 → 기술적 세부사항를 확인하여 캐싱 메커니즘과 성능 최적화 이해

실용적인 제안

표 서식 문제를 빠르게 해결하고 싶다면 이 섹션의 표 규격을 먼저 읽어보세요. 가장 일반적인 유효하지 않은 표 사례가 포함되어 있습니다.
