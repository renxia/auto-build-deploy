Everything Claude Code
실전에서 검증된 Claude Code 플러그인 설정 모음

Anthropic 해커톤 우승팀이 제공하는 완전한 Claude Code 플러그인 설정 모음. 9개의 agents, 14개의 commands, 8개의 rules 세트, 11개의 skills, 그리고 15개 이상의 hooks를 제공하여 AI 기반 개발 효율성을 향상시킵니다. Windows, macOS 및 Linux 크로스 플랫폼 Node.js 스크립트 지원.

빠른 시작 GitHub
왜 Everything Claude Code를 선택해야 할까요?

처음부터 설정할 필요가 없습니다. 10개월 이상 일상적인 사용을 통해 다듬어진 실전 검증 설정을 받아보세요.

실전 검증

Anthropic 해커톤 우승팀, 10개월 이상 일상 사용으로 다듬어짐, 실제 제품 구축 경험.

완전한 설정

9개의 agents, 14개의 commands, 8개의 rules 세트, 11개의 skills, 15개 이상의 hooks, 모든 개발 시나리오 커버.

크로스 플랫폼 지원

모든 hooks 및 스크립트가 Node.js로 재작성됨, Windows, macOS, Linux에서 원활하게 실행.

스마트 패키지 매니저

npm, pnpm, yarn, bun 자동 감지, 6가지 우선순위 설정 지원, 수동 설정 불필요.

지속적 학습

세션에서 자동으로 재사용 가능한 패턴을 추출하여 learned skills로 저장.

코드 품질 보장

TDD 프로세스, 코드 리뷰, 보안 감사, 80%+ 테스트 커버리지 보장.
