Antigravity Auth
Claude & Gemini 인증의 핵심

API Key 없이 Google OAuth 자격 증명을 통해 Claude Opus 4.5, Sonnet 4.5 및 Gemini 3 Pro/Flash에 직접 액세스하세요. 다중 계정 부하 분산 및 세션 자동 복구를 지원합니다.

빠른 시작 GitHub
왜 Antigravity Auth인가요?

API Key 수동 구성의 번거로움을 해결하고, 자동화된 할당량 관리와 높은 동시성 개발에 최적화되었습니다.

API Key 없음

비싼 API Key를 신청할 필요 없이 Google 계정 자격 증명을 직접 사용하여 원클릭 인증 로그인을 완료하세요.

다중 계정 부하 분산

최대 10개의 Google 계정 자동 로테이션을 지원하고, 요청을 지능적으로 할당하여 속도 제한을 완전히 해결하세요.

Thinking 모델 지원

Claude 및 Gemini 3의 Thinking 모드를 완벽하게 지원하고, Budget을 구성하여 심층 추론 능력을 잠금 해제하세요.

세션 자동 복구

도구 호출 실패? 세션 중단? 플러그인이 자동으로 재시도하고 컨텍스트를 복구하므로 수동 개입이 필요 없습니다.

플러그인 호환성

oh-my-opencode, DCP 등 주요 플러그인과 완벽하게 통합됩니다.

이중 할당량 시스템

Google에서 제공하는 무료 및 유료 할당량 풀을 최대한 활용하세요.
