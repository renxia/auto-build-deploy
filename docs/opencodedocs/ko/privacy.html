개인정보처리방침 ​

최종 업데이트 날짜: 2026-01-01

OpenCodeDocs를 이용해 주셔서 감사합니다. 저희는 귀하의 개인정보와 프라이버시 보호를 매우 중요하게 생각합니다. 본 개인정보처리방침은 저희가 어떻게 귀하의 개인정보를 수집, 사용, 저장하고 보호하는지 이해하실 수 있도록 도와드립니다.

1. 수집하는 정보 ​

OpenCodeDocs 서비스를 이용하실 때 다음 유형의 정보를 수집할 수 있습니다:

계정 정보 ​

  • 이메일 주소: 계정 등록, 로그인 확인 및 중요 알림 전송에 사용
  • 사용자 이름: 귀하가 선택한 표시 이름
  • GitHub 계정 정보: GitHub 로그인을 선택한 경우 공개 정보를 수집

사용 데이터 ​

  • 방문 기록: 귀하가 방문한 튜토리얼 페이지와 방문 시간
  • 상호작용 행동: 코드 복사, 섹션 이동 등 작업
  • 검색 기록: 사이트 내 검색 키워드

기기 정보 ​

  • 브라우저 유형: Chrome, Firefox, Safari 등
  • 운영체제: Windows, macOS, Linux 등
  • IP 주소: 보안 보호 및 지역 최적화에 사용

2. 정보 사용 방법 ​

수집한 정보는 다음 목적에 사용됩니다:

  • 서비스 제공: 튜토리얼 콘텐츠에 정상적으로 접근하고 사용할 수 있도록 보장
  • 개인화 경험: 독서 진도 및 설정 기본값 기억
  • 제품 개선: 사용 패턴을 분석하여 콘텐츠 및 기능 최적화
  • 보안 보호: 비정상적인 행동을 감지하여 남용 방지
  • 알림 전송: 튜토리얼 업데이트 알림 및 중요 공지 발송(구독 취소 가능)

3. 정보 공유 및 공개 ​

개인정보는 판매하지 않습니다. 다음 경우에만 공유할 수 있습니다:

  • 제3자 서비스 제공자: Vercel(호스팅), Plausible(분석) 등 업무 수행에 필요한 정보만 접근 가능
  • 법률 요구: 법률 또는 규정에 따라 공개 필요
  • 권익 보호: OpenCodeDocs 및 사용자 권익을 보호하기 위해 필요한 경우

4. 데이터 저장 및 보안 ​

  • 저장 위치: 미국 및 아시아태평양 지역의 보안 서버에 데이터 저장
  • 암호화 조치: 전송 데이터는 TLS/SSL 암호화 사용, 민감 데이터는 AES-256 암호화 저장
  • 접근 제어: 내부 직원의 데이터 접근 권한을 엄격히 제한
  • 보유 기간: 계정 탈퇴 후 개인 데이터는 30일 이내 삭제

5. 귀하의 권리 ​

『개인정보보호법』및 GDPR에 따라 다음 권리를 가집니다:

  • 접근권: 보유하고 있는 귀하의 개인정보를 볼 권리
  • 정정권: 부정확한 개인정보의 정정을 요청할 권리
  • 삭제권: 개인정보의 삭제를 요청할 권리
  • 내보내기 권리: 데이터를 공통 형식으로 내보낼 권리
  • 동의 철회: 언제든지 이전에 동의한 내용을 철회할 권리

위 권리를 행사하려면 아래 연락처를 통해 저희에게 연락해 주세요.

6. Cookie 정책 ​

다음 유형의 Cookie를 사용합니다:

  • 필수 Cookie: 로그인 상태 유지, 언어 기본값 기억
  • 분석 Cookie: 방문량 및 사용 패턴 통계(개인정보 친화적인 Plausible 사용)
  • 광고 Cookie:
    • Google을 포함한 제3자 공급업체는 귀하의 사이트 및 기타 사이트에서의 방문 기록을 기반으로 광고를 게재하기 위해 Cookie를 사용합니다.
    • Google은 광고 Cookie를 사용하여 Google 및 파트너가 귀하의 사이트 및/또는 인터넷의 다른 사이트에서의 방문 기록을 기반으로 사용자에게 광고를 게시할 수 있도록 합니다.
    • 사용자는 Google 광고 설정에서 개인화된 광고를 끌 수 있습니다. 또는 www.aboutads.info에 방문하여 개인화된 광고에 사용되는 제3자 공급업체의 Cookie를 끄도록 안내할 수 있습니다.

브라우저 설정에서 Cookie를 관리할 수 있습니다. 필수 Cookie를 비활성화하면 일부 기능이 정상적으로 작동하지 않을 수 있습니다.
