시작하기 ​

이 섹션에서는 opencode-notify 플러그인을 빠르게 설치하고, 데스크톱 알림 기능을 체험하며, 핵심 작동 메커니즘을 이해하는 것을 돕습니다.

학습 경로 ​

1. 빠른 시작 ​

5분 만에 설치를 완료하고 첫 번째 알림을 체험합니다.

  • OCX로 원클릭 설치
  • 알림 기능이 정상 작동하는지 확인

2. 작동 원리 ​

플러그인의 핵심 기능, 스마트 필터링 메커니즘, 알림 트리거 조건을 이해합니다.

  • 네 가지 알림 유형과 트리거 시기 파악
  • 스마트 필터링이 알림 폭격을 피하는 원리 이해

전제 조건 ​

  • OpenCode Agent가 설치되어 있어야 함 (공식 웹사이트에서 다운로드)
  • 기본적인 터미널 조작 지식

다음 단계 ​

이 섹션을 완료한 후 다음 내용을 계속 학습할 수 있습니다:

  • macOS 플랫폼 기능 - macOS 전용 포커스 감지 및 사운드 커스터마이징 기능
  • 구성 참조 - 모든 구성 옵션에 대한 심층 이해
  • 문제 해결 - 문제 발생 시 해결 방법
