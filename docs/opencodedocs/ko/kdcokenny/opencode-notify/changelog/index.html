업데이트 로그 ​

시간 역순으로 기록된 플러그인 변경 히스토리를 확인하세요.

포함 내용:

  • 기능 업데이트 기록
  • 문제 수정 설명
  • 설정 개선 상세 정보
  • 하위 호환성 변경 알림

버전 안내

이 플러그인은 OCX를 통해 배포되며 버전 번호 없이 Git 커밋 히스토리를 통해 변경 사항을 추적하는 전략을 사용합니다.

사전 요구사항 ​

사전 요구사항은 없으며 누구나 업데이트 로그를 확인할 수 있습니다.

다음 단계 ​

  • 빠른 시작 - 새로운 사용자라면 여기서 시작하는 것을 권장합니다
  • 작동 원리 - 플러그인의 핵심 기능과 알림 메커니즘을 알아보세요
  • 플랫폼 기능 - 사용 중인 플랫폼의 특수 기능을 확인하세요
  • 설정 참조 - 모든 설정 옵션에 대해 자세히 알아보세요
  • 자주 묻는 질문 - 일반적인 질문에 대한 답변을 빠르게 찾아보세요
