버전 정책 ​

이 플러그인은 OCX 생태계의 일부로, 다음 버전 정책을 따릅니다:

  • 버전 번호 없음: Git 커밋 히스토리를 통해 변경 사항 추적
  • 지속적 배포: OCX 메인 저장소와 동기화하여 업데이트
  • 하위 호환성: 설정 형식 및 API의 하위 호환성 유지

하위 호환성이 깨지는 변경이 있을 경우, 업데이트 로그에 명시적으로 표시됩니다.
