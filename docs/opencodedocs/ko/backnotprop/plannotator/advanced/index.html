고급 기능 ​

기본 계획 검토와 코드 검토를 익힌 후, 이 챕터에서는 Plannotator의 고급 기능을 활용하는 방법을 소개합니다: 팀 협업 공유, 노트 앱 통합, 원격 개발 지원, 그리고 유연한 환경 변수 설정.

사전 조건

이 챕터를 학습하기 전에 다음 사항을 완료했는지 확인하세요:

  • 빠른 시작: Plannotator의 기본 개념 이해
  • 계획 검토 기초: 기본적인 계획 검토 작업 마스터

이 챕터의 내용 ​

과정설명적합한 상황
URL 공유URL을 통해 계획과 주석을 공유하여 백엔드 없이 팀 협업 구현팀원들과 검토 결과를 공유해야 할 때
Obsidian 통합승인된 계획을 자동으로 Obsidian vault에 저장Obsidian으로 지식 베이스를 관리할 때
Bear 통합x-callback-url을 통해 계획을 Bear에 저장macOS/iOS 사용자가 Bear 노트를 사용할 때
원격 모드SSH, devcontainer, WSL 등 원격 환경에서 사용원격 개발 시나리오
환경 변수 설정사용 가능한 모든 환경 변수와 용도 이해Plannotator 동작을 커스터마이징해야 할 때

│  추천 학습 순서                                                    │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │

└─────────────────────────────────────────────────────────────────┘

필요에 따라 학습

모든 내용을 순서대로 학습할 필요는 없습니다. 공유 기능만 필요하다면 URL 공유만 학습하면 되고, 원격 개발을 사용하지 않는다면 원격 모드를 건너뛸 수 있습니다.
