plannotator
AI 계획 및 코드 리뷰 도구

Claude Code와 OpenCode를 위한 대화형 계획 리뷰 및 코드 리뷰 도구를 제공합니다. 시각적 주석, 팀 협업, URL 공유 및 서드파티 노트 앱 통합을 지원합니다. 로컬에서 실행되며 데이터 보안이 보장되고 서버 없이 리뷰 내용을 공유할 수 있습니다.

빠른 시작 GitHub
Plannotator를 선택해야 하는 이유?

AI가 생성한 계획과 코드를 시각적으로 리뷰하고, 정확한 피드백과 팀 협업을 경험하세요.

시각적 계획 리뷰

브라우저에서 AI가 생성한 계획을 보고 삭제, 교체, 삽입, 댓글 주석으로 정확한 피드백을 제공하세요.

시각적 코드 리뷰

Git diff side-by-side 또는 unified 뷰를 지원하며, 행 수준 주석과 코드 제안을 제공합니다.

다중 플랫폼 지원

Claude Code와 OpenCode를 지원하며 기존 개발 워크플로우에 완벽하게 통합됩니다.

URL 기반 백엔드 없는 공유

URL 해시에 압축하여 팀 구성원과 리뷰 내용을 서버 없이 공유할 수 있습니다.

이미지 주석 도구

이미지를 첨부하고 브러시, 화살표, 원형 도구로 주석을 추가하여 피드백 표현력을 높이세요.

서드파티 노트 통합

승인된 계획은 frontmatter와 태그와 함께 Obsidian 또는 Bear에 자동으로 저장됩니다.

빠른 시작

Claude Code 플러그인 설치

Claude Code에 Plannotator 플러그인을 설치하고 구성하세요.

OpenCode 플러그인 설치

OpenCode에 Plannotator 플러그인을 설치하고 구성하세요.

핵심 기능
계획 리뷰 기초

AI가 생성한 계획을 시각적으로 리뷰하고 승인하거나 거부하세요.

계획 주석 추가

삭제, 교체, 삽입, 댓글 등 다양한 유형의 주석을 추가하세요.

이미지 주석 추가

이미지를 첨부하고 브러시, 화살표, 원형 도구로 주석을 추가하세요.

코드 리뷰 기초

/plannotator-review 명령어를 사용하여 git diff를 리뷰하세요.

코드 주석 추가

행 수준 주석(comment/suggestion/concern)을 추가하세요.

Diff 뷰 전환

uncommitted/staged/last commit/branch 등 diff 유형을 전환하세요.

고급 기능
URL 공유

URL을 통해 계획과 주석을 공유하고 백엔드 없이 팀 협업을 수행하세요.

Obsidian 통합

승인된 계획을 Obsidian vault에 자동으로 저장하세요.

Bear 통합

x-callback-url을 통해 계획을 Bear에 저장하세요.

원격/Devcontainer 모드

SSH, devcontainer, WSL 환경에서 Plannotator를 사용하세요.

환경 변수 구성

사용 가능한 모든 환경 변수와 용도를 알아보세요.

자주 묻는 질문
자주 묻는 질문

포트 점유, 브라우저 열림 실패, 계획 표시 안됨 등 문제 해결.

문제 해결

로그 확인, 오류 처리 및 디버깅 팁.
