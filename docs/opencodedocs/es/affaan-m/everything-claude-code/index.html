Tema

Everything Claude Code
Colección de configuraciones de complementos de Claude Code probadas en combate

Colección completa de configuraciones de complementos de Claude Code del ganador del hackathon de Anthropic. Proporciona 9 agentes, 14 comandos, 8 conjuntos de reglas, 11 habilidades y más de 15 hooks para mejorar la eficiencia del desarrollo asistido por IA. Compatible con scripts de Node.js multiplataforma en Windows, macOS y Linux.

Inicio Rápido GitHub
¿Por qué elegir Everything Claude Code?

Sin necesidad de configuración desde cero. Obtén configuraciones probadas en combate durante 10+ meses de uso diario.

Probado en combate

Del ganador del hackathon de Anthropic, 10+ meses de uso diario, construyendo productos reales.

Configuración completa

9 agentes, 14 comandos, 8 conjuntos de reglas, 11 habilidades, más de 15 hooks, cubriendo todos los escenarios de desarrollo.

Compatibilidad multiplataforma

Todos los hooks y scripts reescritos en Node.js, ejecutándose sin problemas en Windows, macOS y Linux.

Gestor de paquetes inteligente

Detecta automáticamente npm, pnpm, yarn o bun, soporta 6 configuraciones de prioridad, sin necesidad de configuración manual.

Aprendizaje continuo

Extrae automáticamente patrones reutilizables de sesiones, guardándolos como habilidades aprendidas.

Garantía de calidad del código

Flujo TDD, revisión de código, auditoría de seguridad, asegurando más del 80% de cobertura de pruebas.
