Apéndice: Guía de Referencia y Compatibilidad ​

Este capítulo contiene materiales de referencia detallados y directrices de compatibilidad para oh-my-opencode, que cubre la documentación completa de configuración, el soporte de migración de Claude Code y presentaciones detalladas de los servidores de extensión integrados.

Contenido del Capítulo ​

Este capítulo incluye tres secciones importantes:

SubpáginaDescripciónDificultad
Compatibilidad con Claude CodeGuía completa para migrar desde Claude Code a OpenCode, incluyendo mecanismos de compatibilidad para Comandos, Habilidades, Agentes, MCPs y Hooks⭐⭐
Referencia de ConfiguraciónDocumentación completa del esquema para los archivos de configuración de oh-my-opencode, que cubre todos los campos, tipos y valores predeterminados⭐⭐⭐
MCPs IntegradosCaracterísticas y uso de 3 servidores MCP integrados (Exa Websearch, Context7, Grep.app)⭐⭐

Rutas de Aprendizaje Recomendadas ​

Elige tu orden de aprendizaje según tus necesidades:

Ruta 1: Migración desde Claude Code ​

Si eres un usuario que migra desde Claude Code:

  1. Primero lee Compatibilidad con Claude Code para entender cómo migrar sin problemas tu configuración existente
  2. Luego revisa Referencia de Configuración para obtener una comprensión más profunda de las opciones de configuración disponibles
  3. Finalmente aprende sobre MCPs Integrados para entender cómo configurar capacidades de búsqueda adicionales

Ruta 2: Personalización Profunda ​

Si deseas personalizar profundamente el comportamiento de oh-my-opencode:

  1. Comienza con Referencia de Configuración para entender todos los elementos configurables
  2. Aprende sobre MCPs Integrados para configurar capacidades de búsqueda y consulta de documentación
  3. Consulta Compatibilidad con Claude Code para entender las opciones de configuración de la capa de compatibilidad

Ruta 3: Referencia Rápida ​

Si solo necesitas buscar información específica:

  • Preguntas de configuración → Ve directamente a Referencia de Configuración
  • Preguntas de migración → Ve directamente a Compatibilidad con Claude Code
  • Configuración de MCP → Ve directamente a MCPs Integrados

Prerrequisitos ​

Para este capítulo, se recomienda que ya hayas:

  • ✅ Completado Instalación y Configuración
  • ✅ Entendido los conceptos básicos de Orquestador Sisyphus
  • ✅ Familiaridad con la edición de archivos de configuración JSON

Pasos Siguientes ​

Después de completar este capítulo, puedes:

  • 🚀 Probar Características Avanzadas para aprender un uso más avanzado
  • 🔧 Consultar Preguntas Frecuentes para resolver problemas de uso
  • 📖 Leer Registro de Cambios para mantenerte actualizado sobre las mejoras de nuevas características

Consejo

El contenido de este capítulo sirve principalmente como material de referencia y no necesita leerse secuencialmente. Puedes consultar las secciones relevantes según sea necesario.
