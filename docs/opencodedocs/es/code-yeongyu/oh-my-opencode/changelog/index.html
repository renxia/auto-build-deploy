Registro de Cambios: Evolución de Versiones y Seguimiento de Características ​

Resumen del Capítulo ​

Este capítulo registra el historial de versiones de oh-my-opencode, incluyendo nuevas características, mejoras importantes, optimizaciones funcionales y cambios de compatibilidad. Al leer el registro de cambios, puedes entender el proceso de evolución del proyecto y dominar rápidamente las mejoras funcionales de la última versión.

Recomendaciones de Ruta de Aprendizaje ​

El registro de cambios de este capítulo está ordenado de versión más reciente a más antigua. Se recomienda leer de la siguiente manera:

  • 🎯 Conocer rápidamente las nuevas funciones: Lee directamente el registro de cambios de la última versión (como v3.0)
  • 📚 Guía de actualización de versiones: Comienza desde la versión actual, lee en orden inverso para entender la ruta de actualización
  • 🔍 Explorar en profundidad nuevas características: Lee la documentación detallada de la versión de interés, combinada con el capítulo correspondiente del tutorial avanzado

Requisitos Previos ​

Para leer el registro de cambios de este capítulo, se recomienda que hayas completado:

  • ✅ Instalación y Configuración Rápida - Entiende la instalación y configuración básica de oh-my-opencode
  • ✅ Primeros Pasos con Sisyphus: Orquestador Principal - Comprende los conceptos básicos y métodos de uso del agente Sisyphus

Siguientes Pasos ​

Después de leer el registro de cambios, puedes continuar aprendiendo:

  • Aprender en profundidad nuevas características:

    • Categories y Skills: Composición Dinámica de Agentes - Aprende el uso detallado del nuevo sistema v3.0
    • Skills Integrados: Automatización del Navegador y Experto en Git - Domina el uso de 3 paquetes de habilidades profesionales integrados
    • Personalización Profunda de Configuración: Gestión de Agentes y Permisos - Entiende cómo personalizar Categories y Skills

  • Explorar funciones avanzadas:

    • Equipo de Agentes de IA: Introducción a 10 Expertos - Conoce en detalle los 10 agentes integrados
    • Planificación Prometheus: Recopilación de Requisitos Entrevistadora - Aprende a usar el agente Prometheus para la planificación de proyectos
    • Tareas Paralelas en Segundo Plano: Trabajar como un Equipo - Domina el uso del sistema de gestión de agentes en segundo plano
