Tema

plannotator
Herramienta IA de Revisión de Planes y Código

Proporciona herramientas interactivas de revisión de planes y código para Claude Code y OpenCode, admite anotaciones visuales, colaboración en equipo, uso compartido de URL e integración con aplicaciones de notas de terceros. Funciona localmente, datos seguros, comparte revisiones sin servidor.

Inicio Rápido GitHub
¿Por qué elegir Plannotator?

Revisa visualmente planes y código generados por IA, retroalimentación precisa, colaboración en equipo.

Revisión Visual de Planes

Visualiza planes generados por IA en el navegador, proporciona retroalimentación precisa con anotaciones de eliminación, reemplazo, inserción y comentarios.

Visualización de Revisión de Código

Vista side-by-side o unificada de Git diff, admite anotaciones a nivel de línea y sugerencias de código.

Soporte Multiplataforma

Admite Claude Code y OpenCode, integración perfecta con el flujo de desarrollo existente.

Uso Compartido de URL Sin Backend

Comprimido en hash de URL, comparte revisiones con miembros del equipo sin servidor.

Herramientas de Anotación de Imágenes

Adjunta imágenes y usa herramientas de pincel, flecha y círculo para anotar, mejorando la expresión de retroalimentación.

Integración con Notas de Terceros

Los planes aprobados se guardan automáticamente en Obsidian o Bear, incluyendo frontmatter y etiquetas.
