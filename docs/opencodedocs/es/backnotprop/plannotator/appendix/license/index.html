Explicación de la Licencia de Plannotator ​

Lo que Aprenderás ​

  • Comprender el tipo de licencia y los términos principales que utiliza Plannotator
  • Identificar claramente qué escenarios de uso están permitidos y cuáles están restringidos
  • Conocer el calendario de cambios futuros de la licencia
  • Saber cómo obtener una licencia comercial o contactar al licenciante

Lo que Necesitas Saber Ahora ​

Plannotator adopta la Business Source License 1.1 (BSL) como su licencia de código abierto. Esta es una licencia especial diseñada para proteger el valor comercial mientras eventualmente hace que el proyecto sea completamente de código abierto.

¿Qué es BSL?

Business Source License es una licencia especial desarrollada por MariaDB. Restringe ciertos usos comerciales a corto plazo, pero promete convertir el software en una licencia completamente de código abierto (como Apache 2.0) en una fecha específica. Este modelo permite que los proyectos de código abierto obtengan ingresos sostenibles durante el proceso de comercialización, al tiempo que garantiza la apertura a largo plazo para la comunidad.

Usos Permitidos ​

Según los términos de BSL 1.1, puedes:

  • ✅ Copiar, modificar y crear obras derivadas

    • Puedes realizar desarrollo secundario basado en Plannotator
    • Puedes modificar el código fuente para adaptarlo a tus necesidades

  • ✅ Redistribuir

    • Puedes distribuir versiones originales o modificadas
    • Debes conservar la declaración de licencia original

  • ✅ Uso en Entornos No Productivos

    • Desarrollo interno, pruebas y evaluación
    • Aprendizaje personal e investigación

Uso Extensivo Personal y Empresarial Interno

BSL define "uso en entornos no productivos" de manera amplia, incluyendo varios escenarios personales y dentro de la empresa. Tú y los miembros de tu equipo (incluidos contratistas independientes) pueden usar Plannotator libremente para desarrollo, pruebas y despliegue.

Restricciones de Uso (¡Importante!) ​

Restricción Principal: No se permite usar Plannotator para servicios de Planificación, Contexto o Aprobación (Planning, Context, or Approval Service).

¿Qué son los "Servicios de Planificación, Contexto o Aprobación"? ​

Cualquier producto comercial que permita a terceros (distintos de empleados y contratistas independientes) acceder a las funciones de Plannotator mediante operaciones directas o indirectas se considera un servicio restringido. Esto incluye específicamente, pero no se limita a:

  • Proveedores de Servicios en la Nube

    • Plataformas SaaS que ofrecen funciones de Plannotator como parte de sus servicios
    • Proveedores de servicios gestionados que ofrecen Plannotator a clientes o suscriptores

  • Servicios de Infraestructura

    • Proveedores de servicios de centros de datos
    • Proveedores de plataformas de computación en la nube

  • Terceros con Servicios Similares

    • Incluye cualquier servicio proporcionado por las entidades anteriores y sus empresas afiliadas

Advertencia de Uso Comercial

Si planeas ofrecer Plannotator como parte de un servicio comercial para uso de terceros, debes obtener una licencia comercial con anticipación. La violación de este término terminará automáticamente tus derechos de uso de Plannotator en la versión actual y todas las versiones futuras.

Calendario de Cambio de Licencia ​

¿Qué Sucederá Después del Cambio? ​

A partir del 1 de enero de 2030, Plannotator se convertirá automáticamente a Apache License 2.0. Esto significa:

  • ✅ Todas las restricciones comerciales se eliminarán
  • ✅ Se podrá usar libremente para cualquier propósito comercial
  • ✅ Se podrá integrar en cualquier servicio SaaS
  • ✅ Cumple con los estándares de licencias de código abierto permisivas

Licencia Comercial ​

Si necesitas usar Plannotator sin cumplir con las restricciones de BSL (por ejemplo, para servicios de planificación, contexto o aprobación), puedes:

  1. Comprar una Licencia Comercial

    • Adquirir de backnotprop o sus distribuidores autorizados
    • Obtener derechos de uso sin las restricciones de BSL

  2. Contactar al Licenciante

    • Enviar correo a: backnotprop
    • Explicar tu escenario de uso y necesidades

Preguntas Frecuentes ​

¿Puedo usar Plannotator dentro de mi empresa? ​

Sí. BSL permite explícitamente que tú y tus empleados, contratistas independientes usen Plannotator para desarrollo interno, pruebas y despliegue. La restricción solo se aplica a escenarios donde se proporcionan servicios a terceros (no empleados ni contratistas).

¿Puedo desarrollar mi propio producto basado en Plannotator? ​

Sí, con condiciones. Si tu producto es:

  • Herramienta interna: ✅ Permitido
  • Proyecto personal: ✅ Permitido
  • Proyecto de código abierto: ✅ Permitido (conservando la licencia BSL)
  • Producto comercial: ❓ Se requiere evaluar si pertenece a "servicios de planificación, contexto o aprobación"

Mi empresa es un proveedor SaaS, ¿puede usarlo? ​

Se requiere licencia comercial. Si tu producto SaaS ofrece las funciones de Plannotator a clientes para su uso (incluso como parte de las funcionalidades), esto se considera un servicio comercial restringido y debes adquirir una licencia comercial.

¿Podré usar Plannotator para cualquier propósito después de 2030? ​

Sí. A partir de la fecha de cambio, Plannotador adoptará la licencia Apache 2.0, todas las restricciones comerciales se eliminarán y podrás usarlo libremente para cualquier propósito.
