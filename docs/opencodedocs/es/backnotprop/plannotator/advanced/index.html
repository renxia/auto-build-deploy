Funcionalidades Avanzadas ​

Después de dominar las revisiones de planes y código básicas, este capítulo te llevará a desbloquear las capacidades avanzadas de Plannotator: compartir colaborativo en equipo, integración de aplicaciones de notas, soporte para desarrollo remoto y configuración flexible de variables de entorno.

Requisitos Previos

Antes de comenzar este capítulo, asegúrate de haber completado:

  • Inicio Rápido: Comprender los conceptos básicos de Plannotator
  • Fundamentos de Revisión de Planes: Dominar las operaciones básicas de revisión de planes

Contenido del Capítulo ​

CursoDescripciónCasos de Uso Adecuados
Compartir por URLCompartir planes y comentarios a través de URL para lograr colaboración en equipo sin backendNecesidad de compartir resultados de revisión con miembros del equipo
Integración con ObsidianGuardar automáticamente los planes aprobados en el vault de ObsidianUsar Obsidian para gestionar la base de conocimiento
Integración con BearGuardar planes en Bear a través de x-callback-urlUsuarios de macOS/iOS que usan Bear para notas
Modo RemotoUsar en entornos remotos como SSH, devcontainer, WSLEscenarios de desarrollo remoto
Configuración de Variables de EntornoConocer todas las variables de entorno disponibles y sus usosNecesidad de personalizar el comportamiento de Plannotator

Aprendizaje Bajo Demanda

No es necesario completar todo el contenido en orden. Si solo necesitas la función de compartir, con aprender Compartir por URL es suficiente; si no usas desarrollo remoto, puedes omitir el Modo Remoto.

Próximos Pasos ​

Después de completar el aprendizaje de funcionalidades avanzadas, puedes:

  • Ver Problemas Comunes: Resolver problemas encontrados durante el uso
  • Ver Referencia de API: Conocer la API completa de Plannotator
  • Ver Modelos de Datos: Comprender profundamente las estructuras de datos internas
