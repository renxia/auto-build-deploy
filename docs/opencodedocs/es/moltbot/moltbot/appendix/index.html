Apéndice ​

Este capítulo proporciona documentación de referencia avanzada y recursos de desarrollo para Clawdbot, incluyendo la referencia completa de configuración, especificaciones del protocolo API WebSocket de Gateway, opciones de despliegue y guías de desarrollo.

Escenarios de uso

Este capítulo es adecuado para usuarios que necesitan comprender profundamente los mecanismos internos de Clawdbot, realizar configuraciones avanzadas, despliegues o participar en el desarrollo. Si estás empezando, te recomendamos completar primero el capítulo Inicio rápido.

Navegación de subpáginas ​

Referencia de configuración completa ​

Referencia detallada del archivo de configuración - Cubre todos los elementos de configuración, valores predeterminados y ejemplos. Busca explicaciones completas de configuración para módulos como Gateway, Agent, canales y herramientas.

Protocolo API WebSocket de Gateway ​

Documento de especificación del protocolo - Especificaciones completas del protocolo WebSocket de Gateway, incluyendo definiciones de endpoints, formatos de mensajes, métodos de autenticación y mecanismos de suscripción a eventos. Adecuado para desarrolladores que necesitan personalizar clientes o integrar Gateway.

Opciones de despliegue ​

Guía de métodos de despliegue - Métodos de despliegue en diferentes plataformas: instalación local, Docker, VPS, Fly.io, Nix, etc. Aprende a ejecutar Clawdbot en varios entornos.

Guía de desarrollo ​

Documentación para desarrolladores - Construcción desde código fuente, desarrollo de plugins, pruebas y proceso de contribuciones. Aprende cómo participar en el desarrollo del proyecto Clawdbot o escribir plugins personalizados.

Recomendaciones de ruta de aprendizaje ​

Según tus necesidades, elige la ruta de aprendizaje adecuada:

Personal de configuración y operaciones ​

  1. Primero lee la Referencia de configuración completa - Comprende todos los elementos configurables
  2. Consulta las Opciones de despliegue - Elige el esquema de despliegue adecuado
  3. Según sea necesario, consulta la documentación de la API WebSocket de Gateway para la integración

Desarrolladores de aplicaciones ​

  1. Lee el Protocolo API WebSocket de Gateway - Comprende los mecanismos del protocolo
  2. Consulta la Referencia de configuración completa - Comprende cómo configurar las funcionalidades relacionadas
  3. Construye el cliente basándote en ejemplos del protocolo

Desarrolladores de plugins/funcionalidades ​

  1. Lee la Guía de desarrollo - Comprende el entorno de desarrollo y el proceso de construcción
  2. Profundiza en el Protocolo API WebSocket de Gateway - Comprende la arquitectura de Gateway
  3. Consulta la Referencia de configuración completa - Comprende el sistema de configuración

Prerrequisitos ​

Conocimientos previos

Antes de profundizar en este capítulo, se recomienda que hayas completado lo siguiente:

  • ✅ Completado Inicio rápido
  • ✅ Configurado al menos un canal (como WhatsApp o Telegram)
  • ✅ Comprendido la configuración básica de modelos de AI (consulta Modelos de AI y autenticación)
  • ✅ Conocimiento básico de archivos de configuración JSON y TypeScript

Siguientes pasos ​

Después de completar el estudio de este capítulo, puedes:

  • Realizar configuraciones avanzadas - Consulta la Referencia de configuración completa para personalizar tu Clawdbot
  • Desplegar en producción - Según las Opciones de despliegue, elige el esquema de despliegue adecuado
  • Desarrollar funcionalidades personalizadas - Consulta la Guía de desarrollo para escribir plugins o contribuir código
  • Profundizar en otras funcionalidades - Explora el capítulo Funciones avanzadas, como gestión de sesiones, sistema de herramientas, etc.

Buscar ayuda

Si encuentras problemas durante el uso, puedes consultar Solución de problemas para obtener soluciones a problemas comunes.
