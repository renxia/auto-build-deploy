Preguntas Frecuentes y Optimización ​

Este capítulo te ayuda a resolver varios problemas al usar Clawdbot, dominar los métodos completos de operación de línea de comandos y aprender cómo optimizar el rendimiento del sistema.

Recomendación de Orden de Aprendizaje

Se recomienda leer este capítulo en el siguiente orden:

  1. Primero aprende Referencia de Comandos CLI —— Domina las operaciones básicas y herramientas de depuración
  2. Luego aprende Solución de Problemas —— Para localizar y resolver problemas rápidamente cuando ocurren
  3. Finalmente aprende Optimización de Rendimiento —— Para mejorar la eficiencia del sistema

Navegación de Subpáginas ​

Solución de Problemas ​

Una guía completa de solución de problemas para problemas comunes, que cubre:

  • Fallos de inicio de Gateway
  • Problemas de conexión de canales (WhatsApp, Telegram, Slack, etc.)
  • Errores de autenticación y problemas de permisos
  • Fallos en la invocación de herramientas
  • Excepciones en la gestión de sesiones

Cuándo usar: Cuando encuentras problemas específicos y necesitas solucionarlos y resolverlos rápidamente.

Referencia de Comandos CLI ​

Lista completa de comandos e instrucciones de uso, que incluye:

  • Comandos de control de Gateway (inicio, reinicio, estado)
  • Comandos de interacción con Agent
  • Gestión y configuración de modelos
  • Gestión de canales (inicio de sesión, estado, cierre de sesión)
  • Gestión de sesiones (lista, restablecimiento, eliminación, compresión)
  • Gestión de nodos (lista, invocación, emparejamiento)
  • Gestión de habilidades y asistente de configuración

Cuándo usar: Cuando necesitas encontrar rápidamente el uso y parámetros de un comando, o aprender la funcionalidad completa de CLI.

Optimización de Rendimiento ​

Técnicas de optimización para mejorar el rendimiento y la velocidad de respuesta de Gateway, que cubre:

  • Caché de Prompts (Prompt Caching) para reducir costos
  • Poda de Sesiones (Session Pruning) para reducir contexto
  • Control de concurrencia y gestión de recursos
  • Optimización de memoria y recolección de basura
  • Ajuste de niveles de registro

Cuándo usar: Cuando el sistema se ejecuta lentamente, el uso de recursos es alto, o deseas mejorar la experiencia del usuario.

Requisitos Previos ​

Antes de aprender este capítulo, se recomienda que hayas completado:

  • ✅ Inicio Rápido —— Comprender los procesos básicos de instalación y configuración
  • ✅ Iniciar Gateway —— Saber cómo iniciar y gestionar Gateway

Próximos Pasos ​

Después de completar este capítulo, puedes continuar aprendiendo:

  • Referencia Completa de Configuración —— Comprender en profundidad las explicaciones detalladas de todas las opciones de configuración
  • Opciones de Despliegue —— Aprender cómo desplegar Clawdbot en diferentes plataformas
  • Guía de Desarrollo —— Comprender el desarrollo de complementos y el proceso de contribución
