Plataformas e Integraciones ​

Este capítulo te ayuda a configurar el entorno de desarrollo de IA e instalar los complementos necesarios, asegurando que el flujo de trabajo funcione correctamente.

Ruta de Aprendizaje ​

1. Guía de Integración con Claude Code ​

Aprende a usar Claude Code para ejecutar el flujo de trabajo, incluyendo la configuración de permisos y mejores prácticas.

  • Configurar permisos de Claude Code
  • Comprender las limitaciones de acceso a archivos
  • Dominar los pasos para ejecutar el flujo de trabajo

2. OpenCode y Otros Asistentes de IA ​

Conoce cómo usar OpenCode, Cursor y otros asistentes de IA para ejecutar el flujo de trabajo.

  • Configurar OpenCode CLI
  • Configurar proyectos de Cursor
  • Notas de compatibilidad

3. Instalación de Complementos Necesarios ​

Instala los complementos superpowers y ui-ux-pro-max para asegurar que el flujo de trabajo funcione correctamente.

  • Instalar el complemento superpowers
  • Instalar el complemento ui-ux-pro-max
  • Verificar la disponibilidad de los complementos

Requisitos Previos ​

Antes de comenzar con este capítulo, asegúrate de haber completado:

Requisitos Previos

  • Has completado Inicio Rápido
  • Has completado Instalación y Configuración
  • Has completado Inicializar Proyecto Factory

Siguiente Pasos ​

Después de completar este capítulo, te recomendamos continuar con:

  • Visión General del Flujo de Trabajo de 7 Etapas - Conoce el flujo de trabajo completo
  • Etapa 1: Bootstrap - Aprende en profundidad los detalles de implementación de cada etapa
