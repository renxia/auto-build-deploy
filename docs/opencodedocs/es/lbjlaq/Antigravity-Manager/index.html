Tema

Antigravity-Manager
Sistema de Gestión de Cuentas de IA Local y Proxy Inverso de Protocolos

Convierte identidades Web/Session/OAuth de múltiples plataformas de IA en un gateway de API unificado, proporcionando capacidades de programación de múltiples cuentas, rotación inteligente, monitoreo de cuotas y compatibilidad multiprotocolo. Aplicación de escritorio basada en Tauri v2 + React + Rust.

Inicio Rápido GitHub
¿Por qué elegir Antigravity-Manager?

Tu gateway de programación de IA de alto rendimiento local. No es solo gestión de cuentas, es la solución definitiva para romper las barreras de invocación de API.

Endpoints Compatibles Multiprotocolo

Expone endpoints OpenAI /v1/chat/completions, Anthropic /v1/messages, SDK nativo Gemini para reutilizar clientes y SDK existentes.

Programación Inteligente de Cuentas

Compatible con rotación, cuenta fija, sesión pegajosa y reintento de fallos. Rotación automática en 429/401, desconexión automática en invalid_grant, auto-recuperación a nivel de milisegundos.

Adición Mediante Autorización OAuth

Compatible con autorización OAuth en navegador y Refresh Token para agregar cuentas en doble canal, importación y exportación por lotes, migración en caliente V1/DB.

Gobernanza y Protección de Cuotas

Monitoreo en tiempo real de cuotas restantes de cada modelo, compatible con protección de cuotas a nivel de modelo y Smart Warmup para evitar agotamiento inadvertido.

Monitoreo y Estadísticas de Tráfico

Proxy Monitor registra registros de solicitudes en tiempo real, Token Stats estadiza consumo por hora/día/semana, compatible con análisis por modelo y cuenta.

Despliegue Multiplataforma

Soporte nativo para escritorio macOS, Windows, Linux. Proporciona soluciones Docker noVNC y Headless Xvbf, adaptado para despliegue NAS/servidores.
