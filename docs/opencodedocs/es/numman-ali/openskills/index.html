Tema

openskills
Cargador de habilidades universal para agentes de codificación AI

Permite que cualquier agente de codificación AI (Claude Code, Cursor, Windsurf, Aider, etc.) utilice habilidades en formato estándar SKILL.md. Unifica estándares, carga progresiva, soporte multi-agente.

Comienza Rápido GitHub
¿Por qué OpenSkills?

Un formato de habilidades unificado, disponible en cualquier agente, eliminando el bloqueo de plataforma.

Totalmente Compatible con Claude Code

Mismo formato de prompts, mismo marketplace, misma estructura de carpetas, migración sin fricciones.

Soporte Multi-Agente

Admite Claude Code, Cursor, Windsurf, Aider, Codex y cualquier agente que pueda leer AGENTS.md.

Carga Progresiva

Carga habilidades bajo demanda, manteniendo el contexto del agente limpio y enfocado, evitando la contaminación de contexto.

Fuentes de Instalación Flexibles

Admite la instalación de habilidades desde GitHub, rutas locales y repositorios git privados, adaptándose a diversos escenarios de uso.

Modo Universal

Un conjunto de habilidades sirve a múltiples agentes, evitando conflictos con Claude Code, gestión de habilidades unificada.

Ejecución 100% Local

La herramienta CLI no requiere servidor, todas las operaciones se completan en el sistema de archivos local, garantizando la seguridad de la privacidad.
