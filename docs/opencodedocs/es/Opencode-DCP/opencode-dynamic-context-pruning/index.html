Tema

opencode-dynamic-context-pruning
Plugin de Optimización Inteligente de Tokens

Plugin de OpenCode que reduce automáticamente el uso de Tokens eliminando llamadas redundantes de herramientas en el historial de conversaciones. Soporta tres estrategias: deduplicación automática, detección de sobrescritura y limpieza de errores, además de herramientas discard/extract impulsadas por IA, permitiendo que el modelo decida cuándo limpiar el contexto.

Inicio Rápido GitHub
¿Por qué elegir DCP?

Ahorre una gran cantidad de Tokens en conversaciones largas, reduzca los costos de LLM y mejore la calidad de las respuestas.

Estrategia de Deduplicación Automática

Detecta llamadas con el mismo nombre y parámetros de herramienta, conservando automáticamente la más reciente sin intervención manual.

Detección de Sobrescritura

Rastrea el orden temporal de operaciones de escritura y lectura en archivos, limpiando automáticamente las escrituras que han sido sobrescritas por lecturas posteriores.

Estrategia de Limpieza de Errores

Recorta automáticamente las entradas de herramientas erróneas que exceden el número de rondas especificado, manteniendo los mensajes de error para depuración.

Recorte Impulsado por IA

Proporciona herramientas discard y extract, permitiendo que la IA decida autónomamente cuándo limpiar el contexto para lograr optimización a nivel semántico.

Transparente y Controlable

Use el comando /dcp para ver el uso de Tokens, estadísticas acumuladas y admitir recorte manual activado.

Persistencia de Estado

Mantiene el estado de recorte y las estadísticas entre sesiones para rastrear el ahorro de Tokens a largo plazo.
