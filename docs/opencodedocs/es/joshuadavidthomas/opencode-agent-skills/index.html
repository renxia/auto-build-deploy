opencode-agent-skills
Plugin de gestión dinámica de Skills para OpenCode AI

Sistema de plugins que proporciona capacidades de descubrimiento y gestión dinámica de skills para OpenCode, permitiendo que los AI Agent carguen y utilicen bibliotecas de skills reutilizables. Logra la reutilización de skills entre proyectos y sesiones, recomendando automáticamente skills relacionados basados en similitud semántica.

Comenzar ahora GitHub
¿Por qué elegir OpenCode Agent Skills?

Proporciona capacidades de descubrimiento y gestión dinámica de skills para OpenCode, logrando la reutilización de skills entre proyectos y sesiones.

Descubrimiento dinámico de skills

Descubre automáticamente skills desde proyectos, usuarios y directorios de plugins, escaneando 6 ubicaciones y devolviendo el primer skill coincidente según prioridad.

Compatible con Claude Code

Admite los formatos de skills y plugins existentes de Claude Code, reduciendo los costos de migración y reutilizando recursos existentes.

Recomendación de coincidencia semántica

Utiliza modelos de embedding de HuggingFace para calcular similitudes, recomendando automáticamente skills relacionados basados en la entrada del usuario.

Inyección de contexto

El contenido del skill se inyecta en el contexto de la sesión en formato XML, permitiendo que la IA siga las instrucciones del skill para generar resultados de alta calidad.

Recuperación tras compresión

Los skills se reinyectan automáticamente después de la compresión de sesión, manteniendo la disponibilidad de skills durante sesiones largas.

Soporte de ejecución de scripts

Los directorios de skills pueden contener scripts ejecutables para realizar tareas de automatización en el contexto del directorio del skill.
