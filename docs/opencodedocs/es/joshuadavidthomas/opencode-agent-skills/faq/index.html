Preguntas Frecuentes ​

Este capítulo te ayuda a resolver los problemas que encuentres al usar el plugin OpenCode Agent Skills y a comprender los mecanismos de seguridad del plugin. Ya sea fallos en la carga de habilidades, errores de ejecución de scripts, o si quieres entender cómo el plugin protege tu sistema, aquí encontrarás las respuestas.

Contenido del Capítulo ​

🔧 Solución de Problemas Comunes

Métodos de solución y respuestas para 9 tipos de problemas comunes: fallos en la carga de habilidades, errores de ejecución de scripts, problemas de seguridad de rutas, fallos en la carga de modelos, entre otros.

🔒 Consideraciones de Seguridad

Descubre los mecanismos centrales de seguridad y las mejores prácticas del plugin: verificación de seguridad de rutas, análisis seguro de YAML, validación de entradas, protección de ejecución de scripts, entre otros.

Ruta de Aprendizaje ​

Se recomienda seguir este orden de aprendizaje:

Solución de Problemas Comunes → Consideraciones de Seguridad
  1. Primero la solución de problemas: Cuando encuentres problemas, consulta primero "Solución de Problemas Comunes" para identificar y resolver rápidamente
  2. Luego la seguridad: Comprende los mecanismos de seguridad del plugin para desarrollar hábitos de uso seguro

¿Qué hacer si tienes problemas

Si encuentras un problema específico al usar el plugin, salta directamente a "Solución de Problemas Comunes" para buscar los síntomas correspondientes. Si deseas comprender sistemáticamente el diseño de seguridad del plugin, lee "Consideraciones de Seguridad".

Prerrequisitos ​

Se recomienda completar primero los siguientes capítulos

  • Guía de Inicio: Comprende los conceptos básicos y los métodos de instalación del plugin
  • Plataformas e Integraciones: Familiarízate con las funciones principales como descubrimiento, carga y ejecución de habilidades

El contenido de este capítulo asume que ya tienes un conocimiento básico del plugin. Si es la primera vez que lo usas, se recomienda completar primero el aprendizaje de los capítulos mencionados anteriormente.

Siguiente Paso ​

Después de completar este capítulo, puedes continuar explorando:

  • Apéndice: Consulta las mejores prácticas de desarrollo de habilidades y las referencias de herramientas de API
  • Registro de Cambios: Conoce la evolución de versiones y las nuevas funciones del plugin
