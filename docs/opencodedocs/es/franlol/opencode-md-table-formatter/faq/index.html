Preguntas Frecuentes ​

Resumen del Capítulo ​

Este capítulo cubre las preguntas frecuentes y los métodos de solución de problemas del plugin opencode-md-table-formatter, ayudándote a identificar y resolver rápidamente varios problemas que encuentres durante el proceso de formato de tablas.

Navegación de Subpáginas ​

Preguntas Frecuentes: ¿Qué hacer si las tablas no se formatean ​

Presentación completa de los problemas comunes y soluciones en el uso del plugin:

  • Tablas no formateadas: Verificar configuración, requisitos de versión, reiniciar OpenCode
  • Estructura de tabla inválida: Comprender el significado del error "invalid structure", aprender la sintaxis correcta de tablas
  • Fallo en el formateo: Manejar el error "table formatting failed" y situaciones anormales
  • Tipos de tablas no compatibles: Conocer las limitaciones de las tablas HTML, celdas multilínea, etc.
  • Desalineación después del formateo: Investigar problemas como modo oculto, celdas demasiado largas, código en línea, etc.

Recomendaciones de Ruta de Aprendizaje ​

Se recomienda consultar este capítulo cuando encuentres problemas reales después de completar Visión General de Funciones, o usarlo como manual de referencia para la solución de problemas.

Puedes seguir el siguiente orden de aprendizaje:

  1. Cuando encuentres problemas: Primero identifica el tipo de problema según los síntomas (tablas no formateadas, mensajes de error, etc.)
  2. Investigar paso a paso: Verifica uno por uno según la lista de verificación de la guía de solución de problemas
  3. Comprensión profunda: Lee Especificaciones de Tablas y Principios del Modo Oculto para conocer los detalles técnicos
  4. Limitaciones conocidas: Consulta Limitaciones Conocidas para conocer los límites del plugin

Requisitos Previos ​

Antes de estudiar este capítulo, se recomienda completar primero:

  • ✅ Comenzar en un Minuto: Instalación y Configuración - Conocer cómo instalar y configurar el plugin
  • ✅ Visión General de Funciones: La Magia del Formateo Automático - Comprender las funciones principales y el principio de funcionamiento del plugin

Además, deberías tener:

  • Familiaridad con la sintaxis básica de tablas Markdown
  • Conocimiento del método de configuración de plugins de OpenCode (opencode.jsonc)

Próximos Pasos ​

Después de completar este capítulo, puedes continuar aprendiendo:

  • Conocimientos Avanzados:

    • Especificaciones de Tablas: Qué tablas pueden ser formateadas - Dominar los requisitos de estructura de tablas válidas
    • Principios del Modo Oculto: Por qué el cálculo de ancho es tan importante - Comprender el principio de funcionamiento del modo oculto de OpenCode
    • Explicación de Alineación: Izquierda, Centro, Derecha - Dominar la sintaxis y los efectos de los tres métodos de alineación

  • Detalles Técnicos:

    • Limitaciones Conocidas: ¿Dónde están los límites del plugin - Conocer las limitaciones conocidas y los escenarios no aplicables del plugin
    • Detalles Técnicos: Mecanismo de Caché y Optimización de Rendimiento - Conocer el mecanismo de caché interno y las estrategias de optimización de rendimiento del plugin
