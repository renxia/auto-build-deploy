opencode-md-table-formatter
Alineación Automática de Tablas con IA

Plugin de OpenCode que formatea automáticamente tablas Markdown generadas por IA. Reconoce inteligentemente el ancho del contenido, soporta formatos como negrita, cursiva, bloques de código, y mantiene las tablas siempre ordenadas y estéticas.

Ver Tutorial GitHub
Funciones Principales

Reconoce automáticamente la sintaxis Markdown, calcula inteligentemente el ancho de visualización

Reconocimiento Inteligente de Formato Markdown

Calcula correctamente el ancho de visualización de formatos como negrita, cursiva, bloques de código, asegurando que las tablas siempre estén perfectamente alineadas

Formateo Automático Después de la Generación por IA

Se activa automáticamente a través del hook experimental.text.complete, formateando después de que la IA genera texto sin necesidad de operación manual

Protección Inteligente de Bloques de Código

Protege los símbolos Markdown dentro de bloques de código en línea para que no se eliminen incorrectamente, asegurando que la sintaxis del código permanezca intacta

Soporte para Tres Métodos de Alineación

Soporta tres formatos: alineación izquierda (:---), alineación central (:---:) y alineación derecha (---:)

Procesamiento de Markdown Anidado

Utiliza un algoritmo de expresiones regulares de múltiples pasadas para procesar sintaxis Markdown anidada como negrita, cursiva, texto tachado, etc.

Compatibilidad con Casos Límite

Maneja correctamente casos límite como emojis, caracteres Unicode, celdas vacías, contenido largo, etc.

Ejecución Silenciosa

Sin salida de registros de consola, los errores no interrumpen el flujo de trabajo, completa el formateo silenciosamente

Comentarios de Validación

Agrega comentarios de error amigables para tablas inválidas, ayudando a los usuarios a localizar rápidamente los problemas
