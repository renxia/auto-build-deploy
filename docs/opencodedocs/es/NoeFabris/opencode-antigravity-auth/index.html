Antigravity Auth
Herramienta de autenticación para Claude & Gemini

Accede a Claude Opus 4.5, Sonnet 4.5 y Gemini 3 Pro/Flash directamente mediante credenciales OAuth de Google, sin necesidad de API Key. Soporta balanceo de carga multi-cuenta y recuperación automática de sesiones.

Inicio rápido GitHub
¿Por qué elegir Antigravity Auth?

Resuelve las molestias de configurar API Keys manualmente, gestiona automáticamente los quotas, diseñado específicamente para desarrollo de alta concurrencia.

Sin API Key

No necesitas solicitar costosas API Keys, usa directamente las credenciales de tu cuenta Google con autorización de un solo clic.

Balanceo de carga multi-cuenta

Soporta hasta 10 cuentas Google con rotación automática, distribución inteligente de solicitudes, despidiéndose permanentemente de los límites de tasa.

Soporte de modelos Thinking

Soporte perfecto para el modo thinking de Claude y Gemini 3, configurable Budget, desbloqueando capacidades de razonamiento profundo.

Recuperación automática de sesiones

¿Fallo en la llamada de herramienta? ¿Interrupción de sesión? El plugin reintenta automáticamente y recupera el contexto sin intervención humana.

Compatibilidad con plugins

Integración perfecta con plugins populares como oh-my-opencode, DCP.

Sistema de doble quota

Maximiza la utilización de las pools de quotas gratuitas y pagadas proporcionadas por Google.
