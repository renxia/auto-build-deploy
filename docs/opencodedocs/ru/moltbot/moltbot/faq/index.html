Часто задаваемые вопросы и оптимизация ​

Эта глава помогает вам решать различные проблемы при использовании Clawdbot, осваивать полные методы работы с командной строкой и учиться оптимизировать производительность системы.

Рекомендация по порядку обучения

Рекомендуется читать эту главу в следующем порядке:

  1. Сначала изучите Справка по командам CLI —— Освойте базовые операции и инструменты отладки
  2. Затем изучите Устранение неполадок —— Чтобы быстро выявлять и решать проблемы при их возникновении
  3. Наконец изучите Оптимизация производительности —— Чтобы повысить эффективность системы

Навигация по подстраницам ​

Устранение неполадок ​

Полное руководство по устранению неполадок для распространённых проблем, охватывающее:

  • Сбои при запуске Gateway
  • Проблемы с подключением каналов (WhatsApp, Telegram, Slack и др.)
  • Ошибки аутентификации и проблемы с правами доступа
  • Сбои при вызове инструментов
  • Исключения при управлении сессиями

Когда использовать: Когда вы сталкиваетесь с конкретными проблемами и необходимо быстро устранить и решить их.

Справка по командам CLI ​

Полный список команд и инструкции по использованию, включая:

  • Команды управления Gateway (запуск, перезапуск, статус)
  • Команды взаимодействия с Agent
  • Управление и настройка моделей
  • Управление каналами (вход, статус, выход)
  • Управление сессиями (список, сброс, удаление, сжатие)
  • Управление узлами (список, вызов, сопряжение)
  • Управление навыками и мастер настройки

Когда использовать: Когда вам нужно быстро найти использование и параметры команды, или изучить полную функциональность CLI.

Оптимизация производительности ​

Методы оптимизации для повышения производительности и скорости ответа Gateway, охватывающие:

  • Кэширование промптов (Prompt Caching) для снижения затрат
  • Обрезка сессий (Session Pruning) для уменьшения контекста
  • Управление параллелизмом и управление ресурсами
  • Оптимизация памяти и сборка мусора
  • Настройка уровней журналирования

Когда использовать: Когда система работает медленно, использование ресурсов высоко, или вы хотите улучшить взаимодействие с пользователем.

Предварительные условия ​

Перед изучением этой главы рекомендуется, чтобы вы завершили:

  • ✅ Быстрый старт —— Понимание базовых процессов установки и настройки
  • ✅ Запуск Gateway —— Знание того, как запускать и управлять Gateway

Следующие шаги ​

После завершения этой главы вы можете продолжить обучение:

  • Полная справка по настройке —— Глубокое понимание подробных объяснений всех параметров настройки
  • Варианты развёртывания —— Изучение того, как развернуть Clawdbot на различных платформах
  • Руководство по разработке —— Понимание разработки плагинов и процесса внесения вклада
