История версий ​

Этот проект фиксирует все функциональные обновления, улучшения и исправления версий.

v1.0.0 - Январь 2026 ​

🎉 Первый выпуск ​

Это первая официальная версия Agent Skills, включающая полные пакеты навыков и цепочку инструментов сборки.

Новые возможности ​

React-правила производительности

  • 40+ правил оптимизации производительности React/Next.js
  • 8 основных категорий: устранение каскадной загрузки, оптимизация сборки, производительность сервера, получение данных на клиенте, оптимизация повторного рендеринга, производительность рендеринга, микрооптимизация JavaScript
  • Классификация по уровням влияния: CRITICAL, HIGH, MEDIUM-HIGH, MEDIUM, LOW-MEDIUM, LOW
  • Каждое правило содержит примеры кода Incorrect/Correct

Развертывание Vercel одним кликом

  • Поддержка 40+ фреймворков
  • Автоматическое определение типа фреймворка
  • Процесс развертывания без аутентификации
  • Автоматическая генерация ссылок на превью и передачи прав

Руководства по веб-дизайну

  • 100+ правил аудита веб-интерфейса
  • Проверка доступности, производительности и UX
  • Динамическое получение последних правил с GitHub

Цепочка инструментов сборки

  • pnpm build — генерация AGENTS.md и test-cases.json
  • pnpm validate — проверка целостности файлов правил
  • pnpm extract-tests — извлечение тестовых случаев
  • pnpm dev — процесс разработки (сборка + проверка)

Технологический стек ​

  • TypeScript 5.3.0
  • Node.js 20+
  • pnpm 10.24.0+
  • tsx 4.7.0 (выполнение TypeScript)

Документация ​

  • AGENTS.md — полное руководство по правилам React Best Practices
  • SKILL.md — файлы определения навыков
  • README.md — инструкции по установке и использованию
  • Полная справочная документация API и команд

Семантическое версионирование ​

Проект следует семантическому версионированию:

ВерсияОписаниеПример
Major (Основной номер)Несовместимые изменения API1.0.0 → 2.0.0
Minor (Младший номер)Новые функции, обратная совместимость1.0.0 → 1.1.0
Patch (Номер редакции)Исправления ошибок, обратная совместимость1.0.0 → 1.0.1

Справочник ​

Типы изменений ​

  • Добавлено (Added): новая функциональность
  • Улучшено (Improved): улучшение существующей функциональности
  • Исправлено (Fixed): исправление ошибки
  • Изменено (Changed): изменение поведения
  • Удалено (Removed): удаление функциональности

Категории изменений ​

  • Функциональность: новые возможности
  • Производительность: улучшения производительности
  • Документация: обновления документации
  • Инструменты сборки: обновления инструментов сборки
  • Безопасность: исправления безопасности
  • Прочее: изменения без влияния на функциональность

Получение версий ​

Для получения версии Agent Skills:

bash
# Просмотр версии через package.json
cat package.json | grep '"version"'

# Просмотр версии через npx
npx agent-skills --version

Примечание: эта история версий поддерживается в актуальном состоянии. Каждое новое обновление будет добавлено в этот файл.
