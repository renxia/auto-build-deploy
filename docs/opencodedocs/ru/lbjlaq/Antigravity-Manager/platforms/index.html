Платформы и интеграция ​

Основная возможность Antigravity Tools — преобразовывать протоколы нескольких платформ AI в унифицированную локальную API-шлюз. В этой главе подробно описываются способы подключения, границы совместимости и лучшие практики каждого протокола.

Содержание главы ​

УрокОписание
OpenAI-совместимый APIСтратегия реализации /v1/chat/completions и /v1/responses, безболезненный доступ OpenAI SDK
Anthropic-совместимый APIКлючевой контракт /v1/messages с Claude Code, поддержка цепочки размышлений, системных подсказок и других основных возможностей
------
Imagen 3 генерация изображенийАвтоматическое отображение параметров size/quality OpenAI Images, поддержка произвольных соотношений сторон
Транскрибация аудиоОграничения /v1/audio/transcriptions и обработка больших пакетов
Конечные точки MCPПредоставление Web Search/Reader/Vision в качестве вызываемых инструментов
Туннель CloudflaredОдним кликом безопасно открывать локальный API в публичную сеть (не является безопасным по умолчанию)

Предварительные условия ​

Перед началом убедитесь

  • Установлена и обновлена установка
  • Выполнено добавление учетной записи
  • Выполнен запуск локального обратного прокси (минимум доступ к /healthz)

Следующие шаги ​

После изучения этой главы вы можете продолжить обучение:

  • Расширенная конфигурация: маршрутизация моделей, управление квотой, высокодоступное планирование и другие расширенные функции
  • Часто задаваемые вопросы: руководство по устранению неполадок при ошибках 401/404/429
