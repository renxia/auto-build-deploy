Журнал обновлений ​

Эта глава записывает версионную эволюцию и примечания к выпуску Antigravity Tools. Перед обновлением сначала посмотрите Changelog, это уменьшит вероятность столкнуться с проблемами совместимости или регрессией.

Содержание главы ​

СтраницаОписание
Версионная эволюцияКак читать встроенный Changelog README, быстро оценить влияние обновления и выполнить проверку после обновления

Путь обучения ​

  1. Перед обновлением: Сначала подтвердите текущую версию в Settings → About, затем перейдите к Changelog README и читайте только те сегменты версии, которые вы перескочили
  2. После обновления: Запустите один раз /healthz для минимальной проверки, подтвердите, что прокси еще работает нормально

Следующие шаги ​

  • Если вы еще не установили или хотите узнать способ установки, сначала смотрите Установка и обновление
  • Если вы хотите узнать базовое использование прокси, смотрите Запуск локального reverse proxy и подключение первого клиента
