Быстрый старт ​

Этот раздел поможет вам быстро освоить OpenSkills — от установки инструментов до использования навыков ИИ-агентом всего за 10-15 минут.

Путь обучения ​

Рекомендуем изучать материал в следующем порядке:

1. Быстрый старт ​

За 5 минут установите инструменты, навыки и выполните синхронизацию, чтобы оценить ключевые возможности OpenSkills.

  • Установка инструментов OpenSkills
  • Установка навыков из официального репозитория Anthropic
  • Синхронизация навыков в AGENTS.md
  • Проверка доступности навыков для ИИ-агента

2. Что такое OpenSkills? ​

Понимание основных концепций и принципов работы OpenSkills.

  • Связь OpenSkills с Claude Code
  • Единый формат навыков, постепенная загрузка, поддержка нескольких агентов
  • Когда использовать OpenSkills вместо встроенной системы навыков

3. Руководство по установке ​

Подробные инструкции по установке и настройке окружения.

  • Проверка окружения Node.js и Git
  • Временное использование через npx vs глобальная установка
  • Устранение типичных проблем при установке

4. Установка первого навыка ​

Установка навыка из официального репозитория Anthropic с интерактивным выбором.

  • Использование команды openskills install
  • Интерактивный выбор нужных навыков
  • Понимание структуры каталога навыков (.claude/skills/)

5. Синхронизация навыков в AGENTS.md ​

Генерация файла AGENTS.md, чтобы ИИ-агент знал о доступных навыках.

  • Использование команды openskills sync
  • Понимание XML-формата AGENTS.md
  • Выбор навыков для синхронизации, управление размером контекста

6. Использование навыков ​

Узнайте, как ИИ-агент загружает содержимое навыков.

  • Использование команды openskills read
  • 4 уровня приоритета поиска навыков
  • Чтение нескольких навыков одновременно

Предварительные требования ​

Перед началом обучения убедитесь, что:

  • Установлен Node.js версии 20.6.0 или выше
  • Установлен Git (для установки навыков с GitHub)
  • Установлен хотя бы один ИИ-агент для программирования (Claude Code, Cursor, Windsurf, Aider и др.)

Быстрая проверка окружения

bash
node -v  # Должно отобразиться v20.6.0 или выше
git -v   # Должно отобразиться git version x.x.x

Следующие шаги ​

После завершения этого раздела можно продолжить изучение:

  • Подробное описание команд: Глубокое изучение всех команд и параметров
  • Источники установки: Установка навыков с GitHub, из локальных путей, приватных репозиториев
  • Создание собственных навыков: Разработка ваших собственных навыков
