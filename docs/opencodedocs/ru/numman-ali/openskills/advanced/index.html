Продвинутые функции ​

В этой главе рассматриваются продвинутые сценарии использования OpenSkills, включая настройку мульти-агентных сред, пользовательский вывод, разработку через символические ссылки, создание скиллов, интеграцию CI/CD и механизмы безопасности. Освоив эти материалы, вы сможете эффективно управлять скиллами в сложных сценариях и создавать собственную библиотеку скиллов.

Предварительные требования

Перед изучением этой главы убедитесь, что вы завершили:

  • Быстрый старт: базовая установка и процесс использования
  • Установка первого скилла: методы установки скиллов
  • Синхронизация скиллов в AGENTS.md: понимание механизма синхронизации скиллов

Рекомендуемые пути обучения ​

В зависимости от вашего сценария использования выберите подходящий путь обучения:

Путь A: Пользователь мульти-агентов ​

Если вы одновременно используете несколько инструментов AI-кодирования (Claude Code + Cursor + Windsurf и т.д.):

Режим Universal → Пользовательский путь вывода → Лучшие практики

Путь B: Создатель скиллов ​

Если вы хотите создавать собственные скиллы и делиться ими с командой:

Создание пользовательских скиллов → Подробное описание структуры скиллов → Поддержка символических ссылок → Лучшие практики

Путь C: DevOps/Автоматизация ​

Если вам нужно интегрировать OpenSkills в процессы CI/CD:

Интеграция CI/CD → Описание безопасности → Лучшие практики

Путь D: Полное обучение ​

Если вы хотите полностью освоить все продвинутые функции, изучайте в следующем порядке:

  1. Режим Universal — основы мульти-агентных сред
  2. Пользовательский путь вывода — гибкая настройка вывода
  3. Поддержка символических ссылок — эффективный рабочий процесс разработки
  4. Создание пользовательских скиллов — введение в создание скиллов
  5. Подробное описание структуры скиллов — глубокое понимание формата скиллов
  6. Интеграция CI/CD — автоматизированное развертывание
  7. Описание безопасности — подробное описание механизмов безопасности
  8. Лучшие практики — резюме опыта

Следующие шаги ​

После завершения этой главы вы можете:

  • Обратиться к FAQ для решения проблем, возникших при использовании
  • Изучить Справочник CLI API для понимания полного интерфейса командной строки
  • Прочитать Спецификацию формата AGENTS.md для глубокого понимания формата генерируемых файлов
  • Просмотреть Журнал изменений для ознакомления с новейшими функциями и изменениями
