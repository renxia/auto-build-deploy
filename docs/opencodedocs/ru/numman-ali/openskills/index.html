Тема

openskills
Универсальный загрузчик навыков для ИИ-агентов по кодированию

Позволяет любому ИИ-агенту по кодированию (Claude Code, Cursor, Windsurf, Aider и др.) использовать навыки в стандартном формате SKILL.md. Единый стандарт, прогрессивная загрузка, поддержка нескольких агентов.

Быстрый старт GitHub
Почему OpenSkills?

Единый формат навыков, позволяющий использовать навыки в любом агенте, без привязки к платформе.

Полная совместимость с Claude Code

Тот же формат промптов, та же маркетплейс, та же структура папок — без проблемная миграция.

Поддержка нескольких агентов

Поддерживает все агенты, которые могут читать AGENTS.md: Claude Code, Cursor, Windsurf, Aider, Codex и другие.

Прогрессивная загрузка

Загружайте навыки по требованию, сохраняете контекст агента сфокусированным и чистым, избегая загрязнения контекста.

Гибкие источники установки

Поддерживает установку навыков из GitHub, локальных путей и частных git-репозиториев, адаптируясь к различным сценариям использования.

Режим Universal

Набор навыков для нескольких агентов, избегая конфликтов с Claude Code, унифицированное управление навыками.

Полностью локальная работа

CLI-инструмент не требует сервера, все операции выполняются в локальной файловой системе, обеспечивая конфиденциальность и безопасность.
