opencode-notify
Пусть OpenCode AI мягко напоминает вам, когда вы сосредоточены

Попрощайтесь с тревогой постоянного переключения окон. opencode-notify предоставляет нативные уведомления на рабочем столе для вашего OpenCode AI с интеллектуальным обнаружением фокуса, тихими часами и кликом для фокусировки (macOS), позволяя вам работать сосредоточенно и пробуждаться точно, когда AI нуждается в вас.

Быстрый старт GitHub
Почему стоит выбрать opencode-notify?

Сосредоточенная работа, интеллектуальные напоминания

🔔 Нативные уведомления на рабочем столе

Использует нативную систему уведомлений операционной системы (macOS Notification Center, Windows Toast, Linux notify-send), бесшовно интегрируясь в ваш рабочий процесс.

🧠 Интеллектуальное обнаружение фокуса

Автоматически обнаруживает фокус окна терминала, избегая повторяющихся уведомлений. По умолчанию уведомляет только родительскую сессию, фильтруя шум от дочерних задач, позволяя вам сосредоточиться на действительно важной информации.

⏰ Настройка тихих часов

Установите периоды тишины (например, 22:00-08:00), ночью автоматически подавляются уведомления, поддерживаются периоды через полночь, гарантируя ваше время отдыха.

🖱️ Клик для фокусировки терминала (macOS)

Пользователи macOS могут автоматически фокусировать окно терминала одним кликом по уведомлению, экономя время на переключение приложений и удваивая производительность.

🎨 Пользовательские звуки (macOS)

Устанавливайте уникальные звуки для различных типов событий (Glass, Basso, Submarine и другие), быстро распознавая тип уведомления по звуку.

💻 Поддержка 37+ терминалов

Поддержка Ghostty, iTerm2, WezTerm, VS Code интегрированного терминала и более 37 эмуляторов терминала, автоматически распознавая тип вашего терминала.

🔒 Локальный запуск, нулевой риск конфиденциальности

Все уведомления генерируются локально, данные не загружаются в облако, ваш рабочий контент полностью конфиденциален и безопасен.

⚡ Легковесный плагин

Построен на TypeScript + Bun, установка и использование из коробки, конфигурационный файл лаконичен, не требуется изучать сложную конфигурацию.
