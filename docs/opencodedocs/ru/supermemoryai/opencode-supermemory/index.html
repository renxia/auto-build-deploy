Тема

opencode-supermemory
Постоянная память для OpenCode Agent

Плагин OpenCode, обеспечивающий агентам кодирования постоянную память на основе Supermemory, поддерживающий сохранение контекста между сеансами и проектами. Поддерживает сохранение, поиск и удаление памяти, автоматическое внедрение контекста при запуске сеанса и предварительное сжатие на основе использования токенов.

Быстрый старт GitHub
Почему выбрать opencode-supermemory?

Дайте агенту постоянную память и забудьте о проблеме начала с нуля в каждом сеансе.

Постоянство памяти

Решает проблему потери контекста после завершения сеанса агента, важная информация сохраняется постоянно и автоматически восстанавливается в следующем сеансе.

Повторное использование между проектами

Предпочтения в кодировании пользователя (например, «предпочитает функциональное программирование») автоматически применяются в разных проектах без повторных объяснений.

Умное сжатие

Когда контекст исчерпан, автоматически суммирует и сохраняет ключевую информацию, предотвращая «забывание» и поддерживая непрерывность сеанса.

Обезличивание конфиденциальных данных

Поддерживает автоматическое обезличивание с помощью тега <private>, конфиденциальные данные не загружаются в облако, защищая вашу конфиденциальность.

Полный набор инструментов

Предоставляет пять основных инструментов: add, search, profile, list, forget, позволяющих управлять памятью с помощью естественного языка.

Автоматическое внедрение контекста

Автоматически внедряет профиль пользователя и знания о проекте при запуске сеанса, агент сразу входит в состояние, не требуя предварительного разогрева.
