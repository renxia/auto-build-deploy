Часто задаваемые вопросы ​

Эта глава поможет вам решить различные проблемы, возникающие при использовании Plannotator. Занятость портов, неоткрытый браузер или неудачная интеграция — для каждой проблемы есть соответствующее решение и методы отладки.

Содержание главы ​

🔧 Частые проблемы

Решение распространённых проблем при использовании: занятость портов, неоткрытый браузер, отсутствие отображения планов, ошибки Git, неудачная загрузка изображений, проблемы интеграции с Obsidian/Bear и др.

🔍 Устранение неисправностей

Освойте базовые методы устранения неисправностей: просмотр журналов, обработка ошибок и методы отладки. Научитесь быстро находить источник проблемы по выводу журналов.

Путь обучения ​

Частые проблемы → Устранение неисправностей
      ↓              ↓
 Быстрое решение   Глубокая отладка

Рекомендуемый порядок:

  1. Сначала ознакомьтесь с частыми проблемами: для большинства проблем здесь можно найти готовые решения
  2. Затем изучите устранение неисправностей: если частые проблемы не охватывают вашу ситуацию, научитесь самостоятельно диагностировать с помощью журналов и методов отладки

Рекомендации при возникновении проблем

Сначала поискайте по ключевым словам в разделе «Частые проблемы» (например, «порт», «браузер», «Obsidian»), чтобы найти соответствующее решение. Если проблема сложная или отсутствует в списке, обратитесь к разделу «Устранение неисправностей», чтобы изучить методы отладки.

Предварительные условия ​

Перед изучением этой главы рекомендуется выполнить:

  • ✅ Быстрый старт — ознакомьтесь с основными концепциями Plannotator
  • ✅ Установите плагин Claude Code или OpenCode (выберите один):
    • Установка плагина Claude Code
    • Установка плагина OpenCode

Дальнейшие шаги ​

После завершения этой главы вы можете продолжить изучение:

  • Справочник по API — ознакомьтесь со всеми конечными точками API и форматами запросов/ответов
  • Модели данных — ознакомьтесь со структурами данных, используемыми в Plannotator
  • Конфигурация переменных окружения — ознакомьтесь подробно со всеми доступными переменными окружения
