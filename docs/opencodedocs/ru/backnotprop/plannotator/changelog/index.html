Журнал изменений ​

В этом разделе представлена история версий Plannotator, релизы новых функций и исправления ошибок, что поможет вам понять эволюцию проекта и последние достижения.

Содержание этой главы ​

📋 История версий

Просмотрите полную историю версий Plannotator, включая новые функции, улучшения и исправления ошибок в каждой версии. Узнайте временные линии релизов таких важных функций, как код-ревью, аннотация изображений, интеграция с Obsidian/Bear и другие.

Рекомендации по чтению ​

Журнал изменений подходит для чтения в следующих сценариях:

СценарийРекомендация
Проверка перед обновлениемПросмотрите новые функции и критические изменения целевой версии
Изучение функцийУзнайте, какие функции добавлены в последней версии
Устранение неполадокПодтвердите, была ли ошибка исправлена в новой версии
Совместимость версийПроверьте соответствие версий Plannotator с Claude Code/OpenCode

Оставайтесь обновлёнными

Рекомендуем регулярно просматривать журнал изменений и своевременно обновляться до последней версии для получения новых функций и исправлений безопасности.

Следующие шаги ​

После завершения чтения журнала изменений вы можете:

  • Вернуться к Быстрому началу для изучения базового использования
  • Просмотреть Часто задаваемые вопросы для решения проблем при использовании
  • Прочитать Справочник по API для знакомства со всеми конечными точками API
