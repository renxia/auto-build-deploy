Приложение: Справочник по конфигурации и миграции ​

Эта глава содержит подробные справочные материалы и рекомендации по совместимости для oh-my-opencode, включая полную документацию по конфигурации, поддержку миграции из Claude Code и детальное описание встроенных серверов расширений.

Содержание главы ​

Эта глава включает три важных раздела:

ПодстраницаОписаниеСложность
Совместимость с Claude CodeПолное руководство по миграции из Claude Code в OpenCode, включая механизмы совместимости для Commands, Skills, Agents, MCPs и Hooks⭐⭐
Справочник по конфигурацииПолная документация Schema для файлов конфигурации oh-my-opencode, охватывающая все поля, типы и значения по умолчанию⭐⭐⭐
Встроенные MCPФункциональность и использование 3 встроенных MCP-серверов (Exa Websearch, Context7, Grep.app)⭐⭐

Рекомендуемые пути обучения ​

Выберите порядок обучения в соответствии с вашими потребностями:

Путь 1: Миграция из Claude Code ​

Если вы пользователь, мигрирующий из Claude Code:

  1. Сначала прочитайте Совместимость с Claude Code, чтобы понять, как бесшовно мигрировать существующую конфигурацию
  2. Затем изучите Справочник по конфигурации, чтобы глубже понять доступные параметры конфигурации
  3. Наконец, изучите Встроенные MCP, чтобы понять, как настроить дополнительные возможности поиска

Путь 2: Глубокая настройка ​

Если вы хотите глубоко настроить поведение oh-my-opencode:

  1. Начните с Справочника по конфигурации, чтобы понять все настраиваемые элементы
  2. Изучите Встроенные MCP, чтобы настроить возможности поиска и запросов документации
  3. Обратитесь к Совместимости с Claude Code, чтобы понять параметры конфигурации слоя совместимости

Путь 3: Быстрый справочник ​

Если вам нужно только найти конкретную информацию:

  • Вопросы по конфигурации → Перейдите сразу к Справочнику по конфигурации
  • Вопросы по миграции → Перейдите сразу к Совместимости с Claude Code
  • Конфигурация MCP → Перейдите сразу к Встроенным MCP

Предварительные требования ​

Для этой главы рекомендуется, чтобы вы уже:

  • ✅ Завершили Установку и конфигурацию
  • ✅ Поняли основные концепции Оркестратора Sisyphus
  • ✅ Были знакомы с редактированием файлов конфигурации JSON
