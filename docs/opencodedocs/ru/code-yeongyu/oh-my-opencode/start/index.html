Содержание

Быстрый старт ​

Этот раздел поможет вам быстро установить и начать использовать oh-my-opencode: от базовой настройки до основного оркестратора, а затем до режима Ultrawork, который активирует все функции одним щелчком.

Путь обучения ​

Следуйте этой последовательности, чтобы постепенно освоить основные функции oh-my-opencode:

1. Быстрая установка и настройка ​

Завершите установку oh-my-opencode, настройку Provider и базовую проверку.

  • Установите oh-my-opencode
  • Настройте AI Provider (Claude, OpenAI, Gemini и др.)
  • Проверьте правильность конфигурации
  • Изучите структуру файла конфигурации

2. Познакомьтесь с Сизифом: основным оркестратором ​

Поймите основные концепции оркестрации агентов и изучите базовое использование агента Сизифа.

  • Что такое система оркестрации агентов
  • Основные функции агента Сизифа
  • Базовое использование и лучшие практики
  • Понимание рабочего процесса совместной работы нескольких моделей

3. Режим Ultrawork: активация всех функций одним щелчком ​

Изучите, как использовать ключевое слово ultrawork для быстрой активации всех профессиональных агентов и функций параллельных задач.

  • Что такое режим Ultrawork
  • Как вызвать ключевое слово ultrawork
  • Совместная работа 10 профессиональных агентов
  • Управление параллельными фоновыми задачами

Предварительные требования ​

Перед началом работы с этим разделом убедитесь, что:

  • Установлен Node.js 18+ (рекомендуется версия 20 или выше)
  • Вы знакомы с базовыми операциями командной строки
  • У вас есть хотя бы один ключ API AI-сервиса (Claude, OpenAI или Gemini)

Нет ключа API?

Если у вас ещё нет ключа API AI-сервиса, вы можете начать с заявки на Claude, который в настоящее время является наиболее рекомендуемым выбором.

Следующие шаги ​

После завершения этого раздела вы можете продолжить обучение:

  • Платформы и интеграции — углублённое изучение настройки нескольких Provider и стратегий моделей
  • Расширенные функции — изучение полной команды из 10 агентов и расширенных функций
