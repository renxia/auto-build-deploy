Содержание

Быстрый старт ​

Этот раздел поможет вам начать работу с плагином DCP с нуля. Вы научитесь устанавливать плагин, проверять его работу и настраивать его под свои нужды.

Содержание раздела ​

Установка и быстрый старт

Установите плагин DCP за 5 минут и сразу увидите эффект экономии токенов. Узнайте, как использовать команду /dcp для мониторинга статистики обрезки.

Полное руководство по настройке

Освойте трёхуровневую систему конфигурации (глобальная, переменные окружения, уровень проекта), поймите приоритеты настроек и адаптируйте стратегии обрезки и защитные механизмы под свои нужды.

Путь обучения ​

Установка и быстрый старт → Полное руководство по настройке
      ↓                              ↓
   Плагин работает               Знаете, как настроить

Рекомендуемый порядок:

  1. Сначала пройдите Установка и быстрый старт: убедитесь, что плагин работает корректно, и оцените эффект обрезки по умолчанию
  2. Затем изучите Полное руководство по настройке: настройте стратегии обрезки под требования вашего проекта

Совет новичкам

Если вы используете DCP впервые, рекомендуем сначала поработать с настройками по умолчанию какое-то время, оценить эффект обрезки, а затем корректировать конфигурацию.

Предварительные требования ​

Перед началом изучения этого раздела убедитесь, что:

  • [x] Установлен OpenCode (версия с поддержкой плагинов)
  • [x] Вы знакомы с базовым синтаксисом JSONC (JSON с поддержкой комментариев)
  • [x] Вы знаете, как редактировать файл конфигурации OpenCode

Что дальше ​

После завершения этого раздела вы можете продолжить изучение:

  • Автоматические стратегии обрезки: глубоко понять принципы работы трёх стратегий: дедупликация, перезапись и очистка ошибок
  • Инструменты обрезки на базе LLM: узнать, как ИИ может主动 вызывать инструменты discard и extract для оптимизации контекста
  • Использование slash-команд: освоить команды /dcp context, /dcp stats, /dcp sweep и другие
