v1.0.0 - v1.0.1：начальный выпуск и исправление команды слеша ​

Обзор версии ​

v1.0.0 (2026-01-11) — это начальная версия opencode-mystatus, которая принесла основные функции запроса квоты для нескольких платформ.

v1.0.1 (2026-01-11) немедленно последовала, исправив ключевую проблему с поддержкой команд слеша.

v1.0.1 - Исправление команды слеша ​

Исправленные проблемы ​

Включение каталога command/ в пакет npm

  • Описание проблемы: после выпуска v1.0.0 команда слеша /mystatus не работала должным образом
  • Анализ причины: при упаковке npm каталог command/ был пропущен, что привело к тому, что OpenCode не мог распознать команду слеша
  • Решение: обновление поля files в package.json, чтобы убедиться, что каталог command/ включён в публикуемый пакет
  • Область влияния: влияет только на пользователей, установленных через npm, ручная установка не затронута

Рекомендации по обновлению ​

Если вы уже установили v1.0.0, рекомендуется немедленно обновиться до v1.0.1, чтобы получить полную поддержку команд слеша:

bash
## Обновление до последней версии
npm update @vbgate/opencode-mystatus

v1.0.0 - Начальный выпуск ​

Новые функции ​

1. Запрос квоты нескольких платформ

Поддерживает запрос квоты использования следующих платформ одним кликом:

OpenAIPlus/Team/ProЛимит за 3 часа、Лимит за 24 часа
智谱 AICoding PlanЛимит Token за 5 часов、Ежемесячная квота MCP
Google CloudAntigravityG3 Pro、G3 Image、G3 Flash、Claude

2. Визуализированные индикаторы прогресса

Интуитивно отображает использование квоты:

━━━━━━━━━━━━━━━━━━━━ 75%

3. Многоязычная поддержка

  • Китайский (упрощённый)
  • Английский

Язык определяется автоматически, без необходимости ручного переключения.

4. Маскировка API Key

Все конфиденциальные данные (API Key、OAuth Token) автоматически маскируются при отображении:

Способ использования ​

Команда слеша (рекомендуется) ​

В OpenCode введите:

/mystatus

Естественный язык ​

Вы также можете спросить на естественном языке:

Посмотреть все квоты моих AI платформ

Руководство по обновлению ​

Обновление с v1.0.0 до v1.0.1 ​

bash
npm update @vbgate/opencode-mystatus

После обновления перезапустите OpenCode, чтобы использовать команду слеша /mystatus.

Первичная установка ​

bash
npm install -g @vbgate/opencode-mystatus

После установки введите /mystatus в OpenCode, чтобы запросить квоту всех платформ.

Известные ограничения ​

  • v1.0.0 не поддерживает GitHub Copilot (добавлено в v1.2.0)
  • v1.0.0 не поддерживает Z.ai (добавлено в v1.1.0)

Если вы хотите использовать эти функции, обновитесь до последней версии.
