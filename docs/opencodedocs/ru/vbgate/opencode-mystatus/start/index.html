Быстрый старт ​

Этот раздел поможет вам быстро начать работу с плагином opencode-mystatus — от установки до использования всего за 5 минут.

Путь обучения ​

Рекомендуем изучать в следующем порядке:

1. Быстрый старт ​

Установите и настройте плагин за 5 минут, выполните первый запрос квоты ИИ-платформы.

  • Три способа установки: позвольте ИИ установить, ручная установка, установка из локального файла
  • Настройте команду /mystatus
  • Проверьте успешность установки

2. Использование mystatus ​

Изучите два способа использования: слэш-команды и естественный язык.

  • Использование команды /mystatus
  • Запросы на естественном языке для вызова инструмента mystatus
  • Различия между двумя способами и сценарии применения

3. Понимание вывода ​

Поймите индикатор прогресса, время сброса и формат вывода для нескольких аккаунтов.

  • Значение индикатора прогресса: сплошные блоки против пустых блоков
  • Циклы лимитов разных платформ (3 часа, 5 часов, ежемесячно)
  • Формат вывода при наличии нескольких аккаунтов

Предварительные требования ​

Перед началом обучения убедитесь:

  • Установлен OpenCode
  • Настроены данные аутентификации минимум одной ИИ-платформы (OpenAI, Zhipu AI, Z.ai, GitHub Copilot или Google Cloud)

Следующие шаги ​

После завершения этого раздела вы можете продолжить изучение Функции платформ, чтобы узнать подробности запроса квот разных платформ.
