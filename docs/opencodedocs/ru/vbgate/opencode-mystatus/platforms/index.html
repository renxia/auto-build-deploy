Функции платформ ​

Этот раздел подробно описывает функции запроса квот ИИ-платформ, поддерживаемых opencode-mystatus.

Поддерживаемые платформы ​

opencode-mystatus поддерживает следующие 4 основные ИИ-платформы:

ПлатформаТип лимитаОсобенности
OpenAI3 часа / 24 часа скользящее окноПоддерживает подписки Plus/Team/Pro
Zhipu AI5 часов Token / ежемесячная квота MCPПоддерживает Coding Plan
GitHub CopilotЕжемесячная квотаОтображает использование Premium Requests
Google CloudПо моделям независимоПоддерживает несколько аккаунтов, 4 модели

Подробности платформ ​

Квота OpenAI ​

Узнайте глубже о механизме запроса квот OpenAI:

  • Различие между скользящими окнами 3 часов и 24 часов
  • Механизм совместного использования квот командных аккаунтов
  • Анализ JWT Token для получения информации об аккаунте

Квота Zhipu AI ​

Узнайте о запросе квот Zhipu AI и Z.ai:

  • Способ расчёта лимита Token за 5 часов
  • Назначение ежемесячной квоты MCP
  • Маскировка отображения API Key

Квота GitHub Copilot ​

Освойте управление квотами GitHub Copilot:

  • Значение Premium Requests
  • Различие в квотах разных типов подписок
  • Расчёт времени ежемесячного сброса

Квота Google Cloud ​

Узнайте запрос квот нескольких аккаунтов Google Cloud:

  • Различие 4 моделей (G3 Pro, G3 Image, G3 Flash, Claude)
  • Управление и переключение нескольких аккаунтов
  • Механизм чтения файлов аутентификации

Руководство по выбору ​

Выберите соответствующее руководство в зависимости от используемых платформ:

  • Только OpenAI: сразу смотрите Квота OpenAI
  • Только Zhipu AI: сразу смотрите Квота Zhipu AI
  • Многоплатформенный пользователь: рекомендуется последовательно прочитать все руководства по платформам
  • Пользователи Google Cloud: нужно сначала установить плагин opencode-antigravity-auth
