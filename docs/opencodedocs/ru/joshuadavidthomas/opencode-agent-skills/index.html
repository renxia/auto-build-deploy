opencode-agent-skills
Плагин динамического управления навыками для OpenCode AI

Система плагинов для динамического обнаружения и управления навыками в OpenCode, позволяющая AI-агентам загружать и использовать переиспользуемые библиотеки навыков. Реализует переиспользование навыков между проектами и сессиями с автоматическими рекомендациями на основе семантического сходства.

Быстрый старт GitHub
Почему OpenCode Agent Skills?

Динамическое обнаружение и управление навыками для OpenCode с переиспользованием между проектами и сессиями.

Динамическое обнаружение навыков

Автоматическое обнаружение навыков из проектных, пользовательских и плагин-директорий. Поддержка сканирования 6 локаций с возвратом первого совпадения по приоритету.

Совместимость с Claude Code

Поддержка существующих навыков и форматов плагинов Claude Code. Снижение затрат на миграцию и переиспользование существующих ресурсов.

Семантические рекомендации

Использование embedding-моделей HuggingFace для вычисления сходства. Автоматические рекомендации релевантных навыков на основе пользовательского ввода.

Инъекция контекста

Содержимое навыков инъецируется в контекст сессии в XML-формате. AI следует руководству навыка для получения качественных результатов.

Восстановление после сжатия

Автоматическая повторная инъекция списка навыков после сжатия сессии. Навыки остаются доступными в длительных сессиях.

Поддержка выполнения скриптов

Директория навыка может содержать исполняемые скрипты. Выполнение автоматизированных задач в контексте директории навыка.
