Содержание

Часто задаваемые вопросы ​

Этот раздел поможет вам решить проблемы, возникающие при использовании плагина OpenCode Agent Skills, и познакомит с механизмами безопасности плагина. Будь то сбой загрузки навыка, ошибка выполнения скрипта или желание узнать, как плагин защищает вашу систему, — здесь вы найдете ответы.

Содержание главы ​

🔧 Устранение распространенных проблем

Решение проблем с загрузкой навыков, ошибками выполнения скриптов, проблемами безопасности путей, сбоями загрузки моделей и другими 9 типами распространенных проблем.

🔒 Механизмы безопасности

Узнайте о ключевых механизмах безопасности плагина, включая проверку путей, безопасный парсинг YAML, валидацию входных данных, защиту выполнения скриптов и лучшие практики.

Путь обучения ​

Рекомендуется изучать материалы этого раздела в следующем порядке:

Устранение распространенных проблем → Механизмы безопасности
  1. Сначала изучите устранение неполадок: при возникновении проблем сначала обратитесь к разделу «Устранение распространенных проблем» для быстрого выявления и решения
  2. Затем изучите безопасность: ознакомьтесь с механизмами безопасности плагина, чтобы выработать безопасные привычки использования

Что делать при возникновении проблем

Если вы столкнулись с конкретной проблемой при использовании плагина, сразу перейдите к разделу «Устранение распространенных проблем» и найдите соответствующие симптомы. Если вы хотите системно ознакомиться с проектом безопасности плагина, прочитайте раздел «Механизмы безопасности».

Предварительные требования ​

Рекомендуется сначала завершить следующие разделы

  • Руководство по началу работы: ознакомьтесь с основными концепциями и методами установки плагина
  • Платформы и интеграции: изучите ключевые функции, такие как обнаружение, загрузка и выполнение навыков

Содержание этого раздела предполагает, что вы уже имеете базовое представление о плагине. Если вы используете его впервые, рекомендуется сначала изучить указанные выше разделы.

Следующий шаг ​

После завершения изучения этого раздела вы можете продолжить изучение:

  • Приложения: ознакомьтесь с лучшими практиками разработки навыков и справочником по инструментам API
  • Журнал изменений: узнайте о версиях плагина и новых функциях
