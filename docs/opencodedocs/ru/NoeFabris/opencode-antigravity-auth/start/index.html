Быстрый старт ​

Эта глава поможет вам начать использование плагина Antigravity Auth с нуля. Вы узнаете о ключевой ценности плагина, завершите установку и аутентификацию OAuth, а также отправите первый запрос к модели для проверки успешности настройки.

Путь обучения ​

Изучайте материалы в следующем порядке — каждый шаг основывается на предыдущем:

1. Введение в плагин ​

Узнайте о ключевой ценности плагина Antigravity Auth, применимых сценариях и предупреждениях о рисках.

  • Определите, подходит ли плагин для вашего сценария использования
  • Узнайте о поддерживаемых моделях ИИ (Claude Opus 4.5, Sonnet 4.5, Gemini 3 Pro/Flash)
  • Выясните риски использования и меры предосторожности

2. Быстрая установка ​

Установите плагин и завершите первую аутентификацию за 5 минут.

  • Два способа установки (с ИИ-помощью / ручная настройка)
  • Настройка определений моделей
  • Выполнение аутентификации Google OAuth

3. Аутентификация OAuth 2.0 PKCE ​

Поймите процесс аутентификации OAuth 2.0 PKCE и завершите первый вход.

  • Поймите механизм безопасности аутентификации PKCE
  • Завершите первый вход и получите доступ к API
  • Узнайте об автоматическом обновлении токенов

4. Первый запрос ​

Отправьте первый запрос к модели и проверьте успешность установки.

  • Отправьте первый запрос к модели Antigravity
  • Поймите параметры --model и --variant
  • Устраните распространённые ошибки запросов к моделям

Предварительные условия ​

Перед началом этой главы убедитесь:

  • ✅ Установлен OpenCode CLI (команда opencode доступна)
  • ✅ Имеется учётная запись Google (для аутентификации OAuth)

Следующие шаги ​

После завершения быстрого старта вы можете:

  • Доступные модели — исследовать все поддерживаемые модели и их конфигурации вариантов
  • Настройка нескольких аккаунтов — настроить несколько учётных записей Google для объединения квот
  • Распространённые проблемы аутентификации — обратиться к руководству по устранению неполадок при возникновении проблем
