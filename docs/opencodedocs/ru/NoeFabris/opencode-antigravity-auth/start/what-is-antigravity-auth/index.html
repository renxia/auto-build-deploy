Узнайте ключевые преимущества плагина Antigravity Auth ​

Antigravity Auth — это плагин для OpenCode, который обеспечивает доступ к Antigravity API через аутентификацию Google OAuth. Он позволяет использовать продвинутые модели, такие как Claude Opus 4.5, Sonnet 4.5 и Gemini 3 Pro/Flash, с помощью вашего привычного аккаунта Google — без необходимости управлять API-ключами. Плагин также предоставляет балансировку нагрузки между несколькими аккаунтами, двойной пул квот и автоматическое восстановление сессий — идеально для пользователей, которым нужны продвинутые модели и автоматизированное управление.

Чему вы научитесь ​

  • Определить, подходит ли этот плагин для вашего сценария использования
  • Узнать, какие AI-модели и основные функции поддерживает плагин
  • Понять риски и меры предосторожности при использовании плагина
  • Принять решение о продолжении установки и настройки

Ваша текущая проблема ​

Вы хотите использовать самые передовые AI-модели (такие как Claude Opus 4.5, Gemini 3 Pro), но официальный доступ ограничен. Вы ищете надёжный способ получить доступ к этим моделям и при этом хотите:

  • Не управлять вручную несколькими API-ключами
  • Автоматически переключаться на другой аккаунт при достижении лимита запросов
  • Автоматически восстанавливать диалог после прерывания без потери контекста

Основная идея ​

Antigravity Auth — это плагин для OpenCode, который обеспечивает доступ к Google Antigravity API через аутентификацию Google OAuth, позволяя вызывать продвинутые AI-модели с помощью вашего привычного аккаунта Google.

Он не проксирует все запросы, а перехватывает и преобразует ваши запросы к моделям, перенаправляя их в Antigravity API, а затем конвертирует ответы обратно в формат, понятный OpenCode.

Основные возможности ​

Поддерживаемые модели ​

Семейство моделейДоступные моделиОсобенности
ClaudeOpus 4.5, Sonnet 4.5Поддержка режима расширенного мышления
Gemini 3Pro, FlashИнтеграция с Google Search, расширенное мышление

Режим мышления (Thinking)

Модели с режимом Thinking выполняют «глубокое размышление» перед генерацией ответа, показывая процесс рассуждения. Вы можете настроить бюджет мышления для баланса между качеством и скоростью ответа.

Балансировка нагрузки между несколькими аккаунтами ​

  • Поддержка до 10 аккаунтов Google
  • Автоматическое переключение на следующий аккаунт при достижении лимита запросов (ошибка 429)
  • Три стратегии выбора аккаунта: sticky (фиксированный), round-robin (ротация), hybrid (интеллектуальный гибрид)

Система двойных квот ​

Плагин одновременно использует два независимых пула квот:

  1. Квота Antigravity: от Google Antigravity API
  2. Квота Gemini CLI: от Google Gemini CLI

Когда один пул достигает лимита, плагин автоматически пробует другой, максимизируя использование квот.

Автоматическое восстановление сессий ​

  • Обнаружение неудачных вызовов инструментов (например, при нажатии ESC для прерывания)
  • Автоматическая инъекция синтетического tool_result для предотвращения сбоя диалога
  • Поддержка автоматической отправки "continue" для продолжения диалога

Интеграция с Google Search ​

Включение веб-поиска для моделей Gemini для повышения фактической точности:

  • Режим Auto: модель сама решает, нужен ли поиск
  • Режим Always-on: поиск выполняется каждый раз

Когда использовать этот плагин ​

Подходит для следующих сценариев

  • У вас несколько аккаунтов Google и вы хотите увеличить общую квоту
  • Вам нужны модели Claude или Gemini 3 с режимом Thinking
  • Вы хотите включить Google Search для моделей Gemini
  • Вы не хотите вручную управлять API-ключами и предпочитаете OAuth-аутентификацию
  • Вы часто сталкиваетесь с лимитами запросов и хотите автоматическое переключение аккаунтов

Не подходит для следующих сценариев

  • Вам нужны модели, которые Google официально не публикует
  • Вы очень чувствительны к рискам нарушения ToS Google (см. предупреждение о рисках ниже)
  • Вам нужны только базовые модели Gemini 1.5 или Claude 3 (официальные интерфейсы стабильнее)
  • Вам сложно открыть браузер в средах WSL, Docker и подобных

⚠️ Важное предупреждение о рисках ​

Использование этого плагина может нарушать Условия использования Google. Небольшое количество пользователей сообщили, что их аккаунты Google были заблокированы или теневым образом ограничены (доступ ограничен без явного уведомления).

Сценарии высокого риска ​

  • 🚨 Совершенно новые аккаунты Google: очень высокая вероятность блокировки
  • 🚨 Аккаунты с недавно оформленной подпиской Pro/Ultra: легко помечаются и блокируются

Перед использованием убедитесь ​

  • Это неофициальный инструмент, не одобренный Google
  • Ваш аккаунт может быть временно или навсегда заблокирован
  • Вы принимаете на себя все риски использования этого плагина

Рекомендации ​

  • Используйте зрелые аккаунты Google, а не создавайте новые специально для этого плагина
  • Избегайте использования важных аккаунтов, связанных с критическими сервисами
  • Если аккаунт заблокирован, обжаловать через этот плагин невозможно

Безопасность аккаунта

Все OAuth-токены хранятся локально в ~/.config/opencode/antigravity-accounts.json и не загружаются ни на какие серверы. Однако убедитесь, что ваш компьютер защищён, чтобы предотвратить утечку токенов.
