Тема

Antigravity Auth
Ключ к аутентификации Claude & Gemini

Доступ к Claude Opus 4.5, Sonnet 4.5 и Gemini 3 Pro/Flash без API Key напрямую через Google OAuth. Поддержка балансировки нагрузки между несколькими аккаунтами и автоматического восстановления сессий.

Быстрый старт GitHub
Почему Antigravity Auth?

Решение проблем ручной настройки API Key, автоматическое управление квотами, создано для высоконагруженной разработки.

Без API Key

Не нужно получать дорогой API Key, просто используйте Google OAuth для входа в один клик.

Балансировка нагрузки

Поддержка до 10 Google аккаунтов с автоматической ротацией и интеллектуальным распределением запросов — навсегда забудьте о лимитах.

Поддержка Thinking моделей

Полная поддержка режима размышлений Claude и Gemini 3 с настраиваемым Budget для раскрытия глубоких возможностей рассуждения.

Автовосстановление сессий

Ошибка вызова инструмента? Прерывание сессии? Плагин автоматически повторяет попытку и восстанавливает контекст без вмешательства человека.

Совместимость

Бесшовная интеграция с oh-my-opencode, DCP и другими популярными плагинами.

Двойная квота

Максимальное использование бесплатных и платных пулов квот от Google.
