История версий ​

В этом документе описана история версий и важные изменения плагина Antigravity Auth.

Последняя версия

Текущая стабильная версия: v1.3.0 (2026-01-17)

Описание версий ​

Стабильная версия (Stable) ​

  • Полностью протестирована, рекомендуется для production-среды
  • Формат номера версии: vX.Y.Z (например, v1.3.0)

Бета-версия (Beta) ​

  • Содержит новейшие функции, возможна нестабильность
  • Формат номера версии: vX.Y.Z-beta.N (например, v1.3.1-beta.3)
  • Подходит для раннего тестирования и обратной связи

Руководство по обновлению версий ​

Обновление с v1.2.x до v1.3.x ​

Совместимость: полная совместимость, изменение конфигурации не требуется

Рекомендуемые действия:

bash
# Обновление плагина
opencode plugin update opencode-antigravity-auth

# Проверка установки
opencode auth status

Обновление с v1.1.x до v1.2.x ​

Совместимость: требуется обновление формата хранения аккаунтов

Рекомендуемые действия:

bash
# Резервное копирование существующих аккаунтов
cp ~/.config/opencode/antigravity-accounts.json ~/.config/opencode/antigravity-accounts.json.bak

# Обновление плагина
opencode plugin update opencode-antigravity-auth@latest

# Повторный вход (при возникновении проблем)
opencode auth login

Обновление с v1.0.x до v1.2.x ​

Совместимость: формат хранения аккаунтов несовместим, требуется повторная аутентификация

Рекомендуемые действия:

bash
# Обновление плагина
opencode plugin update opencode-antigravity-auth@latest

# Повторный вход
opencode auth login

# Добавление конфигурации модели согласно требованиям новой версии

Примечания к бета-версиям ​

Рекомендации по использованию бета-версий:

Сценарий использованияРекомендуемая версияОписание
Production-средаСтабильная (vX.Y.Z)Полностью протестирована, высокая стабильность
Повседневная разработкаПоследняя стабильнаяПолный функционал, меньше ошибок
Раннее тестированиеПоследняя BetaДоступ к новейшим функциям, возможна нестабильность

Установка бета-версии:

json
{
  "plugin": ["opencode-antigravity-auth@beta"]
}

Обновление до стабильной версии:

bash
opencode plugin update opencode-antigravity-auth@latest

Описание номеров версий ​

Номера версий соответствуют спецификации Семантическое версионирование 2.0.0:

  • Мажорная версия (X): несовместимые изменения API
  • Минорная версия (Y): обратно совместимые новые функции
  • Патч-версия (Z): обратно совместимые исправления ошибок

Примеры:

  • 1.3.0 → мажорная версия 1, минорная версия 3, патч 0
  • 1.3.1-beta.3 → 3-я бета-версия для 1.3.1

Получение уведомлений об обновлениях ​

Автоматическое обновление (включено по умолчанию):

json
{
  "auto_update": true
}

Ручная проверка обновлений:

bash
# Просмотр текущей версии
opencode plugin list

# Обновление плагина
opencode plugin update opencode-antigravity-auth
