On this page

Quick Start ​

This chapter helps you get started with Plannotator from scratch. You'll learn what Plannotator is, what problems it solves, and complete the installation setup in your AI Coding Agent (Claude Code or OpenCode).

Chapter Contents ​

Quick Start ​

Understand Plannotator's core features and use cases, and complete your first experience in 5 minutes. Suitable for all new users as an introductory lesson.

Install Claude Code Plugin ​

Install and configure the Plannotator plugin in Claude Code, supporting both plugin system and manual Hook methods, covering local and remote environment configuration.

Install OpenCode Plugin ​

Install and configure the Plannotator plugin in OpenCode, enabling the submit_plan tool and /plannotator-review command.

Learning Path ​

Recommended Sequence

  1. First read Quick Start: Understand what Plannotator is and what it can do, establish overall understanding
  2. Then choose an installation tutorial: Select the installation guide based on the AI Coding Agent you use
    • Using Claude Code → Install Claude Code Plugin
    • Using OpenCode → Install OpenCode Plugin

After completing the installation, you can start using Plannotator for plan review and code review.

Next Steps ​

After completing this chapter, continue learning Platform Features to master the complete usage of plan review and code review:

  • Plan review basics: approve, reject, add comments
  • Image annotation: attach screenshots and annotate in plans
  • Code review: visualize Git diff, add line-level comments
