On this page

FAQ & Best Practices ​

The FAQ section collects common problem solutions and best practice recommendations for using AI App Factory, helping you quickly troubleshoot issues and improve development efficiency.

Subpages Navigation ​

TutorialDescription
FAQ & TroubleshootingSolve common problems encountered during initialization, execution, and deployment
Best PracticesLearn techniques for providing clear product descriptions, leveraging checkpoints, controlling scope, and iterating step-by-step

Learning Path Recommendations ​

Content in the FAQ section can be referenced on-demand without strictly following the order:

  1. Check when encountering problems: If you encounter any issues during use, prioritize checking "FAQ & Troubleshooting"
  2. Must-read for efficiency improvement: The "Best Practices" chapter summarizes experience and lessons. It's recommended to read after completing your first project—development efficiency will significantly improve in subsequent projects

Prerequisites ​

Recommended First

Before reading the FAQ section, it's recommended to complete the following:

  • ✅ start chapters: Complete project initialization and run the full pipeline
  • ✅ platforms chapters: Configure at least one AI assistant (Claude Code or OpenCode)

Next Steps ​

After completing the FAQ section, you can:

  1. Dive into technical details:

    • CLI Command Reference - View complete command list and parameters
    • Directory Structure Explained - Understand project structure and file purposes
    • Code Standards - Learn standards for generated code

  2. Get hands-on:

    • Optimize your project description with best practices
    • Try more complex product ideas
    • Explore every detail of the 7-stage pipeline
