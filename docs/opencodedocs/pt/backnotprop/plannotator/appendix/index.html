Nesta página

Apêndice ​

Esta seção fornece referências técnicas do Plannotator, incluindo documentação da API, definições de modelos de dados e informações de licenciamento. Este conteúdo é destinado principalmente a usuários que precisam compreender profundamente os mecanismos internos do sistema ou desenvolver aplicações secundárias.

Conteúdo deste Capítulo ​

PáginaDescrição
Referência da APIConsulte todos os endpoints de API fornecidos pelo Plannotator e os formatos de requisição/resposta
Modelos de DadosEntenda os modelos de dados usados pelo Plannotator, incluindo Annotation, Block, CodeAnnotation, etc.
LicenciamentoSaiba mais sobre a licença do Plannotator (BSL-1.1) e restrições de uso

Recomendação de Percurso de Aprendizagem ​

O conteúdo do apêndice é relativamente independente, você pode ler seletivamente de acordo com suas necessidades:

  1. Quer entender interfaces de API → Leia diretamente Referência da API
  2. Quer entender estruturas de dados → Leia Modelos de Dados
  3. Quer entender restrições de uso → Leia Licenciamento

Dica

O conteúdo do apêndice é principalmente para consulta, não precisa ser aprendido em sequência. Recomenda-se consultar os capítulos relevantes quando encontrar problemas específicos.

Pré-requisitos ​

Antes de ler o apêndice, recomenda-se concluir o seguinte conteúdo:

  • Início Rápido - Entenda os conceitos básicos do Plannotator
  • Fundamentos de Revisão de Planos - Familiarize-se com operações básicas de revisão

Próximos Passos ​

Após concluir a leitura do apêndice, você pode:

  • Consultar Notas de Lançamento para entender o histórico de versões e novos recursos
  • Retornar a Recursos Avançados para explorar mais usos avançados
  • Consultar Perguntas Frequentes quando encontrar problemas
