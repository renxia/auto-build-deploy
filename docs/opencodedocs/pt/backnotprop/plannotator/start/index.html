Nesta página

Início Rápido ​

Este capítulo ajuda você a começar a usar o Plannotator do zero. Você entenderá o que é o Plannotator, quais problemas ele resolve e completará a instalação e configuração no seu AI Coding Agent (Claude Code ou OpenCode).

Conteúdo do Capítulo ​

Início Rápido ​

Entenda as funcionalidades principais e cenários de uso do Plannotator, e complete sua primeira experiência em 5 minutos. Adequado como primeira aula de introdução para todos os novos usuários.

Instalar Plugin Claude Code ​

Instale e configure o plugin Plannotator no Claude Code, suportando dois métodos: sistema de plugins e Hook manual, cobrindo configuração de ambientes locais e remotos.

Instalar Plugin OpenCode ​

Instale e configure o plugin Plannotator no OpenCode, ativando a ferramenta submit_plan e o comando /plannotator-review.

Caminho de Aprendizado ​

Sequência Recomendada

  1. Primeiro leia Início Rápido: Entenda o que é o Plannotator, o que ele pode fazer, e estabeleça uma compreensão geral
  2. Depois escolha um tutorial de instalação: Selecione o guia de instalação correspondente baseado no AI Coding Agent que você usa
    • Usando Claude Code → Instalar Plugin Claude Code
    • Usando OpenCode → Instalar Plugin OpenCode

Após completar a instalação, você pode começar a usar o Plannotator para revisão de planos e revisão de código.

Próximos Passos ​

Após completar este capítulo, continue aprendendo Funcionalidades da Plataforma para dominar o uso completo de revisão de planos e revisão de código:

  • Fundamentos de revisão de planos: aprovar, rejeitar, adicionar comentários
  • Anotação de imagens: anexar capturas de tela nos planos e adicionar anotações
  • Revisão de código: visualizar Git diff, adicionar comentários de nível de linha
