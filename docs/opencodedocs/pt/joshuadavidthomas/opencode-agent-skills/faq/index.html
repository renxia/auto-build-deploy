Nesta página

Perguntas Frequentes ​

Este capítulo ajuda você a resolver problemas ao usar o plugin OpenCode Agent Skills e a entender os mecanismos de segurança do plugin. Seja uma falha no carregamento de habilidades, erro de execução de script, ou entender como o plugin protege seu sistema, você encontrará respostas aqui.

Conteúdo do Capítulo ​

🔧 Solução de Problemas Comuns

Resolva 9 tipos de problemas comuns como falhas no carregamento de habilidades, erros de execução de script, problemas de segurança de caminho, falhas no carregamento de modelo, etc., com métodos de solução de problemas e soluções.

🔒 Considerações de Segurança

Entenda os mecanismos centrais de segurança e melhores práticas do plugin, como verificação de segurança de caminho, segurança de análise YAML, validação de entrada, proteção de execução de script, etc.

Caminho de Aprendizado ​

Recomenda-se estudar o conteúdo deste capítulo na seguinte ordem:

Solução de Problemas Comuns → Considerações de Segurança
  1. Primeiro, Solução de Problemas: Ao encontrar problemas, consulte primeiro "Solução de Problemas Comuns" para localizar e resolver rapidamente
  2. Depois, Segurança: Entenda os mecanismos de segurança do plugin e desenvolva hábitos de uso seguro

O Que Fazer Quando Surgem Problemas

Se você está encontrando um problema específico ao usar o plugin, vá diretamente para "Solução de Problemas Comuns" para encontrar os sintomas correspondentes. Se quiser entender sistematicamente o design de segurança do plugin, leia "Considerações de Segurança".

Pré-requisitos ​

Recomenda-se completar os seguintes capítulos primeiro

  • Guia de Início Rápido: Entenda os conceitos básicos e métodos de instalação do plugin
  • Plataformas e Integrações: Familiarize-se com funções centrais como descoberta, carregamento e execução de habilidades

O conteúdo deste capítulo assume que você já tem um entendimento básico do plugin. Se é a primeira vez que o usa, recomenda-se completar o estudo dos capítulos acima primeiro.

Próximos Passos ​

Após completar o estudo deste capítulo, você pode continuar explorando:

  • Apêndice: Consulte as melhores práticas de desenvolvimento de habilidades e referência de ferramentas API
  • Registro de Alterações: Entenda a evolução da versão do plugin e novos recursos
