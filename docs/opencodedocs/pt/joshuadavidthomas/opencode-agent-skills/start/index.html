Nesta página

Início Rápido ​

Este capítulo ajuda você a começar a usar o plugin OpenCode Agent Skills do zero. Você entenderá o valor principal do plugin, completará a instalação e configuração, e criará sua primeira habilidade.

O Que Você Vai Aprender ​

O que é OpenCode Agent Skills?

Entenda o valor principal e as características do plugin, incluindo descoberta dinâmica de habilidades, injeção de contexto, mecanismos de compressão e recuperação, entre outros.

Guia de Instalação

Complete a instalação do plugin e verifique que está funcionando, com suporte para instalação básica, instalação de versão fixa e instalação para desenvolvimento local.

Crie Sua Primeira Habilidade

Domine a estrutura de diretórios de habilidades e o formato de especificação SKILL.md, então crie e teste uma habilidade simples na prática.

Caminho de Aprendizado ​

Recomendamos aprender na seguinte ordem:

  1. O que é OpenCode Agent Skills? — Primeiro entenda o que o plugin pode fazer e construa uma compreensão geral
  2. Guia de Instalação — Instale o plugin para dar ao OpenCode capacidades de gerenciamento de habilidades
  3. Crie Sua Primeira Habilidade — Prática hands-on criando sua primeira habilidade

Tempo Estimado

Este capítulo leva aproximadamente 30-45 minutos para ser concluído.

Próximos Passos ​

Após concluir este capítulo, você pode continuar para Recursos da Plataforma para aprender mais sobre recursos avançados como mecanismos de descoberta de habilidades, carregamento de habilidades e correspondência automática.
