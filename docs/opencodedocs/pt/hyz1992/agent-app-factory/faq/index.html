FAQ e Melhores Práticas ​

A seção de FAQ reúne soluções para problemas comuns e recomendações de melhores práticas ao usar o AI App Factory, ajudando você a resolver problemas rapidamente e melhorar a eficiência do desenvolvimento.

Navegação de Subpáginas ​

CursoDescrição
FAQ e Solução de ProblemasResolva problemas comuns durante a inicialização, execução e implantação
Melhores PráticasAprenda técnicas para fornecer descrições claras de produtos, aproveitar checkpoints, controlar escopo e iterar gradualmente

Recomendações de Rota de Aprendizado ​

O conteúdo da seção de FAQ pode ser consultado conforme necessário, sem necessidade de seguir uma ordem rigorosa:

  1. Consultar quando encontrar problemas: Se encontrar algum problema durante o uso, verifique primeiro o "FAQ e Solução de Problemas"
  2. Leitura obrigatória para melhorar a eficiência: A seção "Melhores Práticas" resume lições aprendidas; recomenda-se lê-la após concluir o primeiro projeto, pois a eficiência do desenvolvimento de projetos subsequentes será significativamente melhorada

Pré-requisitos ​

Recomendação de Pré-requisito

Antes de ler a seção de FAQ, recomenda-se concluir o seguinte:

  • ✅ Seção start: Concluir a inicialização do projeto e executar o pipeline completo
  • ✅ Seção platforms: Configurar pelo menos um assistente de IA (Claude Code ou OpenCode)

Próximos Passos ​

Após concluir o aprendizado do FAQ, você pode:

  1. Aprofundar-se nos detalhes técnicos:

    • Referência de Comandos CLI - Ver a lista completa de comandos e parâmetros
    • Detalhes da Estrutura de Diretórios - Compreender a estrutura do projeto e o propósito dos arquivos
    • Padrões de Código - Conhecer os padrões que o código gerado deve seguir

  2. Começar a prática:

    • Otimize a descrição do seu projeto com as melhores práticas
    • Experimente ideias de produtos mais complexas
    • Explore cada detalhe do pipeline de 7 fases
