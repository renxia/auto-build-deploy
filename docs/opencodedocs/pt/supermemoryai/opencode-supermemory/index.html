opencode-supermemory
Memória persistente para OpenCode Agent

Plugin OpenCode que fornece capacidade de memória persistente para agentes de codificação baseados em Supermemory, permitindo retenção de contexto entre sessões e projetos. Suporta armazenamento, recuperação e exclusão de memória, injeção automática de contexto ao iniciar a sessão e compressão preemptiva com base no uso de tokens.

Início Rápido GitHub
Por que escolher opencode-supermemory?

Dê ao agente memória persistente e elimine a frustração de começar do zero em cada sessão.

Memória Persistente

Resolve o problema de perda de contexto após o término da sessão do agente, salvando informações importantes permanentemente e restaurando automaticamente na próxima sessão.

Reutilização entre Projetos

As preferências de codificação do usuário (como «prefere programação funcional») são automaticamente aplicadas em diferentes projetos, sem necessidade de explicações repetidas.

Compressão Inteligente

Quando o contexto está prestes a se esgotar, resume e salva automaticamente informações-chave, evitando «esquecimento» e mantendo a continuidade da sessão.

Anonimização de Privacidade

Suporta anonimização automática com a tag <private>, dados sensíveis não são enviados para a nuvem, protegendo sua privacidade.

Conjunto Completo de Ferramentas

Fornece cinco ferramentas principais: add, search, profile, list, forget, permitindo controlar a memória através de linguagem natural.

Injeção Automática de Contexto

Ao iniciar a sessão, injeta automaticamente o perfil do usuário e o conhecimento do projeto, o agente entra no estado imediatamente, sem necessidade de aquecimento.
