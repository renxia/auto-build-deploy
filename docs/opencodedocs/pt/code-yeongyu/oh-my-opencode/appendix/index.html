Apêndice: Guia de Referência e Compatibilidade ​

Este capítulo contém materiais de referência detalhados e diretrizes de compatibilidade para o oh-my-opencode, cobrindo documentação completa de configuração, suporte de migração do Claude Code e introduções detalhadas aos servidores de extensão integrados.

Conteúdo do Capítulo ​

Este capítulo inclui três seções importantes:

SubpáginaDescriçãoDificuldade
Compatibilidade com Claude CodeGuia completo para migrar do Claude Code para o OpenCode, incluindo mecanismos de compatibilidade para Commands, Skills, Agents, MCPs e Hooks⭐⭐
Referência de ConfiguraçãoDocumentação completa do Schema para arquivos de configuração do oh-my-opencode, cobrindo todos os campos, tipos e valores padrão⭐⭐⭐
MCPs IntegradosRecursos e uso de 3 servidores MCP integrados (Exa Websearch, Context7, Grep.app)⭐⭐

Caminhos de Aprendizagem Recomendados ​

Escolha sua ordem de aprendizado com base em suas necessidades:

Caminho 1: Migração do Claude Code ​

Se você é um usuário migrando do Claude Code:

  1. Primeiro leia Compatibilidade com Claude Code para entender como migrar perfeitamente sua configuração existente
  2. Em seguida, revise Referência de Configuração para obter uma compreensão mais profunda das opções de configuração disponíveis
  3. Por fim, aprenda sobre MCPs Integrados para entender como configurar recursos adicionais de pesquisa

Caminho 2: Personalização Profunda ​

Se você deseja personalizar profundamente o comportamento do oh-my-opencode:

  1. Comece com Referência de Configuração para entender todos os itens configuráveis
  2. Aprenda sobre MCPs Integrados para configurar recursos de pesquisa e consulta de documentação
  3. Consulte Compatibilidade com Claude Code para entender as opções de configuração da camada de compatibilidade

Caminho 3: Referência Rápida ​

Se você só precisa consultar informações específicas:

  • Perguntas sobre configuração → Vá diretamente para Referência de Configuração
  • Perguntas sobre migração → Vá diretamente para Compatibilidade com Claude Code
  • Configuração de MCP → Vá diretamente para MCPs Integrados

Pré-requisitos ​

Para este capítulo, recomenda-se que você já tenha:

  • ✅ Concluído Instalação e Configuração
  • ✅ Entendido os conceitos básicos do Orquestrador Sisyphus
  • ✅ Familiaridade com edição de arquivos de configuração JSON

Próximos Passos ​

Após concluir este capítulo, você pode:

  • 🚀 Experimentar Recursos Avançados para aprender mais sobre uso avançado
  • 🔧 Verificar Perguntas Frequentes para resolver problemas de uso
  • 📖 Ler Changelog para se manter atualizado sobre melhorias de novos recursos

Dica

O conteúdo deste capítulo serve principalmente como material de referência e não precisa ser lido sequencialmente. Você pode consultar as seções relevantes conforme necessário.
