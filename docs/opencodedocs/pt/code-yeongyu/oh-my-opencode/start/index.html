Nesta página

Início Rápido ​

Esta seção ajuda você a instalar e começar a usar o oh-my-opencode rapidamente, da configuração básica ao orquestrador principal, e depois ao modo Ultrawork que ativa todos os recursos com um clique.

Caminho de Aprendizado ​

Siga esta sequência para dominar progressivamente as funções básicas do oh-my-opencode:

1. Instalação e Configuração Rápidas ​

Conclua a instalação do oh-my-opencode, configuração do Provider e verificação básica.

  • Instalar o oh-my-opencode
  • Configurar o AI Provider (Claude, OpenAI, Gemini, etc.)
  • Verificar se a configuração está correta
  • Entender a estrutura do arquivo de configuração

2. Conheça Sisyphus: O Orquestrador Principal ​

Entenda os conceitos centrais da orquestração de agentes e aprenda o uso básico do agente Sisyphus.

  • O que é um sistema de orquestração de agentes
  • Funções principais do agente Sisyphus
  • Uso básico e melhores práticas
  • Entender o fluxo de trabalho de colaboração entre múltiplos modelos

3. Modo Ultrawork: Ative Todos os Recursos com um Clique ​

Aprenda como usar a palavra-chave ultrawork para ativar rapidamente todos os agentes profissionais e recursos de tarefas paralelas.

  • O que é o modo Ultrawork
  • Como acionar a palavra-chave ultrawork
  • Trabalho colaborativo de 10 agentes profissionais
  • Gerenciamento de tarefas em segundo plano paralelas

Pré-requisitos ​

Antes de começar esta seção, certifique-se de:

  • Node.js 18+ está instalado (versão 20 ou superior é recomendada)
  • Familiaridade com operações básicas de linha de comando
  • Ter pelo menos uma chave de API de serviço de IA (Claude, OpenAI ou Gemini)

Sem chave de API?

Se você ainda não tem uma chave de API de serviço de IA, pode começar solicitando o Claude, que é atualmente a escolha mais recomendada.

Próximos Passos ​

Após concluir esta seção, você pode continuar aprendendo:

  • Plataformas e Integrações - Aprofunde-se na configuração multi-Provider e estratégias de modelos
  • Recursos Avançados - Explore a equipe completa de 10 agentes e recursos avançados
