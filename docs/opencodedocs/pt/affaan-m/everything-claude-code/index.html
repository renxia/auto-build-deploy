Everything Claude Code
Uma coleção comprovada de plugins para Claude Code

Coleção completa de plugins para Claude Code dos vencedores do hackathon da Anthropic. Inclui 9 agents, 14 commands, 8 conjuntos de rules, 11 skills e 15+ hooks para aprimorar o desenvolvimento assistido por IA. Scripts Node.js multiplataforma para Windows, macOS e Linux.

Início Rápido GitHub
Por que escolher Everything Claude Code?

Não comece do zero. Obtenha configurações práticas refinadas por 10+ meses de uso diário.

Testado em Produção

Dos vencedores do hackathon da Anthropic, refinado por 10+ meses de uso diário, construindo produtos reais.

Configuração Completa

9 agents, 14 commands, 8 conjuntos de rules, 11 skills, 15+ hooks, cobrindo todos os cenários de desenvolvimento.

Suporte Multiplataforma

Todos os hooks e scripts reescritos em Node.js, rodando perfeitamente em Windows, macOS e Linux.

Gerenciador de Pacotes Inteligente

Detecta automaticamente npm, pnpm, yarn ou bun, com 6 configurações de prioridade, sem configuração manual necessária.

Aprendizado Contínuo

Extrai automaticamente padrões reutilizáveis de sessões e os salva como learned skills.

Garantia de Qualidade de Código

Fluxo de TDD, revisão de código, auditoria de segurança, garantindo 80%+ de cobertura de testes.
