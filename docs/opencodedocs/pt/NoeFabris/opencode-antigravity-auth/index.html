Tema

Antigravity Auth
Ferramenta de Autenticação Claude & Gemini

Acesse Claude Opus 4.5, Sonnet 4.5 e Gemini 3 Pro/Flash diretamente através de credenciais Google OAuth, sem precisar de API Key. Suporta balanceamento de carga multi-conta e recuperação automática de sessão.

Início Rápido GitHub
Por que escolher Antigravity Auth?

Resolva a dor de cabeça da configuração manual de API Key, gerencie cotas automaticamente, projetado para desenvolvimento de alta concorrência.

Sem API Key

Não precisa solicitar API Keys caras, use credenciais de conta Google diretamente, login com um clique.

Balanceamento Multi-Conta

Suporta rotação automática de até 10 contas Google, distribuição inteligente de requisições, adeus rate limits.

Suporte a Modelos Thinking

Suporte perfeito aos modos thinking do Claude e Gemini 3, configuração de Budget disponível, desbloqueia capacidade de raciocínio profundo.

Recuperação Automática de Sessão

Falha na chamada de ferramenta? Sessão interrompida? O plugin tenta novamente automaticamente e recupera o contexto, sem intervenção manual.

Compatibilidade com Plugins

Integração perfeita com plugins populares como oh-my-opencode, DCP.

Sistema de Cotas Duplo

Maximize o uso dos pools de cotas gratuitas e pagas fornecidos pelo Google.
