Tema

Antigravity-Manager
本地 AI 账号管理与协议反代系统

将多家 AI 平台的 Web/Session/OAuth 身份转化为统一的 API 网关，提供多账号调度、智能轮换、配额监控与多协议兼容能力。基于 Tauri v2 + React + Rust 构建的桌面应用。

快速开始 GitHub
为什么选择 Antigravity-Manager？

你的本地高性能 AI 调度网关。不只是账号管理，更是打破 API 调用壁垒的终极方案。

多协议兼容端点

对外暴露 OpenAI /v1/chat/completions、Anthropic /v1/messages、Gemini 原生 SDK 端点，复用现有客户端与 SDK。

智能账号调度

支持轮换、固定账号、粘性会话与失败重试。遇到 429/401 自动轮换，invalid_grant 自动下线，毫秒级自愈。

OAuth 授权添加

支持浏览器 OAuth 授权与 Refresh Token 双通道添加账号，批量导入导出，V1/DB 热迁移。

配额治理与保护

实时监控各模型剩余配额，支持模型级配额保护与 Smart Warmup，避免无感耗尽。

流量监控与统计

Proxy Monitor 实时记录请求日志，Token Stats 按小时/日/周维度统计消耗，支持按模型与账号分析。

跨平台部署

原生支持 macOS、Windows、Linux 桌面端。提供 Docker noVNC 与 Headless Xvfb 方案，适配 NAS/服务器部署。
