Introdução ​

Este capítulo ajuda-o a instalar rapidamente o plugin opencode-notify, experimentar a funcionalidade de notificações de ambiente de trabalho e compreender os seus mecanismos de funcionamento principais.

Caminho de Aprendizagem ​

1. Início Rápido ​

Complete a instalação e experimente o primeiro aviso em 5 minutos.

  • Instalação com um clique através do OCX
  • Verifique se a funcionalidade de notificações está a funcionar corretamente

2. Como Funciona ​

Compreenda as funcionalidades principais do plugin, os mecanismos de filtragem inteligente e as condições de ativação de notificações.

  • Domine os quatro tipos de notificações e os seus momentos de ativação
  • Compreenda como a filtragem inteligente evita a sobrecarga de notificações

Pré-requisitos ​

  • OpenCode Agent instalado (descarregar do site oficial)
  • Conhecimento básico de operações de terminal

Próximos Passos ​

Após completar este capítulo, pode continuar a aprender:

  • Recursos do macOS - Compreenda a detecção de foco e personalização de sons específicos para macOS
  • Referência de Configuração - Compreenda profundamente todas as opções de configuração
  • Resolução de Problemas - Soluções quando encontrar problemas
