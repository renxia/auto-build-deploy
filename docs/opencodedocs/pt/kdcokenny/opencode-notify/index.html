opencode-notify
Deixe o OpenCode AI lembrá-lo suavemente enquanto você se concentra

Diga adeus à ansiedade de alternar entre janelas constantemente. O opencode-notify oferece notificações desktop nativas para seu OpenCode AI, com suporte a detecção inteligente de foco, períodos silenciosos e clique para focar (macOS), permitindo que você trabalhe com foco e seja acordado precisamente apenas quando a IA precisar de você.

Começar Agora GitHub
Por que escolher o opencode-notify?

Trabalho focado, lembretes inteligentes

🔔 Notificações Desktop Nativas

Usa o sistema de notificações nativo do sistema operacional (macOS Notification Center, Windows Toast, Linux notify-send), integrando-se perfeitamente ao seu fluxo de trabalho.

🧠 Detecção Inteligente de Foco

Detecta automaticamente o foco da janela do terminal, evitando notificações repetidas. Por padrão, notifica apenas a sessão pai, filtrando o ruído de tarefas filhas, permitindo que você se concentre nas informações realmente importantes.

⏰ Configuração de Períodos Silenciosos

Defina períodos silenciosos (por exemplo, 22:00-08:00), suprimindo automaticamente notificações à noite, com suporte a períodos que cruzam a meia-noite, garantindo seu tempo de descanso.

🖱️ Clique para Focar Terminal (macOS)

Usuários macOS podem clicar na notificação para focar automaticamente a janela do terminal, economizando tempo de alternância entre aplicativos e dobrando a eficiência do trabalho.

🎨 Efeitos Sonoros Personalizados (macOS)

Defina efeitos sonoros exclusivos para diferentes tipos de eventos (Glass, Basso, Submarine, etc.), identificando rapidamente o tipo de notificação pelo som.

💻 Suporte a 37+ Terminais

Suporta Ghostty, iTerm2, WezTerm, VS Code Integrated Terminal e mais de 37 emuladores de terminal, identificando automaticamente seu tipo de terminal.

🔒 Execução Local, Zero Risco de Privacidade

Todas as notificações são geradas localmente, sem upload de dados para a nuvem, seu conteúdo de trabalho é completamente privado e seguro.

⚡ Plugin Leve

Construído com TypeScript + Bun, instale e use imediatamente, arquivo de configuração conciso, sem necessidade de aprender configurações complexas.
