opencode-mystatus
AI 平台额度一键查询

OpenCode 插件，一键查询 OpenAI、智谱 AI、Z.ai、Google Cloud、GitHub Copilot 等多个 AI 平台的账号额度使用情况。可视化进度条展示剩余额度，实时显示重置时间倒计时。

快速开始 GitHub
为什么选择 opencode-mystatus？

告别繁琐的多平台登录，一条命令掌握所有 AI 平台额度状态。

多平台整合

支持 OpenAI、智谱 AI、Z.ai、Google Cloud、GitHub Copilot 五大平台，无需分别登录查看。

可视化进度条

直观的进度条展示剩余额度比例，一眼看清使用情况，合理规划 API 调用。

重置时间倒计时

实时显示各平台额度重置时间，帮助你安排开发计划，避免超额使用。

本地安全运行

只读本地认证文件，不上传任何数据，API Key 自动脱敏显示，保护你的隐私。

双重使用方式

支持 /mystatus 斜杠命令和自然语言提问两种方式，随时随地查询额度。

中英双语支持

自动检测系统语言，中英文界面无缝切换，适配全球开发者。
