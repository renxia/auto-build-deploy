Nesta página

Início Rápido ​

Este capítulo ajuda você a começar rapidamente com o plugin opencode-mystatus, da instalação ao uso em apenas 5 minutos.

Caminho de Aprendizado ​

Recomenda-se estudar na seguinte ordem:

1. Início Rápido ​

Complete a instalação e configuração do plugin em 5 minutos, verifique a cota da primeira plataforma de IA.

  • Três métodos de instalação: deixar o AI instalar, instalação manual, instalação a partir de arquivo local
  • Configuração do comando de barra /mystatus
  • Verificação de instalação bem-sucedida

2. Usando mystatus ​

Aprenda os dois métodos de uso: comando de barra e linguagem natural.

  • Uso do comando de barra /mystatus
  • Acionar a ferramenta mystatus com perguntas em linguagem natural
  • Diferença entre os dois métodos e cenários aplicáveis

3. Entendendo a Saída ​

Entenda as barras de progresso, tempo de reinicialização e formato de saída de múltiplas contas.

  • Significado da barra de progresso: blocos sólidos vs blocos vazios
  • Períodos de cota de diferentes plataformas (3 horas, 5 horas, mensal)
  • Formato de saída em cenários de múltiplas contas

Pré-requisitos ​

Antes de começar a estudar, confirme:

  • Já instalou o OpenCode
  • Já configurou informações de autenticação de pelo menos uma plataforma de IA (OpenAI, Zhipu AI, Z.ai, GitHub Copilot ou Google Cloud)

Próximo Passo ​

Após completar este capítulo, você pode continuar aprendendo Funcionalidades da Plataforma, para entender os detalhes da verificação de cota de cada plataforma.
