opencode-mystatus
AI Platform Quota at a Glance

OpenCode plugin for checking account quota usage across multiple AI platforms including OpenAI, Zhipu AI, Z.ai, Google Cloud, and GitHub Copilot. Visual progress bars show remaining quota with real-time reset countdown timers.

Quick Start GitHub
Why opencode-mystatus?

Stop logging into multiple platforms. One command to check all your AI quota status.

Multi-Platform Integration

Supports OpenAI, Zhipu AI, Z.ai, Google Cloud, and GitHub Copilot. No need to log in separately.

Visual Progress Bars

Intuitive progress bars display remaining quota at a glance. Plan your API usage wisely.

Reset Time Countdown

Real-time countdown to quota reset for each platform. Schedule your development work accordingly.

Local & Secure

Reads only local auth files. No data upload. API keys are automatically masked for privacy.

Dual Usage Modes

Use /mystatus slash command or natural language queries. Check quota anytime, anywhere.

Bilingual Support

Auto-detects system language. Seamless switching between Chinese and English interfaces.
