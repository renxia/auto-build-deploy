Thema

Moltbot
Dein privater KI-Assistent, überall verfügbar (ehemals Clawdbot)

Betreibe deinen KI-Assistenten auf eigenen Geräten, nahtlos verbunden mit WhatsApp, Telegram, Slack und 12+ weiteren Kanälen. Unterstützt Browser-Automatisierung, Multi-Geräte-Synchronisation, Sprachinteraktion - alle Daten bleiben lokal, volle Privatsphäre.

Schnellstart GitHub
Warum Moltbot?

Leistungsstarker, datenschutzfreundlicher und einfach zu installierender persönlicher KI-Assistent

Multi-Kanal-Integration

Eine Konfiguration für WhatsApp, Telegram, Slack und 12+ weitere Messaging-Plattformen

Local-First

Alle Daten und die KI-Laufzeit verbleiben auf deinem Gerät – keine Datenschutzrisiken

Leistungsstarke Tools

Browser-Automatisierung, Cron-Jobs, Webhooks u. v. m.

Multi-Geräte-Sync

macOS/iOS/Android-Knoten können lokale Operationen ausführen

Sprachinteraktion

Unterstützt Voice Wake, Talk Mode und verschiedene TTS-Engines

Gedächtnissystem

Vektorspeicherung und semantische Suche - die KI erinnert sich an alle Gespräche

Sicherheitsisolierung

Sandbox-Isolierung, DM-Pairing-Schutz, Tool-Berechtigungskontrolle
