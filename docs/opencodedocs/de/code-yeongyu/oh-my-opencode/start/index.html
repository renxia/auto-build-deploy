Auf dieser Seite

Schnellstart ​

Dieser Abschnitt hilft Ihnen, oh-my-opencode schnell zu installieren und zu verwenden – von der Grundkonfiguration über den Haupt-Orchestrator bis zum Ultrawork-Modus, der alle Funktionen mit einem Klick aktiviert.

Lernpfad ​

Befolgen Sie diese Reihenfolge, um die grundlegenden Funktionen von oh-my-opencode schrittweise zu beherrschen:

1. Schnelle Installation und Konfiguration ​

Absolvieren Sie die Installation von oh-my-opencode, die Provider-Konfiguration und die grundlegende Überprüfung.

  • Installieren Sie oh-my-opencode
  • Konfigurieren Sie den AI-Provider (Claude, OpenAI, Gemini usw.)
  • Überprüfen Sie, ob die Konfiguration korrekt ist
  • Verstehen Sie die Struktur der Konfigurationsdatei

2. Kennenlernen von Sisyphus: Der Haupt-Orchestrator ​

Verstehen Sie die Kernkonzepte der Agenten-Orchestrierung und lernen Sie die grundlegende Verwendung des Sisyphus-Agenten.

  • Was ist ein Agenten-Orchestrierungssystem
  • Kernfunktionen des Sisyphus-Agenten
  • Grundlegende Verwendung und Best Practices
  • Verstehen Sie den Workflow der Multi-Modell-Zusammenarbeit

3. Ultrawork-Modus: Alle Funktionen mit einem Klick aktivieren ​

Lernen Sie, wie Sie das ultrawork-Schlüsselwort verwenden, um schnell alle professionellen Agenten und parallelen Aufgabenfunktionen zu aktivieren.

  • Was ist der Ultrawork-Modus
  • Wie Sie das ultrawork-Schlüsselwort auslösen
  • Zusammenarbeit von 10 professionellen Agenten
  • Verwaltung paralleler Hintergrundaufgaben

Voraussetzungen ​

Bevor Sie mit diesem Abschnitt beginnen, stellen Sie sicher:

  • Node.js 18+ ist installiert (Version 20 oder höher wird empfohlen)
  • Sie sind mit grundlegenden Befehlszeilenoperationen vertraut
  • Sie verfügen über mindestens einen API-Schlüssel für einen KI-Dienst (Claude, OpenAI oder Gemini)

Kein API-Schlüssel?

Wenn Sie noch keinen API-Schlüssel für einen KI-Dienst haben, können Sie mit Claude beginnen. Dies ist derzeit die empfohlene Wahl.

Nächste Schritte ​

Nach Abschluss dieses Abschnitts können Sie mit folgenden Themen fortfahren:

  • Plattformen und Integrationen - Vertiefen Sie Ihr Wissen über Multi-Provider-Konfiguration und Modellstrategien
  • Erweiterte Funktionen - Erkunden Sie das vollständige Team aus 10 Agenten und erweiterte Funktionen
