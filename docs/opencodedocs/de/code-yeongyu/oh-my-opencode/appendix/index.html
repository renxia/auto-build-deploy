Anhang: Referenzmaterial und Kompatibilitätsleitfaden ​

Dieses Kapitel enthält detaillierte Referenzmaterialien und Kompatibilitätsleitfäden für oh-my-opencode, einschließlich vollständiger Konfigurationsbeschreibungen, Claude Code-Migrationsunterstützung und detaillierten Einführungen in die integrierten Erweiterungsserver.

Kapitelinhalt ​

Dieses Kapitel enthält drei wichtige Abschnitte:

UnterseiteBeschreibungSchwierigkeit
Claude Code-KompatibilitätVollständiger Leitfaden für die Migration von Claude Code zu OpenCode, einschließlich Kompatibilitätsmechanismen für Commands, Skills, Agents, MCPs und Hooks⭐⭐
KonfigurationsreferenzVollständige Schema-Beschreibung der oh-my-opencode-Konfigurationsdatei, einschließlich aller Felder, Typen und Standardwerte⭐⭐⭐
Integrierte MCPsFunktionen und Verwendungsmethoden der 3 integrierten MCP-Server (Exa Websearch, Context7, Grep.app)⭐⭐

Empfohlener Lernpfad ​

Wählen Sie die Lernreihenfolge basierend auf Ihren Anforderungen:

Pfad 1: Migration von Claude Code ​

Wenn Sie ein Benutzer sind, der von Claude Code migriert:

  1. Lesen Sie zuerst Claude Code-Kompatibilität, um zu erfahren, wie Sie Ihre vorhandene Konfiguration nahtlos migrieren können
  2. Sehen Sie sich dann Konfigurationsreferenz an, um die verfügbaren Konfigurationsoptionen besser zu verstehen
  3. Lernen Sie schließlich Integrierte MCPs, um zu erfahren, wie Sie zusätzliche Suchfunktionen konfigurieren können

Pfad 2: Tiefgreifende Konfigurationsanpassung ​

Wenn Sie das Verhalten von oh-my-opencode tiefgreifend anpassen möchten:

  1. Beginnen Sie mit Konfigurationsreferenz, um alle konfigurierbaren Elemente zu verstehen
  2. Lernen Sie Integrierte MCPs, um Such- und Dokumentabfragefunktionen zu konfigurieren
  3. Sehen Sie sich Claude Code-Kompatibilität an, um die Konfigurationsoptionen der Kompatibilitätsschicht zu verstehen

Pfad 3: Schnelle Referenz ​

Wenn Sie nur bestimmte Informationen nachschlagen müssen:

  • Konfigurationsfragen → Sehen Sie sich direkt Konfigurationsreferenz an
  • Migrationsfragen → Sehen Sie sich direkt Claude Code-Kompatibilität an
  • MCP-Konfiguration → Sehen Sie sich direkt Integrierte MCPs an

Voraussetzungen ​

Es wird empfohlen, dass Sie vor dem Studium dieses Kapitels bereits:

  • ✅ Installation und Konfiguration abgeschlossen haben
  • ✅ Die Grundkonzepte des Sisyphus-Orchestrators verstanden haben
  • ✅ Mit der Bearbeitung von JSON-Konfigurationsdateien vertraut sind

Nächste Schritte ​

Nach Abschluss dieses Kapitels können Sie:

  • 🚀 Erweiterte Funktionen ausprobieren, um mehr fortgeschrittene Verwendungsmöglichkeiten zu lernen
  • 🔧 Häufig gestellte Fragen ansehen, um Probleme bei der Verwendung zu lösen
  • 📖 Änderungsprotokoll lesen, um die neuesten Funktionsverbesserungen zu erfahren

Hinweis

Der Inhalt dieses Kapitels dient hauptsächlich als Referenzmaterial und muss nicht in der angegebenen Reihenfolge gelesen werden. Sie können jederzeit den entsprechenden Abschnitt nach Bedarf nachschlagen.
