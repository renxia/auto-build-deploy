Everything Claude Code
Battle-tested Claude Code Plugin-Konfigurations-Sammlung

Die vollständige Claude Code Plugin-Konfigurationssammlung vom Gewinner des Anthropic Hackathons. Bietet 9 Agents, 14 Commands, 8 Rulesets, 11 Skills und 15+ Hooks zur Verbesserung der KI-gestützten Entwicklung. Unterstützt Windows, macOS und Linux mit Node.js-Skripten.
