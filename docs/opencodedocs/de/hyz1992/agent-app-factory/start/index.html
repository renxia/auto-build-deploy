Auf dieser Seite

Schnellstart ​

Dieses Kapitel hilft Ihnen, AI App Factory schnell zu starten, den Kernwert zu verstehen, die Tools zu installieren und zu konfigurieren sowie die Funktionsweise der 7-Phasen-Pipeline zu begreifen, um sich auf die Generierung Ihrer ersten Anwendung vorzubereiten.

Lernpfad ​

Es wird empfohlen, den Inhalt dieses Kapitels in der folgenden Reihenfolge zu erlernen:

1. Schnellstart: Von der Idee zur Anwendung ​

Lernen Sie den Kernwert und die Voraussetzungen von AI App Factory kennen und erleben Sie durch ein vollständiges Beispiel den End-to-End-Prozess von der Idee bis zur Anwendung.

  • Verstehen des Checkpoint-basierten Multi-Agent-Kooperationsmechanismus
  • Verstehen des vollständigen Arbeitsablaufs der 7-Phasen-Pipeline
  • Beherrschen der Techniken für klare Produktbeschreibungen

2. Installation und Konfiguration ​

Installieren Sie das CLI-Tool global und konfigurieren Sie den AI-Assistenten (Claude Code/OpenCode) sowie die zugehörige Umgebung.

  • Globale Installation von agent-app-factory über npm
  • Konfiguration von Claude Code oder OpenCode
  • Überprüfung der erfolgreichen Installation

3. Factory-Projekt initialisieren ​

Initialisieren Sie das Verzeichnis mit dem Befehl factory init und verstehen Sie die generierte Projektstruktur und Konfigurationsdateien.

  • Ausführen des Projektinitialisierungsbefehls
  • Verstehen der Projektverzeichnisstruktur und Konfigurationsdateien
  • Vorbereiten der Produktidee-Eingabe

4. Übersicht über die 7-Phasen-Pipeline ​

Vertiefen Sie Ihr Verständnis des vollständigen Pipeline-Prozesses von Bootstrap bis Preview, der Eingaben und Ausgaben der einzelnen Phasen sowie des Checkpoint-Mechanismus.

  • Verstehen der Ziele und Ergebnisse der 7 Phasen
  • Beherrschen des Checkpoint-Mechanismus und des manuellen Bestätigungsprozesses
  • Lernen, wie Sie die Pipeline-Ausführung steuern

Voraussetzungen ​

Bevor Sie mit diesem Kapitel beginnen, benötigen Sie:

Voraussetzungen

  1. Node.js-Umgebung: Node.js v18 oder höher ist installiert
  2. KI-Programmierassistent: Eines der folgenden Tools muss konfiguriert sein
    • Claude Code (empfohlen) ⭐
    • OpenCode oder ein anderer KI-Assistent mit Agent-Modus
  3. Grundlegende Befehlszeilenoperationen: Fähigkeit, Befehle im Terminal auszuführen

Nächste Schritte ​

Nach Abschluss dieses Kapitels werden Sie folgendes beherrschen:

  • ✅ Die Kernarbeitsweise von AI App Factory
  • ✅ Installation und Konfiguration des CLI-Tools
  • ✅ Projektinitialisierung und Verständnis der Struktur
  • ✅ Der vollständige Ablauf der 7-Phasen-Pipeline

Als Nächstes wird empfohlen, das Kapitel Plattformen und Integrationen zu erlernen, um zu verstehen, wie Sie die Pipeline auf verschiedenen KI-Assistenten ausführen.
