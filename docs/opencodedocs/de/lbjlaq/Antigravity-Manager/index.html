Thema

Antigravity-Manager
Lokale KI-Kontoverwaltung und Protokoll-Reverse-Proxy-System

Wandelt Web/Session/OAuth-Identitäten mehrerer KI-Plattformen in ein einheitliches API-Gateway um. Bietet Multi-Account-Dispatch, intelligente Rotation, Kontingentüberwachung und Multi-Protokoll-Kompatibilität. Eine Desktop-Anwendung basierend auf Tauri v2 + React + Rust.

Schnellstart GitHub
Warum Antigravity-Manager?

Ihr leistungsstarker lokaler KI-Dispatch-Gateway. Nicht nur Kontoverwaltung, sondern die ultimative Lösung zum Durchbrechen von API-Aufrufbarrieren.

Multi-Protokoll-kompatible Endpunkte

Bietet OpenAI /v1/chat/completions, Anthropic /v1/messages und native Gemini SDK-Endpunkte nach außen. Wiederverwendung vorhandener Clients und SDKs.

Intelligente Kontoverwaltung

Unterstützt Rotation, feste Accounts, Sticky Sessions und Wiederholungsversuche bei Fehlern. Bei 429/401 automatische Rotation, bei invalid_grant automatische Deaktivierung, Millisekunden-Selbstheilung.

OAuth-Autorisierung hinzufügen

Unterstützt Browser-OAuth-Autorisierung und Refresh-Token-Dualkanal zum Hinzufügen von Konten. Massenimport/-export, V1/DB-Hot-Migration.

Kontingentverwaltung und Schutz

Echtzeitüberwachung der verbleibenden Kontingente aller Modelle, support für modellbasierten Kontingentschutz und Smart Warmup, um unbemerktes Aufbrauchen zu vermeiden.

Verkehrsüberwachung und Statistiken

Proxy Monitor protokolliert Anfragen in Echtzeit, Token Stats verbraucht nach Stunde/Tag/Woche, Unterstützung für Analyse nach Modell und Account.

Plattformübergreifende Bereitstellung

Nativ unterstützt für macOS, Windows und Linux Desktop. Bietet Docker noVNC und Headless Xvfb Lösungen, angepasst für NAS/Server-Bereitstellung.
