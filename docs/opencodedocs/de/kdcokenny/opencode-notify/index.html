Thema

opencode-notify
Lassen Sie OpenCode AI Sie sanft erinnern, wenn Sie fokussiert sind

Verabschieden Sie sich von der Angst, ständig Fenster wechseln zu müssen. opencode-notify bietet native Desktop-Benachrichtigungen für Ihren OpenCode AI mit intelligenter Fokuserkennung, lautlosen Zeiten und Klick-zum-Fokus (macOS), damit Sie konzentriert arbeiten und nur dann präzise benachrichtigt werden, wenn die AI Sie braucht.

Schnellstart GitHub
Warum opencode-notify?

Fokussierte Arbeit, intelligente Erinnerungen

🔔 Native Desktop-Benachrichtigungen

Nutzt das native Benachrichtigungssystem des Betriebssystems (macOS Notification Center, Windows Toast, Linux notify-send) und integriert sich nahtlos in Ihren Workflow.

🧠 Intelligente Fokuserkennung

Erkennt automatisch den Fokus des Terminalfensters, um doppelte Benachrichtigungen zu vermeiden. Standardmäßig nur für übergeordnete Sitzungen, Filterung von Unterprozess-Rauschen, damit Sie sich auf die wirklich wichtigen Informationen konzentrieren können.

⏰ Lautlose Zeiten konfigurieren

Legen Sie lautlose Zeitfenster fest (z.B. 22:00-08:00), unterdrückt Benachrichtigungen nachts automatisch, unterstützt Zeiträume über Mitternacht hinweg, um Ihre Ruhezeiten zu schützen.

🖱️ Klick-zum-Fokus Terminal (macOS)

macOS-Benutzer können das Terminalfenster automatisch fokussieren, indem sie auf die Benachrichtigung klicken, sparen Zeit beim Wechseln zwischen Anwendungen und verdoppeln die Arbeitseffizienz.

🎨 Benutzerdefinierte Soundeffekte (macOS)

Legen Sie exklusive Soundeffekte für verschiedene Ereignistypen fest (Glass, Basso, Submarine usw.), identifizieren Sie Benachrichtigungstypen schnell durch den Klang.

💻 37+ Terminal-Unterstützung

Unterstützt Ghostty, iTerm2, WezTerm, VS Code integriertes Terminal und mehr als 37 Terminal-Emulatoren, erkennt Ihren Terminaltyp automatisch.

🔒 Lokal ausgeführt, null Datenschutzrisiko

Alle Benachrichtigungen werden lokal generiert, keine Daten werden in die Cloud hochgeladen, Ihre Arbeitsinhalte bleiben vollständig privat und sicher.

⚡ Leichtgewichtiges Plugin

Aufgebaut mit TypeScript + Bun, installierbar und sofort einsatzbereit, schlanke Konfigurationsdatei, keine komplexe Konfiguration erforderlich.
