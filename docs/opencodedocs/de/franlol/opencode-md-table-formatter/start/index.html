Schnellstart ​

Dieses Kapitel enthält Einstiegsinhalte für den OpenCode Markdown Table Formatter und hilft Ihnen, das Plugin schnell zu installieren und seine Kernfunktionen kennenzulernen.

Kapitelübersicht ​

Dieses Kapitel führt Sie durch die Installation und Konfiguration des Plugins und gibt Ihnen einen umfassenden Überblick über die 8 Kernfunktionen der automatischen Formatierung. In nur wenigen Minuten werden Sie die magischen Effekte der Tabellenformatierung sehen.

Es wird empfohlen, in folgender Reihenfolge zu lernen:
