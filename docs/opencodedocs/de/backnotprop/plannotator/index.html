Thema

plannotator
KI-basiertes Planungs- und Code-Review-Tool

Bietet interaktive Planungs- und Code-Reviews für Claude Code und OpenCode mit Unterstützung für visuelle Anmerkungen, Team-Kollaboration, URL-Freigabe und Integration von Drittanbieter-Notizen. Lokal betrieben, sichere Daten, keine Server erforderlich für den Austausch von Reviews.

Schnellstart GitHub
Warum Plannotator wählen?

Visuelles Review von KI-generierten Plänen und Code, präzises Feedback, Team-Kollaboration.

Visuelles Planungs-Review

Betrachten Sie im Browser KI-generierte Pläne und geben Sie präzises Feedback mit Löschen, Ersetzen, Einfügen und Kommentar-Anmerkungen.

Visualisierte Code-Reviews

Git diff side-by-side oder unified Ansicht, unterstützt Zeilen-Kommentare und Code-Vorschläge.

Multi-Plattform-Unterstützung

Unterstützung für Claude Code und OpenCode, nahtlose Integration in bestehende Entwicklungs-Workflows.

URL-Freigabe ohne Backend

Komprimiert in URL hash, keine Server erforderlich, teilen Sie Reviews mit Teammitgliedern.

Bild-Annotationstool

Hängen Sie Bilder an und verwenden Sie Stift, Pfeile, Kreis-Tools zur Annotation, um das Feedback zu verbessern.

Integration von Drittanbieter-Notizen

Genehmigte Pläne werden automatisch in Obsidian oder Bear gespeichert, inklusive frontmatter und Tags.
