Thema

Antigravity Auth
Claude & Gemini Auth-Tool

Zugriff auf Claude Opus 4.5, Sonnet 4.5 und Gemini 3 Pro/Flash ohne API Key, direkt über Google OAuth-Credentials. Unterstützt Multi-Account-Load-Balancing und automatische Sitzungswiederherstellung.

Schnellstart GitHub
Warum Antigravity Auth?

Beseitigt die Mühe der manuellen API-Key-Konfiguration, automatisiert das Quota-Management, speziell für hochparallele Entwicklung konzipiert.

Kein API Key nötig

Keine teuren API Keys beantragen, direkt Google-Account-Credentials verwenden, One-Click-Authentifizierung.

Multi-Account Load Balancing

Bis zu 10 Google-Accounts automatisch rotieren, intelligente Request-Verteilung, Goodbye Rate-Limits.

Thinking Model Support

Perfekte Unterstützung für Claude und Gemini 3 Thinking-Modi, konfigurierbares Budget, Deep-Reasoning freischalten.

Automatische Sitzungswiederherstellung

Tool-Call fehlgeschlagen? Sitzung unterbrochen? Plugin führt automatisch Retry durch und stellt den Kontext wieder her, keine manuelle Eingriffe nötig.

Plugin-Kompatibilität

Nahtlose Integration mit oh-my-opencode, DCP und anderen Mainstream-Plugins.

Duales Quota-System

Maximale Nutzung der kostenlosen und kostenpflichtigen Quota-Pools von Google.
