Thema

opencode-agent-skills
OpenCode AI Skills dynamisches Verwaltungs-Plugin

Ein Plugin-System, das OpenCode dynamische Skill-Entdeckungs- und -Verwaltungsfähigkeiten bietet, mit Unterstützung für AI Agents, um wiederverwendbare Skill-Bibliotheken zu laden und zu verwenden. Ermöglicht projekt- und sitzungsübergreifende Wiederverwendung von Skills mit automatischer Empfehlung relevanter Skills basierend auf semantischer Ähnlichkeit.

Schnellstart GitHub
Warum OpenCode Agent Skills?

Bietet OpenCode dynamische Skill-Entdeckungs- und -Verwaltungsfähigkeiten für projekt- und sitzungsübergreifende Skill-Wiederverwendung.

Dynamische Skill-Entdeckung

Automatisches Entdecken von Skills aus Projekt-, Benutzer- und Plugin-Verzeichnissen mit Unterstützung für das Scannen von 6 Positionen und Rückgabe des ersten passenden Skills nach Priorität.

Claude Code Kompatibilität

Unterstützt vorhandene Claude Code Skills und Plugin-Formate, reduziert Migrationskosten und ermöglicht die Wiederverwendung vorhandener Ressourcen.

Semantische Übereinstimmungsempfehlung

Verwendet HuggingFace Embedding-Modelle zur Ähnlichkeitsberechnung und empfiehlt automatisch relevante Skills basierend auf Benutzereingaben.

Kontext-Injektion

Skill-Inhalte werden im XML-Format in den Sitzungskontext injiziert, wobei der AI den Skill-Anweisungen folgt, um qualitativ hochwertige Ergebnisse zu erzeugen.

Wiederherstellung nach Komprimierung

Automatisches erneutes Injizieren der Skill-Liste nach Sitzungskomprimierung, um die dauerhafte Verfügbarkeit von Skills in langen Sitzungen zu gewährleisten.

Skriptausführungsunterstützung

Skill-Verzeichnisse können ausführbare Skripte enthalten, die im Skill-Verzeichniskontext zur Ausführung automatisierter Aufgaben genutzt werden.
