Thème

opencode-md-table-formatter
Alignement Automatique de Tableaux IA

Plugin OpenCode qui formate automatiquement les tableaux Markdown générés par l'IA. Reconnaissance intelligente de la largeur du contenu, support du gras, italique, blocs de code et autres formats, pour des tableaux toujours impeccables.

Voir le Tutoriel GitHub
Fonctionnalités Principales

Reconnaissance automatique de la syntaxe Markdown, calcul intelligent de la largeur d'affichage

Reconnaissance Intelligente des Formats Markdown

Calcul correct de la largeur d'affichage du gras, italique, blocs de code et autres formats, garantissant un alignement parfait des tableaux

Formatage Automatique Après Génération IA

Déclenchement automatique via le hook experimental.text.complete, formatage sans manipulation manuelle après génération de texte par l'IA

Protection Intelligente des Blocs de Code

Protection des symboles Markdown dans les blocs de code en ligne contre l'extraction incorrecte, préservant la syntaxe du code

Support de Trois Modes d'Alignement

Support de trois formats : alignement gauche (:---), centré (:---:), droite (---:)

Traitement du Markdown Imbriqué

Algorithme multi-passes avec expressions régulières pour traiter la syntaxe Markdown imbriquée comme le gras, italique, barré

Compatibilité des Cas Limites

Traitement correct des cas limites : Emoji, caractères Unicode, cellules vides, contenu long

Exécution Silencieuse

Aucune sortie console, les erreurs n'interrompent pas le flux de travail, formatage silencieux

Feedback de Validation

Ajout de commentaires d'erreur conviviaux pour les tableaux invalides, aidant les utilisateurs à localiser rapidement les problèmes
