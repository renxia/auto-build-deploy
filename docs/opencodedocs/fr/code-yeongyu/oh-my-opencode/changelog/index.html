Journal des modifications : Évolution des versions et suivi des fonctionnalités ​

Aperçu de la section ​

Ce chapitre enregistre l'historique des versions de oh-my-opencode, y compris les nouvelles fonctionnalités, les améliorations importantes, les améliorations fonctionnelles et les changements de compatibilité. En lisant le journal des modifications, vous pouvez comprendre le parcours d'évolution du projet et saisir rapidement les améliorations de fonctionnalités de la dernière version.

Navigation des sous-pages ​

VersionTitreDescriptionStatut de publication
v3.0Fonctionnalités v3.0Introduit les systèmes tout nouveaux de Catégories et Compétences, 7 catégories de tâches intégrées, 3 packages de compétences professionnelles, l'exécuteur Sisyphus-Junior et les capacités de composition dynamique d'agents✅ Stable

Recommandations de parcours d'apprentissage ​

Les entrées du journal des modifications dans cette section sont classées de la version la plus récente à la plus ancienne. Nous recommandons les lectures suivantes :

  • 🎯 Compréhension rapide des dernières fonctionnalités : Lisez directement le journal des modifications de la dernière version (par exemple, v3.0)
  • 📚 Guide de mise à niveau de version : Commencez à partir de votre version actuelle et lisez dans l'ordre inverse pour comprendre le chemin de mise à niveau
  • 🔍 Exploration approfondie des nouvelles fonctionnalités : Lisez la documentation détaillée des versions qui vous intéressent, ainsi que les chapitres de tutoriels avancés correspondants

Prérequis ​

Avant de lire le journal des modifications de ce chapitre, nous recommandons d'avoir terminé :

  • ✅ Installation rapide et configuration - Comprendre l'installation de oh-my-opencode et la configuration de base
  • ✅ Premiers pas avec Sisyphus : L'orchestrateur principal - Comprendre les concepts de base et l'utilisation des agents Sisyphus

Étapes suivantes ​

Après avoir lu le journal des modifications, vous pouvez continuer à apprendre :

  • Plongée approfondie dans les nouvelles fonctionnalités :

    • Catégories et Compétences : Composition dynamique d'agents - Apprendre l'utilisation détaillée du nouveau système v3.0
    • Compétences intégrées : Automatisation du navigateur et expert Git - Maîtriser l'utilisation des 3 packages de compétences professionnelles intégrés
    • Personnalisation avancée : Agents et gestion des autorisations - Comprendre comment personnaliser les Catégories et Compétences

  • Explorer les fonctionnalités avancées :

    • Équipe d'agents IA : Aperçu des 10 experts - Compréhension complète des 10 agents intégrés
    • Planification Prometheus : Collecte des besoins par entretien - Apprendre à utiliser l'agent Prometheus pour la planification de projet
    • Tâches parallèles en arrière-plan : Travailler comme une équipe - Maîtriser le système de gestion des agents en arrière-plan
