Annexe : Guide de référence et compatibilité ​

Ce chapitre contient des matériaux de référence détaillés et des lignes directrices de compatibilité pour oh-my-opencode, couvrant la documentation complète de configuration, le support de migration Claude Code et des présentations détaillées des serveurs d'extension intégrés.

Contenu du chapitre ​

Ce chapitre comprend trois sections importantes :

Sous-pageDescriptionDifficulté
Compatibilité Claude CodeGuide complet pour migrer de Claude Code vers OpenCode, incluant les mécanismes de compatibilité pour Commands, Skills, Agents, MCPs et Hooks⭐⭐
Référence de configurationDocumentation complète du Schema pour les fichiers de configuration oh-my-opencode, couvrant tous les champs, types et valeurs par défaut⭐⭐⭐
MCP intégrésFonctionnalités et utilisation des 3 serveurs MCP intégrés (Exa Websearch, Context7, Grep.app)⭐⭐

Parcours d'apprentissage recommandés ​

Choisissez votre ordre d'apprentissage en fonction de vos besoins :

Parcours 1 : Migration depuis Claude Code ​

Si vous êtes un utilisateur migrant depuis Claude Code :

  1. Lisez d'abord Compatibilité Claude Code pour comprendre comment migrer sans problème votre configuration existante
  2. Puis consultez Référence de configuration pour approfondir votre compréhension des options de configuration disponibles
  3. Enfin, découvrez MCP intégrés pour comprendre comment configurer des capacités de recherche supplémentaires

Parcours 2 : Personnalisation approfondie ​

Si vous souhaitez personnaliser en profondeur le comportement d'oh-my-opencode :

  1. Commencez par Référence de configuration pour comprendre tous les éléments configurables
  2. Découvrez MCP intégrés pour configurer les capacités de recherche et de requête de documentation
  3. Référez-vous à Compatibilité Claude Code pour comprendre les options de configuration de la couche de compatibilité

Parcours 3 : Référence rapide ​

Si vous avez seulement besoin de rechercher des informations spécifiques :

  • Questions de configuration → Allez directement à Référence de configuration
  • Questions de migration → Allez directement à Compatibilité Claude Code
  • Configuration MCP → Allez directement à MCP intégrés

Prérequis ​

Pour ce chapitre, il est recommandé d'avoir déjà :

  • ✅ Complété Installation et configuration
  • ✅ Compris les concepts de base de Orchestrateur Sisyphus
  • ✅ Une familiarité avec l'édition de fichiers de configuration JSON

Prochaines étapes ​

Après avoir terminé ce chapitre, vous pouvez :

  • 🚀 Essayer Fonctionnalités avancées pour en savoir plus sur l'utilisation avancée
  • 🔧 Consulter FAQ pour résoudre les problèmes d'utilisation
  • 📖 Lire Journal des modifications pour rester informé des nouvelles améliorations de fonctionnalités

Astuce

Le contenu de ce chapitre sert principalement de matériel de référence et n'a pas besoin d'être lu séquentiellement. Vous pouvez consulter les sections pertinentes selon vos besoins.
