Journal des modifications ​

Ce chapitre enregistre l'évolution des versions et les notes de publication d'Antigravity Tools. Avant de mettre à niveau, consultez d'abord le Changelog pour réduire la probabilité de rencontrer des changements de compatibilité ou des problèmes de régression.

Contenu de ce chapitre ​

PageDescription
Évolution des versionsComment lire le Changelog intégré dans le README, juger rapidement l'impact de la mise à niveau et effectuer la vérification après mise à niveau

Parcours d'apprentissage ​

  1. Avant la mise à niveau : d'abord confirmer la version actuelle dans Settings → About, puis aller au Changelog du README et ne lire que les sections de versions que vous avez sautées
  2. Après la mise à niveau : exécuter une vérification minimale /healthz pour confirmer que le proxy fonctionne toujours normalement

Étapes suivantes ​

  • Si vous n'avez pas encore installé ou souhaitez comprendre le mode d'installation, consultez d'abord Installation et mise à niveau
  • Si vous souhaitez comprendre l'utilisation de base du proxy, consultez Démarrer le proxy inverse local et connecter le premier client
