Thème

opencode-agent-skills
Plugin de gestion dynamique de compétences OpenCode AI

Système de plugins offrant des capacités de découverte et de gestion dynamiques de compétences pour OpenCode, permettant aux AI Agents de charger et d'utiliser des bibliothèques de compétences réutilisables. Implémente la réutilisation de compétences inter-projets et inter-sessions, avec recommandation automatique de compétences pertinentes basée sur la similarité sémantique.

Démarrage rapide GitHub
Pourquoi choisir OpenCode Agent Skills ?

Fournit des capacités de découverte et de gestion dynamiques de compétences pour OpenCode, permettant la réutilisation de compétences inter-projets et inter-sessions.

Découverte dynamique de compétences

Découvre automatiquement les compétences à partir des répertoires de projets, utilisateurs et plugins, prend en charge l'analyse de 6 emplacements et renvoie la première compétence correspondante par priorité.

Compatibilité Claude Code

Prend en charge les compétences et formats de plugins Claude Code existants, réduisant les coûts de migration et permettant la réutilisation des ressources existantes.

Recommandation par correspondance sémantique

Utilise le modèle d'embedding HuggingFace pour calculer la similarité et recommander automatiquement les compétences pertinentes en fonction de l'entrée utilisateur.

Injection contextuelle

Le contenu des compétences est injecté dans le contexte de session au format XML, permettant à l'IA de produire des résultats de haute qualité en suivant les instructions des compétences.

Restauration après compression

Réinjecte automatiquement la liste des compétences après la compression de session, maintenant la disponibilité persistante des compétences dans les sessions de longue durée.

Support d'exécution de scripts

Les répertoires de compétences peuvent contenir des scripts exécutables, permettant l'exécution de tâches automatisées dans le contexte du répertoire de compétences.
