Plateformes et intégrations ​

Ce chapitre vous aide à configurer votre environnement de développement IA et à installer les plugins nécessaires pour garantir le bon fonctionnement du pipeline.

Parcours d'apprentissage ​

1. Guide d'intégration Claude Code ​

Apprenez à utiliser Claude Code pour exécuter le pipeline, y compris la configuration des permissions et les bonnes pratiques.

  • Configurer les permissions de Claude Code
  • Comprendre les restrictions d'accès aux fichiers
  • Maîtriser les étapes d'exécution du pipeline

2. OpenCode et autres assistants IA ​

Découvrez comment utiliser OpenCode, Cursor et d'autres assistants IA pour exécuter le pipeline.

  • Configurer OpenCode CLI
  • Paramétrer un projet Cursor
  • Notes de compatibilité

3. Installation des plugins requis ​

Installez les plugins superpowers et ui-ux-pro-max pour garantir le bon fonctionnement du pipeline.

  • Installer le plugin superpowers
  • Installer le plugin ui-ux-pro-max
  • Vérifier la disponibilité des plugins

Prérequis ​

Avant de commencer ce chapitre, assurez-vous d'avoir complété :

Prérequis

  • Démarrage rapide terminé
  • Installation et configuration terminée
  • Initialisation d'un projet Factory terminée

Prochaines étapes ​

Après avoir terminé ce chapitre, nous vous recommandons de poursuivre avec :
