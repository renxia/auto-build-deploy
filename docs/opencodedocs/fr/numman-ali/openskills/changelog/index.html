Changelog ​

Cette section consigne l'historique des versions d'OpenSkills pour vous aider à comprendre les nouvelles fonctionnalités, les améliorations et les corrections de bugs de chaque version.

Contenu de ce chapitre ​

PageDescription
ChangelogEnregistrement complet des modifications de version, incluant les nouvelles fonctionnalités, améliorations, corrections de bugs et mises à jour de sécurité

Comment utiliser le Changelog ​

Le changelog est organisé par ordre décroissant des numéros de version (la version la plus récente en premier), et chaque version contient les catégories suivantes :

  • Nouvelles fonctionnalités - Nouvelles fonctions et commandes ajoutées
  • Améliorations - Optimisations et renforcements des fonctionnalités existantes
  • Corrections de bugs - Bugs corrigés
  • Sécurité - Améliorations liées à la sécurité

Conseil de mise à niveau

Avant de mettre à jour OpenSkills, il est recommandé de consulter le changelog pour comprendre les changements de version, en prêtant une attention particulière à :

  • Breaking Changes (changements cassants)
  • Les méthodes d'utilisation des nouvelles fonctionnalités
  • L'état des corrections des problèmes connus

Prochaines étapes ​

  • Consultez les FAQ pour trouver des réponses aux questions courantes
  • Vous rencontrez un problème ? Consultez le Guide de dépannage pour trouver des solutions
  • Vous voulez en savoir plus ? Consultez la Référence CLI pour une documentation complète des commandes
