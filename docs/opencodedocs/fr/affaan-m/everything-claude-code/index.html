Everything Claude Code
Une collection de configurations Claude Code éprouvées en production

Une collection complète de configurations de plugins Claude Code du gagnant de l'Anthropic hackathon. Offre 9 agents, 14 commandes, 8 jeux de règles, 11 skills et 15+ hooks pour améliorer l'efficacité du développement assisté par IA. Prend en charge les scripts Node.js multiplateformes Windows, macOS et Linux.
