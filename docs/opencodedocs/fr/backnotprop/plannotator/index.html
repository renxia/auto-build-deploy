Thème

plannotator
Outil de planification et de revue de code basé sur l'IA

Fournit des outils de revue de plan et de code interactifs pour Claude Code et OpenCode, avec support pour les annotations visuelles, la collaboration d'équipe, le partage d'URL et l'intégration avec des applications de notes tierces. Fonctionne localement, sécurité des données, partage de revues sans serveur.

Démarrage rapide GitHub
Pourquoi choisir Plannotator ?

Révisez visuellement les plans et le code générés par l'IA, retours précis, collaboration d'équipe.

Revue de plan visuelle

Consultez les plans générés par l'IA dans le navigateur, ajoutez des annotations de suppression, remplacement, insertion et commentaires pour un retour précis.

Revue de code visuelle

Vue côte à côte ou unifiée des diff Git, support pour les annotations au niveau des lignes et les suggestions de code.

Support multi-plateforme

Compatible avec Claude Code et OpenCode, s'intègre de manière transparente dans les flux de travail de développement existants.

Partage d'URL sans backend

Compression dans le hash d'URL, partagez le contenu de revue avec les membres de l'équipe sans serveur.

Outils d'annotation d'images

Joignez des images et utilisez des outils de pinceau, flèche et cercle pour annoter, améliorant l'expression des retours.

Intégration avec notes tierces

Les plans approuvés sont automatiquement enregistrés dans Obsidian ou Bear, avec frontmatter et étiquettes.
