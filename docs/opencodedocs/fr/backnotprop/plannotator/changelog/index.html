Notes de Version ​

Ce chapitre documente l'historique des versions de Plannotator, les publications de nouvelles fonctionnalités et les informations sur les corrections de bugs, vous aidant à comprendre l'évolution et les dernières avancées du projet.

Contenu de ce Chapitre ​

📋 Historique des Versions

Consultez l'historique complet des versions de Plannotator, incluant les nouvelles fonctionnalités, améliorations et corrections de bugs pour chaque version. Découvrez la chronologie des fonctionnalités majeures comme la revue de code, l'annotation d'images et l'intégration Obsidian/Bear.

Recommandations de Lecture ​

Les notes de version sont particulièrement utiles dans les scénarios suivants :

ScénarioRecommandation
Vérification avant Mise à JourConsultez les nouvelles fonctionnalités et changements de rupture de la version cible
Exploration de FonctionnalitésDécouvrez quelles fonctionnalités ont été ajoutées dans les dernières versions
Résolution de ProblèmesConfirmez si un bug spécifique a été corrigé dans une nouvelle version
Compatibilité de VersionsVérifiez la correspondance des versions entre Plannotator et Claude Code/OpenCode

Restez à Jour

Nous vous recommandons de consulter régulièrement les notes de version et de mettre à niveau vers la dernière version dès que possible pour bénéficier des nouvelles fonctionnalités et des corrections de sécurité.

Prochaines Étapes ​

Après avoir lu les notes de version, vous pouvez :

  • Retourner au Guide de Démarrage Rapide pour apprendre les bases d'utilisation
  • Consulter la FAQ pour résoudre les problèmes d'utilisation
  • Lire la Référence API pour découvrir tous les points de terminaison API
