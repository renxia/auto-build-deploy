openskills
Universal Skill Loader for AI Coding Agents

Enable any AI coding agent (Claude Code, Cursor, Windsurf, Aider, etc.) to use standard SKILL.md format skills. Unified standard, progressive loading, multi-agent support.

Get Started GitHub
Why OpenSkills?

Unified skill format that works across any agent. Say goodbye to platform lock-in.

Fully Compatible with Claude Code

Same prompt format, same marketplace, same folder structure. Seamless migration.

Multi-Agent Support

Supports Claude Code, Cursor, Windsurf, Aider, Codex, and all agents that can read AGENTS.md.

Progressive Loading

Load skills on demand, keeping agent context concise and focused. Avoid context pollution.

Flexible Installation Sources

Install skills from GitHub, local paths, or private git repositories. Adapts to various scenarios.

Universal Mode

One skill set serves multiple agents. Avoid conflicts with Claude Code. Unified skill management.

Pure Local Execution

CLI tool requires no server. All operations completed on local file system. Privacy guaranteed.
