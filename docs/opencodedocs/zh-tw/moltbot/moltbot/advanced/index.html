進階功能 ​

本章概述 ​

本章深入介紹 Clawdbot 的高級功能，幫助你充分利用 AI 助手的強大能力。從 AI 模型設定和多 Agent 協作，到瀏覽器自動化、記憶系統和語音功能，你可以根據需求選擇學習。

前置條件

學習本章前，請先完成以下內容：

  • 快速開始
  • 啟動 Gateway

學習路徑 ​

根據你的需求，可以選擇不同的學習路徑：

🚀 快速上手路徑（推薦新手） ​

  1. AI 模型與認證設定 - 設定你喜歡的 AI 模型
  2. 指令執行工具與審批 - 讓 AI 安全地執行指令
  3. Web 搜尋與抓取工具 - 擴展 AI 的知識獲取能力

🤖 AI 能力擴展路徑 ​

  1. 會話管理與多 Agent - 理解 AI 協作機制
  2. 記憶系統與向量搜尋 - 讓 AI 記住重要資訊
  3. 技能平台與 ClawdHub - 使用和分享技能包

🔧 自動化工具路徑 ​

  1. 瀏覽器自動化工具 - 網頁操作自動化
  2. Cron 定時任務與 Webhook - 定時任務與事件觸發
  3. 遠端 Gateway 與 Tailscale - 遠端存取你的 AI 助手

🎨 互動體驗路徑 ​

  1. Canvas 可視化介面與 A2UI - 可視化互動介面
  2. 語音喚醒與文字轉語音 - 語音互動功能

🔒 安全與部署路徑 ​

  1. 安全與沙箱隔離 - 深入了解安全機制
  2. 遠端 Gateway 與 Tailscale - 安全的遠端存取

主題描述預計時間

工具系統 ​

主題描述預計時間
