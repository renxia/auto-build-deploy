頁面導航

常見問題與優化 ​

本章節幫助你解決使用 Clawdbot 時遇到的各种問題，掌握完整的命令行操作方法，並學習如何優化系統性能。

學習順序建議

建議按以下順序閱讀本章節內容：

  1. 先學習 CLI 命令參考 —— 掌握基礎操作和調試工具
  2. 再學習 故障排除 —— 遇到問題時能快速定位和解決
  3. 最後學習 性能優化 —— 提升系統運行效率

子頁面導航 ​

故障排除 ​

解決常見問題的全面排查指南，涵蓋：

  • Gateway 啟動失敗
  • 渠道連接問題（WhatsApp、Telegram、Slack 等）
  • 認證錯誤和權限問題
  • 工具調用失敗
  • 會話管理異常

適用場景：當你遇到具體問題，需要快速排查和解決時。

CLI 命令參考 ​

完整的命令列表和用法說明，包括：

  • Gateway 控制命令（啟動、重啟、狀態）
  • Agent 交互命令
  • 模型管理和配置
  • 渠道管理（登錄、狀態、登出）
  • 會話管理（列表、重置、刪除、壓縮）
  • 節點管理（列表、調用、配對）
  • 技能管理和配置精靈

適用場景：需要快速查找某個命令的用法和參數，或學習 CLI 的完整功能。

性能優化 ​

提升 Gateway 性能和響應速度的優化技巧，涵蓋：

  • 提示詞緩存（Prompt Caching）降低成本
  • 會話修剪（Session Pruning）減少上下文
  • 並發控制和資源管理
  • 記憶體優化和垃圾回收
  • 日誌級別調優

適用場景：系統運行緩慢、資源佔用過高，或希望提升用戶體驗時。

前置條件 ​

在學習本章節之前，建議你已經完成：

  • ✅ 快速開始 —— 了解基本的安裝和配置流程
  • ✅ 啟動 Gateway —— 知道如何啟動和管理 Gateway

下一步指引 ​

完成本章節後，你可以繼續學習：

  • 完整配置參考 —— 深入了解所有配置項的詳細說明
  • 部署選項 —— 學習如何在不同平台部署 Clawdbot
  • 開發指南 —— 了解插件開發和貢獻流程
