主題

plannotator
AI 計畫與程式碼審查工具

為 Claude Code 和 OpenCode 提供互動式計畫審查和程式碼審查工具，支援視覺化註解、團隊協作、URL 分享和第三方筆記應用整合。本機執行，資料安全，無需伺服器即可分享審查內容。

快速開始 GitHub
為什麼選擇 Plannotator？

視覺化審查 AI 產生的計畫和程式碼，精確回饋，團隊協作。

視覺化計畫審查

在瀏覽器中檢視 AI 產生的計畫，用刪除、替換、插入、評論註解精確回饋。

程式碼審查視覺化

Git diff side-by-side 或 unified 檢視，支援行級註解和程式碼建議。

多平台支援

支援 Claude Code 和 OpenCode，無縫融入現有開發工作流程。

URL 無後端分享

壓縮到 URL hash 中，無需伺服器即可分享審查內容給團隊成員。

圖像標註工具

附加圖像並使用畫筆、箭頭、圓形工具進行標註，增強回饋表達。

第三方筆記整合

核准的計畫自動儲存到 Obsidian 或 Bear，包含 frontmatter 和標籤。
