agent-skills
Vercel AI 編碼代理技能包

為 Claude 等 AI 代理提供可擴展的技能集合。內置 57 條 React 性能優化規則、100+ 條 Web 設計指南，支援一鍵部署至 Vercel。

快速開始 GitHub
為什麼選擇 agent-skills？

Vercel 官方出品，將工程團隊的最佳實踐打包成可復用的技能，讓 AI 代理自動應用。

React 性能優化

57 條 Vercel 工程團隊驗證的規則，按 CRITICAL/HIGH/MEDIUM 分級，覆蓋瀑布流消除、打包優化等 8 個類別。

Web 設計指南

100+ 條 UI 審計規則，涵蓋可訪問性、動畫性能、深色模式、表單驗證等多個維度。

Vercel 一鍵部署

自動檢測 40+ 種框架，零配置部署並返回預覽連結和所有權轉移連結。

自動觸發機制

安裝後 Agent 自動識別任務並激活對應技能，無需手動選擇或記憶指令。

標準化格式

遵循 Agent Skills 規範，每個技能包含 SKILL.md、scripts/ 和 references/，便於擴展。

開源自由擴展

MIT 許可證，可自由擴展定製技能或基於現有規則構建自己的技能包。
