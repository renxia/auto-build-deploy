頁面導航

附錄 ​

本章節提供 OpenCode Agent Skills 的參考資料和開發規範，幫助你編寫高品質的技能並快速查閱 API 介面。

本章內容 ​

頁面說明
技能開發最佳實踐掌握編寫高品質技能的技巧和規範，包括命名規範、描述編寫、目錄組織、腳本使用等
API 工具參考查看所有可用工具的詳細參數和返回值，包括 get_available_skills、read_skill_file、run_skill_script、use_skill

學習路徑建議 ​

推薦順序

  1. 先學最佳實踐 → 了解技能開發的規範和技巧，避免常見錯誤
  2. 再查 API 參考 → 在實際開發中遇到問題時，查閱具體的參數和返回值

這兩篇內容的定位不同：

  • 最佳實踐：適合在開始編寫技能前通讀一遍，建立正確的開發習慣
  • API 參考：適合作為工具書，在需要時查閱具體細節

前置條件 ​

建議先完成

在閱讀本章節前，建議你已經：

  • 完成 建立你的第一個技能，了解基本的技能結構
  • 了解 技能發現機制，理解插件如何找到技能

下一步 ​

完成本章節後，你可以：

  • 查看 常見問題排查，解決使用中遇到的問題
  • 了解 安全性說明，確保技能的安全性
  • 回顧 版本歷史，了解插件的演進和新功能
