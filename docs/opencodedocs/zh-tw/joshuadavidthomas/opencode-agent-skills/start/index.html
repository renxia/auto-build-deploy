頁面導航

快速開始 ​

本章節幫助你從零開始使用 OpenCode Agent Skills 外掛。你將了解外掛的核心價值，完成安裝設定，並建立你的第一個技能。

本章內容 ​

什麼是 OpenCode Agent Skills？

了解外掛的核心價值和功能特性，包括動態技能發現、上下文注入、壓縮恢復等機制。

安裝指南

完成外掛安裝並驗證運行，支援基本安裝、固定版本安裝和本地開發安裝三種方式。

建立你的第一個技能

掌握技能目錄結構和 SKILL.md 格式規範，動手建立並測試一個簡單技能。

學習路徑 ​

建議按以下順序學習：

  1. 什麼是 OpenCode Agent Skills？ — 先了解外掛能做什麼，建立整體認知
  2. 安裝指南 — 安裝外掛，讓 OpenCode 具備技能管理能力
  3. 建立你的第一個技能 — 動手實作，建立屬於你的第一個技能

預計時間

完成本章節大約需要 30-45 分鐘。

下一步 ​

完成本章節後，你可以繼續學習 平台功能，深入了解技能發現機制、技能載入、自動匹配等進階功能。
