附錄 ​

本章節匯集了 Antigravity Tools 的技術參考資料，包括 API 端點速查、資料儲存結構和實驗性功能邊界說明。當你需要快速查閱某個技術細節時，這裡是你的"字典"。

本章內容 ​

文件說明適用場景
端點速查表對外 HTTP 路由一覽：OpenAI/Anthropic/Gemini/MCP 端點、鑑權模式與 Header 格式接入新客戶端、排查 404/401
資料與模型帳號檔案結構、SQLite 統計庫表結構、關鍵欄位口径備份遷移、直接查庫、排障
z.ai 整合邊界z.ai 已實作 vs 明確未實作的功能清單評估 z.ai 能力、避免誤用

學習路徑建議 ​

端點速查表 → 資料與模型 → z.ai 整合邊界
    ↓              ↓              ↓
  知道調哪個路徑   知道資料在哪    知道邊界在哪
  1. 先看端點速查表：了解有哪些 API 可以調用，以及鑑權怎麼配
  2. 再看資料與模型：了解資料儲存結構，方便備份、遷移和直接查庫排障
  3. 最後看 z.ai 整合邊界：如果你用到了 z.ai，這篇幫你避免把"未實作的"當成"可用的"

這些文件不是"必讀"

附錄是參考資料，不是教學。你不需要從頭讀到尾，遇到具體問題時再來查閱即可。
