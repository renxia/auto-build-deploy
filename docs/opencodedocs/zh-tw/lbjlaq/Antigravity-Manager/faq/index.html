常見問題 ​

本章節收錄了使用 Antigravity Tools 過程中最常遇到的錯誤碼和異常情境，幫助你快速定位問題根因並找到解決方案。

本章包含 ​

問題類型頁面說明
帳號失效invalid_grant 與帳號自動禁用帳號突然不可用？了解 OAuth Token 失效的原因和恢復流程
鑑權失敗401/鑑權失敗請求被拒絕？檢查 auth_mode 設定和 Header 格式
限流錯誤429/容量錯誤頻繁 429？區分配額不足和上游限流，用輪換策略降低影響
路徑錯誤404/路徑不相容介面 404？解決 Base URL 和 /v1 前綴的拼接問題
串流異常串流中斷/0 Token/簽名失效響應中斷或空？了解代理的自愈機制和排查路徑

學習路徑建議 ​

按錯誤碼排查：遇到具體錯誤時，直接跳轉到對應頁面。

系統性學習：如果你想全面了解可能遇到的問題，建議按以下順序閱讀：

  1. 404/路徑不相容 — 最常見的接入問題，通常是第一個坑
  2. 401/鑑權失敗 — 路徑對了但請求被拒，檢查鑑權設定
  3. invalid_grant 與帳號自動禁用 — 帳號層面的問題
  4. 429/容量錯誤 — 請求成功但被限流
  5. 串流中斷/0 Token/簽名失效 — 進階問題，涉及串流響應和簽名機制

前置條件 ​

建議先完成

  • 啟動本地反代並接入第一個用戶端 — 確保基本環境已跑通
  • 添加帳號 — 了解帳號添加的正確方式

下一步 ​

排查完問題後，可以繼續深入學習：

  • 高可用排程 — 用輪換和重試策略減少錯誤發生
  • Proxy Monitor — 用日誌系統追蹤請求細節
  • 設定全解 — 了解所有設定項的作用
