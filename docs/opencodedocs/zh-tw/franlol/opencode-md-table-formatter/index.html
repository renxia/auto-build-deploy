opencode-md-table-formatter
AI 表格自動對齊

OpenCode 外掛，自動格式化 AI 生成的 Markdown 表格。智慧識別內容寬度，支援粗體、斜體、程式碼區塊等格式，讓表格始終整齊美觀。

查看教學 GitHub
核心功能

自動識別 Markdown 語法，智慧計算顯示寬度

智慧識別 Markdown 格式

正確計算粗體、斜體、程式碼區塊等格式的顯示寬度，確保表格始終完美對齊

AI 生成後自動格式化

透過 experimental.text.complete 鉤子自動觸發，AI 生成文字後無需手動操作即可格式化

程式碼區塊智慧保護

保護行內程式碼區塊內的 Markdown 符號不被錯誤剝離，確保程式碼內語法保持原樣

三種對齊方式支援

支援左對齊（:---）、置中對齊（:---:）、右對齊（---:）三種格式

巢狀 Markdown 處理

使用多遍正則表達式演算法處理粗體、斜體、刪除線等巢狀 Markdown 語法

邊界情況相容

正確處理 Emoji、Unicode 字元、空儲存格、長內容等邊界情況

靜默執行

無控制台日誌日誌輸出，錯誤不中斷工作流程，靜默完成格式化

驗證回饋

無效表格新增友善的錯誤註解提示，幫助使用者快速定位問題
