主題

openskills
AI 編碼代理的通用技能載入器

讓任何 AI 編碼代理（Claude Code、Cursor、Windsurf、Aider 等）都能使用標準的 SKILL.md 格式技能。統一標準、漸進式載入、多代理支援。

快速開始 GitHub
為什麼選擇 OpenSkills？

統一的技能格式，讓技能在任何代理中可用，告別平台鎖定。

Claude Code 完全相容

相同的提示格式、相同的 marketplace、相同的資料夾結構，無縫遷移。

多代理支援

支援 Claude Code、Cursor、Windsurf、Aider、Codex 等所有可讀取 AGENTS.md 的代理。

漸進式載入

按需載入技能，保持代理上下文精簡和聚焦，避免上下文污染。

靈活的安裝來源

支援從 GitHub、本地路徑、私有 git 倉庫安裝技能，適應各種使用場景。

Universal 模式

一套技能服務多個代理，避免與 Claude Code 衝突，統一技能管理。

純本地運行

CLI 工具無需伺服器，所有操作在本地檔案系統完成，確保隱私安全。
