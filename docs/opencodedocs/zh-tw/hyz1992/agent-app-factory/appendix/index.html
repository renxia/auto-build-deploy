附錄 ​

本章節包含參考文件和補充資料，幫助你深入理解 AI App Factory 的技術細節。

子頁面導覽 ​

CLI 命令參考 ​

完整的 CLI 命令列表、參數說明和使用範例，包括 init、run、continue、status、list 和 reset 等命令。

目錄結構詳解 ​

了解 Factory 專案的完整目錄結構和各檔案的用途，包括 agents/、skills/、cli/ 等核心目錄的作用。

程式碼規範 ​

了解生成的程式碼應遵循的 TypeScript 編碼規範、檔案結構、註解要求和提交訊息規範。

技術棧詳解 ​

了解生成的應用使用的技術棧：Node.js + Express + Prisma + React Native，以及各部分的技術選型理由。

學習路徑建議 ​

本章節的文件是參考資料，不需要按順序閱讀。建議在學習主教學時，根據需要查閱對應章節：

  • 學習 安裝與配置 時，可以查閱 CLI 命令參考
  • 學習 初始化 Factory 專案 時，可以查閱 目錄結構詳解
  • 學習流水線各階段時，可以查閱 技術棧詳解
  • 需要理解生成程式碼的品質標準時，可以查閱 程式碼規範

前置條件 ​

無前置要求。但為了更好地理解本章節內容，建議先完成以下教學：
