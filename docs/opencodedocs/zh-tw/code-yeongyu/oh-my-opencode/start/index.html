快速入門 ​

本章節幫助你快速安裝和開始使用 oh-my-opencode，從基礎設定到主編排器，再到一鍵啟動全部功能的 Ultrawork 模式。

學習路徑 ​

按照以下順序學習，循序漸進掌握 oh-my-opencode 的基礎功能：

1. 快速安裝與設定 ​

完成 oh-my-opencode 的安裝、Provider 設定和基礎驗證。

  • 安裝 oh-my-opencode
  • 設定 AI Provider（Claude、OpenAI、Gemini 等）
  • 驗證設定是否正確
  • 理解設定檔結構

2. 初識 Sisyphus：主編排器 ​

理解代理編排的核心概念，學習 Sisyphus 代理的基本使用方法。

  • 什麼是代理編排系統
  • Sisyphus 代理的核心功能
  • 基本使用方法和最佳實踐
  • 理解多模型協作的工作流程

3. Ultrawork 模式：一鍵啟動全部功能 ​

學習如何使用 ultrawork 關鍵字快速啟動所有專業代理和並行任務功能。

  • 什麼是 Ultrawork 模式
  • 如何觸發 ultrawork 關鍵字
  • 10 位專業代理的協同工作
  • 並行背景任務的管理

前置條件 ​

開始學習本章節前，請確保：

  • 已安裝 Node.js 18+（推薦 20 或更高版本）
  • 熟悉基本的命令列操作
  • 擁有至少一個 AI 服務的 API 金鑰（Claude、OpenAI 或 Gemini）

還沒有金鑰？

如果你還沒有 AI 服務的 API 金鑰，可以先從 Claude 開始申請，它是目前最推薦的選擇。

下一步 ​

完成本章節後，你可以繼續學習：

  • 平台與整合 - 深入了解多 Provider 設定和模型策略
  • 進階功能 - 探索完整的 10 位代理團隊和進階特性
