入門 ​

本章節幫助你快速安裝 opencode-notify 外掛，體驗桌面通知功能，並理解其核心工作機制。

學習路徑 ​

1. 快速開始 ​

在 5 分鐘內完成安裝並體驗第一個通知。

  • 透過 OCX 一鍵安裝
  • 驗證通知功能是否正常運作

2. 工作原理 ​

了解外掛的核心功能、智慧過濾機制和通知觸發條件。

  • 掌握四種通知類型及其觸發時機
  • 理解智慧過濾如何避免通知轟炸

前置條件 ​

  • 已安裝 OpenCode Agent（官網下載）
  • 基本的終端機操作知識

下一步 ​

完成本章節後，你可以繼續學習：

  • macOS 平台特性 - 了解 macOS 專用的焦點偵測和音效自訂功能
  • 設定參考 - 深入了解所有設定選項
  • 疑難排解 - 遇到問題時的解決方案
