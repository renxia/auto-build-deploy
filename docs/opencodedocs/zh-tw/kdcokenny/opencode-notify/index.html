opencode-notify
讓 OpenCode AI 在你專注時溫柔提醒你

告別頻繁切換視窗的焦慮。opencode-notify 為你的 OpenCode AI 提供原生桌面通知，支援智慧焦點檢測、靜音時段和點擊聚焦（macOS），讓你專注工作，只在 AI 需要你時被精準喚醒。

快速開始 GitHub
為何選擇 opencode-notify？

專注工作，智慧提醒

🔔 原生桌面通知

使用作業系統原生通知系統（macOS Notification Center、Windows Toast、Linux notify-send），無縫融入你的工作流。

🧠 智慧焦點檢測

自動檢測終端機視窗焦點，避免重複通知。預設只通知父會話，過濾掉子任務的雜訊，讓你專注於真正重要的資訊。

⏰ 靜音時段設定

設定靜音時段（如 22:00-08:00），夜間自動抑制通知，支援跨午夜時段，保障你的休息時間。

🖱️ 點擊聚焦終端機（macOS）

macOS 使用者點擊通知即可自動聚焦終端機視窗，節省切換應用程式的時間，工作效率倍增。

🎨 自訂音效（macOS）

為不同類型事件設定專屬音效（Glass、Basso、Submarine 等），透過聲音快速識別通知類型。

💻 37+ 終端機支援

支援 Ghostty、iTerm2、WezTerm、VS Code 整合終端機等 37+ 終端機模擬器，自動識別你的終端機類型。

🔒 本地執行，零隱私風險

所有通知在本地生成，無資料上傳雲端，你的工作內容完全私密安全。

⚡ 輕量級外掛

基於 TypeScript + Bun 建構，安裝即用，設定檔簡潔，無需學習複雜設定。
