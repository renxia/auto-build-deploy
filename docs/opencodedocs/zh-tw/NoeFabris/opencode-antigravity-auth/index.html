主題

Antigravity Auth
Claude & Gemini 認證神器

無需 API Key，透過 Google OAuth 憑證直接存取 Claude Opus 4.5、Sonnet 4.5 和 Gemini 3 Pro/Flash。支援多帳戶負載平衡與會話自動恢復。

快速開始 GitHub
為什麼選擇 Antigravity Auth？

解決手動設定 API Key 的煩惱，自動化管理配額，專為高並行開發設計。

免 API Key

無需申請昂貴的 API Key，直接使用 Google 帳戶憑證，一鍵授權登入。

多帳戶負載平衡

支援多達 10 個 Google 帳戶自動輪換，智慧分配請求，徹底告別速率限制。

Thinking 模型支援

完美支援 Claude 和 Gemini 3 的思考模式，可設定 Budget，解鎖深度推理能力。

會話自動恢復

工具呼叫失敗？會話中斷？外掛會自動重試並恢復上下文，無需人工干預。

外掛相容性

與 oh-my-opencode、DCP 等主流外掛無縫整合。

雙配額系統

最大限度利用 Google 提供的免費與付費配額池。
