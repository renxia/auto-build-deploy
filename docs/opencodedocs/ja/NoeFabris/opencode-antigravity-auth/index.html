Antigravity Auth
Claude & Gemini 認証の決定版

APIキーは不要。Google OAuth認証情報を使用して、Claude Opus 4.5、Sonnet 4.5、Gemini 3 Pro/Flashに直接アクセスできます。マルチアカウントの負荷分散とセッションの自動復元をサポート。

クイックスタート GitHub
Antigravity Authを選ぶ理由

APIキーの手動設定の悩みを解決し、クォータを自動管理。高並列処理向けに設計されています。

APIキー不要

高価なAPIキーを申請する必要がなく、Googleアカウントの認証情報を直接使用して、ワンクリックで認証ログインできます。

マルチアカウントの負荷分散

最大10個のGoogleアカウントの自動ローテーションをサポートし、リクエストをインテリジェントに分散して、レート制限を完全に解消します。

Thinkingモデルのサポート

ClaudeとGemini 3の思考モードを完璧にサポートし、Budgetを設定して深い推論能力をアンロックできます。

セッションの自動復元

ツール呼び出しの失敗？セッションの中断？プラグインが自動的に再試行してコンテキストを復元するため、手動での介入は不要です。

プラグインの互換性

oh-my-opencode、DCPなどの主流プラグインとシームレスに統合できます。

デュアルクォータシステム

Googleが提供する無料と有料のクォータプールを最大限に活用します。
