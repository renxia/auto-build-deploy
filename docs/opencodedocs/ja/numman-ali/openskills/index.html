openskills
AIコーディングエージェントの汎用スキルローダー

Claude Code、Cursor、Windsurf、Aider など、あらゆる AI コーディングエージェントで標準の SKILL.md 形式スキルを使用可能にします。統一された標準、漸進的ロード、マルチエージェント対応。

クイックスタート GitHub
なぜ OpenSkills を選ぶのか？

統一されたスキルフォーマットにより、あらゆるエージェントでスキルが利用可能になり、プラットフォームロックインから解放されます。

Claude Code 完全互換

同じプロンプト形式、同じ marketplace、同じフォルダ構造でシームレスに移行できます。

マルチエージェント対応

AGENTS.md を読み取れるすべてのエージェント（Claude Code、Cursor、Windsurf、Aider、Codex など）をサポートします。

漸進的ロード

必要なスキルのみをロードし、エージェントのコンテキストをシンプルに保ち、コンテキスト汚染を防ぎます。

柔軟なインストール元

GitHub、ローカルパス、プライベート Git リポジトリからスキルをインストールでき、あらゆる使用シーンに対応します。

Universal モード

1セットのスキルで複数のエージェントに対応し、Claude Code との競合を回避し、スキル管理を統一します。

完全ローカル実行

CLI ツールはサーバー不要ですべての操作がローカルファイルシステムで完結し、プライバシーと安全性を確保します。
