付録 ​

本セクションでは、OpenSkills の技術参考資料を提供します。これには、コマンドライン API、ファイル形式仕様、用語定義が含まれます。具体的なコマンドパラメータを参照したり、生成ファイルの構造を理解したり、特定の用語の意味を確認したりする必要がある場合は、こちらのクイックリファレンスをご利用ください。

本章の内容 ​

ドキュメント説明
CLI API リファレンスすべてのコマンド、オプション、パラメータの完全なドキュメントと使用例
AGENTS.md 形式仕様sync コマンドが生成する AGENTS.md ファイルの構造、XML タグとスキルリスト形式を含む
ファイル構造OpenSkills が生成するディレクトリとファイルの説明、.openskills/ の内容を理解するのに役立ちます
用語集OpenSkills と AI スキルシステムの主要用語の解説

学習パスの推奨 ​

付録は参考資料であり、最初から最後まで読む必要はありません。推奨される使用方法は以下の通りです：

  1. コマンドの問題が発生した → CLI API リファレンス を確認
  2. AGENTS.md の生成方法を知りたい → AGENTS.md 形式仕様 を確認
  3. ファイルの場所が分からない → ファイル構造 を確認
  4. 不明な用語を見つけた → 用語集 を確認

初心者向けの推奨事項

OpenSkills を初めてご利用の場合は、まず クイックスタート チュートリアルを完了することをお勧めします。問題が発生した場合は、こちらの付録に戻ってご確認ください。

次のステップ ​

  • 問題が発生しましたか？ よくある質問 と トラブルシューティング をご確認ください
  • バージョンアップデートを知りたいですか？ 更新ログ をご確認ください
