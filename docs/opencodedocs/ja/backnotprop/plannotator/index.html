plannotator
AI プランレビュー・コードレビューツール

Claude Code と OpenCode 向けのインタラクティブなプランレビュー・コードレビューツールを提供します。ビジュアルアノテーション、チームコラボレーション、URL 共有、サードパーティノートアプリ連携をサポート。ローカル実行でデータ安全、サーバー不要でレビュー内容を共有できます。

クイックスタート GitHub
なぜ Plannotator を選ぶのか？

AI 生成プランとコードをビジュアルレビュー、精確なフィードバック、チームコラボレーション。

ビジュアルプランレビュー

ブラウザで AI 生成プランを確認し、削除・置換・挿入・コメントアノテーションで精確にフィードバック。

コードレビュービジュアライゼーション

Git diff の side-by-side または unified ビュー、行レベルのコメントとコード提案をサポート。

マルチプラットフォーム対応

Claude Code と OpenCode をサポート、既存の開発ワークフローにシームレスに統合。

バックエンド不要の URL 共有

URL ハッシュに圧縮、サーバー不要でチームメンバーとレビュー内容を共有。

画像アノテーションツール

画像を添付し、ブラシ・矢印・円形ツールでアノテーション、フィードバック表現を強化。

サードパーティノート連携

承認されたプランは Obsidian または Bear に自動保存、frontmatter とタグ付き。
