agent-skills
Vercel AI コーディングエージェントスキルセット

Claude などの AI エージェント向けの拡張可能なスキルセットを提供します。React パフォーマンス最適化の 57 ルール、Web デザインガイドの 100+ ルールが内蔵されており、Vercel へのワンクリックデプロイをサポートします。

クイックスタート GitHub
なぜ agent-skills を選ぶのか？

Vercel 公式提供。エンジニアリングチームのベストプラクティスを再利用可能なスキルとしてパッケージ化し、AI エージェントに自動適用させます。

React パフォーマンス最適化

Vercel エンジニアリングチームが検証した 57 ルールが含まれ、CRITICAL/HIGH/MEDIUM に分類され、ウォーターフォール解消、バンドル最適化などの 8 カテゴリをカバーします。

Web デザインガイド

100+ の UI 監査ルールが含まれ、アクセシビリティ、アニメーションパフォーマンス、ダークモード、フォーム検証などの複数の次元をカバーします。

Vercel ワンクリックデプロイ

40 以上のフレームワークを自動検出し、設定なしでデプロイし、プレビューリンクと所有権転送リンクを返します。

自動トリガー

インストール後、Agent がタスクを自動的に認識し、対応するスキルをアクティブにします。手動で選択したり、コマンドを覚えたりする必要はありません。

標準化された形式

Agent Skills 規格に従い、各スキルには SKILL.md、scripts/、references/ が含まれ、拡張が容易です。

オープンソースで自由に拡張

MIT ライセンス。スキルを自由に拡張・カスタマイズしたり、既存のルールに基づいて独自のスキルセットを構築したりできます。
