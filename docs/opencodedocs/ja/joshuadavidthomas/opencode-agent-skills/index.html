opencode-agent-skills
OpenCode AI Skills Dynamic Management Plugin

OpenCode の AI Agent が再利用可能なスキルライブラリを動的に検出・管理するためのプラグインシステム。プロジェクト間・セッション間でのスキル再利用を実現し、意味的類似度に基づいて関連スキルを自動推薦します。

クイックスタート GitHub
なぜ OpenCode Agent Skills を選ぶのか？

OpenCode に動的なスキル検出・管理能力を提供し、プロジェクト間・セッション間でのスキル再利用を実現します。

動的スキル検出

プロジェクト、ユーザー、プラグインディレクトリからスキルを自動検出。6つの位置をスキャンし、優先度に基づいて最初に一致するスキルを返します。

Claude Code 互換性

既存の Claude Code スキル・プラグイン形式をサポートし、移行コストを削減、既存リソースを再利用可能にします。

意味マッチ推薦

HuggingFace embedding モデルを使用して類似度を計算し、ユーザーの入力に基づいて関連スキルを自動推薦します。

コンテキスト注入

スキル内容を XML 形式でセッションコンテキストに注入し、AI がスキルガイダンスに従って高品質な結果を出力します。

圧縮後復元

セッション圧縮後、スキルリストを自動的に再注入し、長時間セッション中もスキルを持続的に利用可能にします。

スクリプト実行サポート

スキルディレクトリ内に実行可能スクリプトを含めることができ、スキルディレクトリのコンテキストで自動化タスクを実行します。
