ページナビゲーション

よくある質問 ​

この章では、OpenCode Agent Skills プラグインの使用中に発生する問題の解決方法と、プラグインのセキュリティ機構について説明します。スキルのロード失敗、スクリプト実行エラー、プラグインがシステムをどのように保護しているかなど、あらゆる疑問にお答えします。

本章の内容 ​

🔧 よくある問題のトラブルシューティング

スキルのロード失敗、スクリプト実行エラー、パス安全性の問題、モデルロード失敗など、9種類のよくある問題のトラブルシューティング方法と解決策を解説します。

🔒 セキュリティについて

プラグインのパス安全性チェック、YAML パース安全性、入力検証、スクリプト実行保護などのコアセキュリティ機構とベストプラクティスを学びます。

学習パス ​

以下の順序で学習することをお勧めします：

よくある問題のトラブルシューティング → セキュリティについて
  1. まずトラブルシューティングを学ぶ：問題が発生したら、まず「よくある問題のトラブルシューティング」で素早く原因を特定し解決する
  2. 次にセキュリティを学ぶ：プラグインのセキュリティ機構を理解し、安全な使用習慣を身につける

問題が発生したら

プラグイン使用中に具体的な問題が発生した場合は、「よくある問題のトラブルシューティング」で該当する症状を直接検索してください。プラグインのセキュリティ設計を体系的に理解したい場合は、「セキュリティについて」をお読みください。

前提条件 ​

以下の章を先に完了することをお勧めします

  • クイックスタート：プラグインの基本概念とインストール方法を理解する
  • プラットフォーム機能：スキルの発見、ロード、実行などのコア機能に慣れる

この章の内容は、プラグインの基本的な理解があることを前提としています。初めて使用する場合は、上記の章を先に完了することをお勧めします。

次のステップ ​

この章を完了した後、さらに以下を探索できます：

  • 付録：スキル開発のベストプラクティスと API ツールリファレンスを参照
  • 更新履歴：プラグインのバージョン変遷と新機能を確認
