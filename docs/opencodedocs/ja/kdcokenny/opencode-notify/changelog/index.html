ページナビゲーション

更新履歴 ​

このセクションでは、opencode-notify プラグインのバージョン履歴と重要な変更を記録しています。機能更新、バグ修正、設定改善の追跡にお役立てください。

サブページ ​

更新履歴 ​

時系列（新しい順）で記録されたプラグインの変更履歴を確認できます。

含まれる内容：

  • 機能更新の記録
  • バグ修正の説明
  • 設定改善の詳細
  • 破壊的変更の注意事項

バージョンについて

このプラグインは OCX 経由で配布されており、バージョン番号なしの方針を採用しています。変更は Git コミット履歴で追跡されます。

前提条件 ​

前提条件はありません。どなたでも更新履歴を閲覧できます。

次のステップ ​

  • クイックスタート - 初めての方はここから始めることをお勧めします
  • 仕組み - プラグインのコア機能と通知メカニズムを理解する
  • プラットフォーム機能 - お使いのプラットフォームの特別な機能を確認する
  • 設定リファレンス - すべての設定オプションを詳しく理解する
  • よくある質問 - よくある質問への回答をすばやく見つける
