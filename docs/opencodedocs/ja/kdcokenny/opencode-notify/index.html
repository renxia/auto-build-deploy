opencode-notify
OpenCode AI が作業中に優しく通知してくれます

ウィンドウの頻繁な切り替えによるストレスを解消しましょう。opencode-notify は、OpenCode AI にネイティブのデスクトップ通知を提供します。インテリジェントなフォーカス検出、サイレント時間帯の設定、クリック時のフォーカス（macOS）をサポートし、作業に集中し、AI が必要なときだけ確実に通知を受け取れます。

クイックスタート GitHub
opencode-notify が選ばれる理由

作業に集中、スマートな通知

🔔 ネイティブデスクトップ通知

OSのネイティブ通知システム（macOS Notification Center、Windows Toast、Linux notify-send）を使用し、ワークフローにシームレスに統合されます。

🧠 インテリジェントなフォーカス検出

ターミナルウィンドウのフォーカスを自動検出し、重複通知を回避します。デフォルトでは親セッションにのみ通知し、サブタスクのノイズをフィルタリングして、本当に重要な情報に集中できます。

⏰ サイレント時間帯の設定

サイレント時間帯（22:00-08:00 など）を設定すると、夜間は通知を自動的に抑制します。日をまたぐ時間帯にも対応し、休息時間を確保します。

🖱️ クリックでターミナルにフォーカス（macOS）

macOS ユーザーは通知をクリックするだけでターミナルウィンドウが自動的にフォーカスされ、アプリの切り替え時間を短縮し、作業効率を倍増させます。

🎨 カスタム通知音（macOS）

異なるタイプのイベントに専用の通知音（Glass、Basso、Submarine など）を設定し、音声で通知タイプを素早く識別できます。

💻 37以上のターミナルをサポート

Ghostty、iTerm2、WezTerm、VS Code 統合ターミナルなど 37以上のターミナルエミュレータをサポートし、ターミナルタイプを自動的に識別します。

🔒 ローカル実行、プライバシーは完全に保護

すべての通知はローカルで生成され、データはクラウドにアップロードされないため、作業内容は完全にプライベートで安全です。

⚡ 軽量プラグイン

TypeScript + Bun で構築され、インストールするだけですぐに使用可能。設定ファイルはシンプルで、複雑な設定を学ぶ必要はありません。
