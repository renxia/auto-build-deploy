ページナビゲーション

付録 ​

この章では参考ドキュメントと補足資料を提供し、AI App Factory の技術詳細を深く理解するのに役立ちます。

サブページナビゲーション ​

CLI コマンドリファレンス ​

完全な CLI コマンドリスト、パラメータ説明と使用例。init、run、continue、status、list、reset などのコマンドが含まれます。

ディレクトリ構造詳細 ​

Factory プロジェクトの完全なディレクトリ構造と各ファイルの用途を理解します。agents/、skills/、cli/ などのコアディレクトリの役割を含みます。

コード規定 ​

生成されたコードが従うべき TypeScript コーディング規定、ファイル構造、コメント要件、コミットメッセージ規定を理解します。

技術スタック詳細 ​

生成されたアプリが使用する技術スタック：Node.js + Express + Prisma + React Native、および各部分の技術選定理由を理解します。

学習パスの提案 ​

この章のドキュメントは参考資料であり、順番通りに読む必要はありません。メインチュートリアルを学習する際、必要に応じて対応する章を参照することをお勧めします：

  • インストールと設定 を学習する際は、CLI コマンドリファレンスを参照
  • Factory プロジェクトの初期化 を学習する際は、ディレクトリ構造詳細を参照
  • パイプラインの各段階を学習する際は、技術スタック詳細を参照
  • 生成されたコードの品質基準を理解する必要がある場合、コード規定を参照

前提条件 ​

特別な前提条件はありません。ただし、この章の内容をより良く理解するために、以下のチュートリアルを完了することをお勧めします：

  • ✅ クイックスタート：アイデアからアプリケーションへ
  • ✅ インストールと設定
  • ✅ Factory プロジェクトの初期化

次のステップ ​

メインチュートリアルを完了したら、さらに深く探求できます：

  • 更新履歴 を確認して最新機能を把握
  • ベストプラクティス を復習して使用効率を向上
  • 上級トピック を深く学習してシステム設計原理を理解
