ページナビゲーション

よくある質問とベストプラクティス ​

FAQ セクションでは、AI App Factory の使用中に遭遇する一般的な問題の解決策とベストプラクティスをまとめています。トラブルシューティングを迅速に行い、開発効率を向上させるのに役立ちます。

サブページナビゲーション ​

コース説明
よくある質問とトラブルシューティング初期化、実行、デプロイ中に遭遇する一般的な問題を解決
ベストプラクティス明確なプロダクト記述の提供、チェックポイントの活用、スコープの制御、段階的なイテレーションなどのテクニックを学習

学習パスの推奨 ​

FAQ セクションの内容は必要に応じて参照でき、厳密な順序で学習する必要はありません：

  1. 問題が発生したら参照：使用中に問題が発生した場合、まず「よくある質問とトラブルシューティング」を確認してください
  2. 効率向上のため必読：「ベストプラクティス」セクションでは経験と教訓をまとめています。最初のプロジェクトを完了した後に読むことをお勧めします。以降のプロジェクトの開発効率が大幅に向上します

前提条件 ​

推奨される事前学習

FAQ セクションを読む前に、以下の内容を完了することをお勧めします：

  • ✅ start セクション：プロジェクトの初期化を完了し、完全なパイプラインを実行済みであること
  • ✅ platforms セクション：少なくとも 1 つの AI アシスタント（Claude Code または OpenCode）を設定済みであること

次のステップ ​

FAQ の学習を完了したら、以下のことができます：

  1. 技術の詳細を深く理解する：

    • CLI コマンドリファレンス - 完全なコマンドリストとパラメータを確認
    • ディレクトリ構造の詳細 - プロジェクト構造とファイルの用途を理解
    • コード規約 - 生成されたコードが従うべき規約を理解

  2. 実践を開始する：

    • ベストプラクティスを使用してプロジェクトの説明を最適化する
    • より複雑なプロダクトのアイデアを試す
    • 7 段階パイプラインの各詳細を探索する
