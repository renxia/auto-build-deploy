変更履歴：バージョン進化と機能追跡 ​

セクション概要 ​

本章では、oh-my-opencodeのバージョン履歴を記録しています。新機能、重要な改善、機能拡張、互換性の変更が含まれます。変更履歴を読むことで、プロジェクトの進化の旅を理解し、最新バージョンの機能拡張を迅速に把握できます。

サブページナビゲーション ​

バージョンタイトル説明リリース状態
v3.0v3.0 機能全新しいカテゴリとスキルシステム、7つの組み込みタスクカテゴリ、3つのプロフェッショナルスキルパッケージ、Sisyphus-Junior実行環境、動的エージェント構成機能を導入✅ 安定版

学習パスの推奨事項 ​

このセクションの変更履歴は、最新バージョンから最古バージョンの順に配置されています。以下の方法で読むことを推奨します：

  • 🎯 最新機能の迅速な理解: 最新バージョン（例：v3.0）の変更履歴を直接読む
  • 📚 バージョンアップグレードガイド: 現在のバージョンから開始し、逆順で読んでアップグレードパスを理解する
  • 🔍 新機能の深い探索: 興味のあるバージョンの詳細なドキュメントと、対応する高度なチュートリアル章を読む

前提条件 ​

本章の変更履歴を読む前に、以下を完了することを推奨します：

  • ✅ クイックインストールと設定 - oh-my-opencodeのインストールと基本設定を理解
  • ✅ Sisyphus入門：メインオーケストレーター - Sisyphusエージェントの基本概念と使用法を理解

次のステップ ​

変更履歴を読んだ後、以下の学習を続けることができます：

  • 新機能の深掘り:

    • カテゴリとスキル：動的エージェント構成 - 新しいv3.0システムの詳細な使用法を学ぶ
    • 組み込みスキル：ブラウザ自動化とGitエキスパート - 3つの組み込みプロフェッショナルスキルパッケージの使用をマスター
    • 深いカスタマイズ：エージェントと権限管理 - カテゴリとスキルをカスタマイズする方法を理解

  • 高度な機能の探索:

    • AIエージェントチーム：10人のエキスパート概要 - 10人の組み込みエージェントの包括的な理解
    • プロメテウス計画：インタビュー形式の要件収集 - プロメテウスエージェントを使用したプロジェクト計画を学ぶ
    • バックグラウンド並列タスク：チームのように働く - バックグラウンドエージェント管理システムをマスター
