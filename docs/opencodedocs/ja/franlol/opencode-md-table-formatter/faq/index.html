よくある質問 ​

章の概要 ​

この章では、opencode-md-table-formatter プラグインのよくある質問とトラブルシューティング方法について説明し、テーブルのフォーマット中に遭遇する様々な問題を素早く特定して解決するのに役立ちます。

サブページナビゲーション ​

よくある質問：テーブルがフォーマットされない場合 ​

プラグイン使用時のよくある質問と解決策を包括的に紹介します：

  • テーブルがフォーマットされない：設定、バージョン要件、OpenCode の再起動を確認
  • 無効なテーブル構造："invalid structure" エラーの意味を理解し、正しいテーブル構文を学習
  • フォーマット失敗："table formatting failed" エラーと例外状況を処理
  • サポートされていないテーブルタイプ：HTML テーブル、複数行セルなどの制限されたシナリオを理解
  • フォーマット後に整列しない：非表示モード、セルの超過長、インラインコードなどの問題を調査

学習パスの推奨 ​

機能概要 を完了した後、実際の問題が発生したときにこの章を参照するか、問題解決のリファレンスマニュアルとして使用することをお勧めします。

以下の順序で学習できます：

  1. 問題が発生したとき：症状に基づいて問題のタイプを特定します（テーブルがフォーマットされない、エラーメッセージなど）
  2. 段階的なトラブルシューティング：トラブルシューティングガイドのチェックリストに従って一つずつ検証
  3. 深い理解：テーブル仕様と非表示モードの原理を読んで技術的な詳細を理解
  4. 既知の制限：既知の制限を確認してプラグインの境界を理解

前提条件 ​

この章を学習する前に、以下を完了することをお勧めします：

  • ✅ 1分で始める：インストールと設定 - プラグインのインストールと設定方法を理解
  • ✅ 機能概要：自動フォーマットの魔法 - プラグインのコア機能と動作原理を理解

また、以下の知識が必要です：

  • Markdown テーブルの基本構文に慣れていること
  • OpenCode のプラグイン設定方法（opencode.jsonc）を理解していること

次のステップ ​

この章を完了した後、以下のトピックを学習できます：

  • 高度な知識：

    • テーブル仕様：どのようなテーブルがフォーマット可能か - 有効なテーブルの構造要件を習得
    • 非表示モードの原理：なぜ幅の計算が重要なのか - OpenCode の非表示モードの動作原理を理解
    • 配置方法の詳細：左揃え、中央揃え、右揃え - 3種類の配置方法の構文と効果を習得

  • 技術的な詳細：

    • 既知の制限：プラグインの境界はどこにあるか - プラグインの既知の制限と適用されないシナリオを理解
    • 技術詳細：キャッシュメカニズムとパフォーマンス最適化 - プラグインの内部キャッシュメカニズムとパフォーマンス最適化戦略を理解
