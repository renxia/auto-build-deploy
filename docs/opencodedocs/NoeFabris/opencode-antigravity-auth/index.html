Antigravity Auth
Authentication Made Easy for Claude & Gemini

Access Claude Opus 4.5, Sonnet 4.5, and Gemini 3 Pro/Flash directly using Google OAuth credentials—no API Key required. Features multi-account load balancing and automatic session recovery.

Get Started GitHub
Why Antigravity Auth?

Eliminate the hassle of manual API Key configuration. Automate quota management. Built for high-concurrency development.

No API Key Required

Skip expensive API Keys. Authenticate directly with your Google account credentials in one click.

Multi-Account Load Balancing

Automatically rotate up to 10 Google accounts with intelligent request distribution. Say goodbye to rate limits.

Thinking Model Support

Full support for Claude and Gemini 3 thinking modes with configurable budgets. Unlock deep reasoning capabilities.

Automatic Session Recovery

Tool call failed? Session interrupted? The plugin automatically retries and restores context—no manual intervention needed.

Plugin Compatibility

Seamlessly integrates with popular plugins like oh-my-opencode and DCP.

Dual Quota System

Maximize both free and paid quota pools provided by Google.
