OpenClaw
Run a personal AI assistant on your device, unifying all communication channels

Stop letting your data flow into cloud black holes. OpenClaw is a local-first multi-channel AI gateway that lets you deploy a private AI assistant on your own device. Supports 12+ platforms including WhatsApp, Telegram, Slack, and Discord, with 60+ built-in skills. Make your AI truly yours.

Quick Start GitHub
Why OpenClaw?

Local-first, privacy-centric, powerful personal AI infrastructure

🏠 AI That Truly Belongs to You

Local-first architecture keeps all data on your device. No more privacy concerns—your conversations belong only to you.

📱 One-Stop Messaging Hub

Unified access to 12+ communication platforms including WhatsApp, Telegram, Slack, Discord, and Signal. One AI assistant manages all your channels.

🤖 Powerful Agent Runtime

Built on Pi Agent's RPC architecture, supporting multi-model configuration, failover, and session isolation—enterprise-grade stability.

🛠️ 60+ Ready-to-Use Skills

GitHub, Spotify, Notion, 1Password, weather, image generation... A rich skill ecosystem to extend AI capabilities with one click.

🌐 Browser Automation

Built-in Chrome/Chromium CDP control, enabling AI to operate browsers, scrape web pages, and execute automation tasks.

🔒 Enterprise-Grade Security

DM pairing mechanism, token/password authentication, Docker sandbox execution, and rate limiting—multi-layer protection ensures system security.
