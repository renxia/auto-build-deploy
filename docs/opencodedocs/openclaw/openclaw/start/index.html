Quick Start ​

Welcome to OpenClaw! This section will guide you from zero to deploying your personal AI assistant.

Section Overview ​

In this section, you will learn:

  • Onboarding - Deploy OpenClaw from scratch
  • Gateway Quick Start - Launch your AI assistant service
  • Send First Message - Have your first conversation with the AI assistant
  • Configuration Basics - Understand the openclaw.json configuration structure

Learning Path ​

Onboarding → Gateway Quick Start → Send First Message → Configuration Basics
    │              │                    │                  │
    ▼              ▼                    ▼                  ▼
 Install & Init  Launch Service    First Conversation   Advanced Config

Prerequisites ​

Before you begin, ensure you meet the following requirements:

  • Node.js >= 22.12.0
  • pnpm 10.23.0 or later
  • Git (for source installation)
  • AI API Key (Anthropic, OpenAI, etc.)
