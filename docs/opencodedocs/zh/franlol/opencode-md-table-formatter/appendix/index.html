本章节包含 ​

📚 已知限制：插件的边界在哪里 ​

了解插件不支持的功能和技术限制，避免在不支持的场景下使用。包括：

  • 不支持 HTML 表格、多行单元格、无分隔行表格
  • 不支持合并单元格和无配置选项
  • 超大表格性能未验证

适合人群：想知道插件能做什么、不能做什么的用户

🔧 技术细节：缓存机制与性能优化 ​

深入理解插件的内部实现，包括缓存机制、性能优化策略和代码结构。包括：

  • widthCache 数据结构与缓存查找流程
  • 自动清理机制与缓存阈值
  • 性能优化效果分析

适合人群：对插件实现原理感兴趣的开发者
