opencode-notify
让 OpenCode AI 在你专注时温柔提醒你

告别频繁切换窗口的焦虑。opencode-notify 为你的 OpenCode AI 提供原生桌面通知，支持智能焦点检测、静音时段和点击聚焦（macOS），让你专注工作，只在 AI 需要你时被精准唤醒。

快速开始 GitHub
为什么选择 opencode-notify？

专注工作，智能提醒

🔔 原生桌面通知

使用操作系统原生通知系统（macOS Notification Center、Windows Toast、Linux notify-send），无缝融入你的工作流。

🧠 智能焦点检测

自动检测终端窗口焦点，避免重复通知。默认只通知父会话，过滤掉子任务的噪声，让你专注于真正重要的信息。

⏰ 静音时段配置

设置静音时间段（如 22:00-08:00），夜间自动抑制通知，支持跨午夜时段，保障你的休息时间。

🖱️ 点击聚焦终端（macOS）

macOS 用户点击通知即可自动聚焦终端窗口，省去切换应用的时间，工作效率翻倍。

🎨 自定义音效（macOS）

为不同类型事件设置专属音效（Glass、Basso、Submarine 等），通过声音快速识别通知类型。

💻 37+ 终端支持

支持 Ghostty、iTerm2、WezTerm、VS Code 集成终端等 37+ 终端模拟器，自动识别你的终端类型。

🔒 本地运行，零隐私风险

所有通知在本地生成，无数据上传云端，你的工作内容完全私密安全。

⚡ 轻量级插件

基于 TypeScript + Bun 构建，安装即用，配置文件简洁，无需学习复杂配置。
