opencode-agent-skills
OpenCode AI 技能动态管理插件

为 OpenCode 提供动态技能发现和管理能力的插件系统，支持 AI Agent 加载和使用可复用的技能库。实现跨项目、跨会话的技能复用，基于语义相似度自动推荐相关技能。

快速开始 GitHub
为什么选择 OpenCode Agent Skills？

为 OpenCode 提供动态技能发现和管理能力，实现跨项目、跨会话的技能复用。

动态技能发现

自动从项目、用户、插件目录发现技能，支持 6 个位置扫描，按优先级返回首个匹配技能。

Claude Code 兼容

支持现有 Claude Code 技能和插件格式，降低迁移成本，复用现有资源。

语义匹配推荐

使用 HuggingFace embedding 模型计算相似度，基于用户输入自动推荐相关技能。

上下文注入

技能内容以 XML 格式注入到会话上下文，AI 遵循技能指导输出高质量结果。

压缩后恢复

会话压缩后自动重新注入技能列表，长时间会话中保持技能持久可用。

脚本执行支持

技能目录下可包含可执行脚本，在技能目录上下文中执行自动化任务。
