agent-skills
Vercel AI 编码代理技能包

为 Claude 等 AI 代理提供可扩展的技能集合。内置 57 条 React 性能优化规则、100+ 条 Web 设计指南，支持一键部署至 Vercel。

快速开始 GitHub
为什么选择 agent-skills？

Vercel 官方出品，将工程团队的最佳实践打包成可复用的技能，让 AI 代理自动应用。

React 性能优化

57 条 Vercel 工程团队验证的规则，按 CRITICAL/HIGH/MEDIUM 分级，覆盖瀑布流消除、打包优化等 8 个类别。

Web 设计指南

100+ 条 UI 审计规则，涵盖可访问性、动画性能、深色模式、表单验证等多个维度。

Vercel 一键部署

自动检测 40+ 种框架，零配置部署并返回预览链接和所有权转移链接。

自动触发机制

安装后 Agent 自动识别任务并激活对应技能，无需手动选择或记忆命令。

标准化格式

遵循 Agent Skills 规范，每个技能包含 SKILL.md、scripts/ 和 references/，便于扩展。

开源自由扩展

MIT 许可证，可自由扩展定制技能或基于现有规则构建自己的技能包。
