主题

Antigravity Auth
Claude & Gemini 认证神器

无需 API Key，通过 Google OAuth 凭证直接访问 Claude Opus 4.5、Sonnet 4.5 和 Gemini 3 Pro/Flash。支持多账号负载均衡与会话自动恢复。

快速开始 GitHub
为什么选择 Antigravity Auth？

解决手动配置 API Key 的烦恼，自动化管理配额，专为高并发开发设计。

免 API Key

无需申请昂贵的 API Key，直接使用 Google 账户凭证，一键授权登录。

多账户负载均衡

支持多达 10 个 Google 账户自动轮换，智能分配请求，彻底告别速率限制。

Thinking 模型支持

完美支持 Claude 和 Gemini 3 的思考模式，可配置 Budget，解锁深度推理能力。

会话自动恢复

工具调用失败？会话中断？插件会自动重试并恢复上下文，无需人工干预。

插件兼容性

与 oh-my-opencode、DCP 等主流插件无缝集成。

双配额系统

最大限度利用 Google 提供的免费与付费配额池。
