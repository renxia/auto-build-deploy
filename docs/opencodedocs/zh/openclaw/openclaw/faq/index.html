遇到问题？ ​

  1. 先运行诊断

    bash
    openclaw doctor

  2. 查看状态

    bash
    openclaw status

  3. 检查日志

    bash
    openclaw logs --follow

命令用途
openclaw doctor运行系统诊断
openclaw status查看系统状态

openclaw --help查看帮助
