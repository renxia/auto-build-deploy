常见问题 ​

本章节收录了使用 Antigravity Tools 过程中最常遇到的错误码和异常场景，帮助你快速定位问题根因并找到解决方案。

本章包含 ​

问题类型页面说明
账号失效invalid_grant 与账号自动禁用账号突然不可用？了解 OAuth Token 失效的原因和恢复流程
鉴权失败401/鉴权失败请求被拒绝？检查 auth_mode 配置和 Header 格式
限流错误429/容量错误频繁 429？区分配额不足和上游限流，用轮换策略降低影响
路径错误404/路径不兼容接口 404？解决 Base URL 和 /v1 前缀的拼接问题
流式异常流式中断/0 Token/签名失效响应中断或空？了解代理的自愈机制和排查路径

学习路径建议 ​

按错误码排查：遇到具体错误时，直接跳转到对应页面。

系统性学习：如果你想全面了解可能遇到的问题，建议按以下顺序阅读：

  1. 404/路径不兼容 — 最常见的接入问题，通常是第一个坑
  2. 401/鉴权失败 — 路径对了但请求被拒，检查鉴权配置
  3. invalid_grant 与账号自动禁用 — 账号层面的问题
  4. 429/容量错误 — 请求成功但被限流
  5. 流式中断/0 Token/签名失效 — 进阶问题，涉及流式响应和签名机制

前置条件 ​

建议先完成

  • 启动本地反代并接入第一个客户端 — 确保基本环境已跑通
  • 添加账号 — 了解账号添加的正确方式

下一步 ​

排查完问题后，可以继续深入学习：

  • 高可用调度 — 用轮换和重试策略减少错误发生
  • Proxy Monitor — 用日志系统追踪请求细节
  • 配置全解 — 了解所有配置项的作用
