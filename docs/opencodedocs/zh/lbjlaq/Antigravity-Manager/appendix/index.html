附录 ​

本章节汇集了 Antigravity Tools 的技术参考资料，包括 API 端点速查、数据存储结构和实验性功能边界说明。当你需要快速查阅某个技术细节时，这里是你的"字典"。

本章内容 ​

文档说明适合场景
端点速查表对外 HTTP 路由一览：OpenAI/Anthropic/Gemini/MCP 端点、鉴权模式与 Header 格式接入新客户端、排查 404/401
数据与模型账号文件结构、SQLite 统计库表结构、关键字段口径备份迁移、直接查库、排障
z.ai 集成边界z.ai 已实现 vs 明确未实现的功能清单评估 z.ai 能力、避免误用

学习路径建议 ​

端点速查表 → 数据与模型 → z.ai 集成边界
    ↓              ↓              ↓
 知道调哪个路径   知道数据在哪    知道边界在哪
  1. 先看端点速查表：了解有哪些 API 可以调用，以及鉴权怎么配
  2. 再看数据与模型：了解数据存储结构，方便备份、迁移和直接查库排障
  3. 最后看 z.ai 集成边界：如果你用到了 z.ai，这篇帮你避免把"没实现的"当成"能用的"
