On this page

高级功能 ​

掌握了基础的计划评审和代码评审后，本章节将带你解锁 Plannotator 的进阶能力：团队协作分享、笔记应用集成、远程开发支持，以及灵活的环境变量配置。

前置条件

学习本章节前，请确保你已完成：

  • 快速开始：了解 Plannotator 的基本概念
  • 计划评审基础：掌握基本的计划评审操作

本章内容 ​

课程说明适合场景
URL 分享通过 URL 分享计划和注释，实现无后端的团队协作需要与团队成员分享评审结果
Obsidian 集成将批准的计划自动保存到 Obsidian vault使用 Obsidian 管理知识库
Bear 集成通过 x-callback-url 将计划保存到 BearmacOS/iOS 用户使用 Bear 笔记
远程模式在 SSH、devcontainer、WSL 等远程环境中使用远程开发场景
环境变量配置了解所有可用的环境变量及其用途需要自定义 Plannotator 行为

按需学习

不必按顺序学完所有内容。如果你只需要分享功能，学完 URL 分享即可；如果你不使用远程开发，可以跳过远程模式。

下一步 ​

完成高级功能的学习后，你可以：

  • 查看 常见问题：解决使用中遇到的问题
  • 查看 API 参考：了解 Plannotator 的完整 API
  • 查看 数据模型：深入理解内部数据结构
