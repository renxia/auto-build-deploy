主题

plannotator
AI 计划与代码评审工具

为 Claude Code 和 OpenCode 提供交互式计划评审和代码评审工具，支持可视化注释、团队协作、URL 分享和第三方笔记应用集成。本地运行，数据安全，无需服务器即可分享评审内容。

快速开始 GitHub
为什么选择 Plannotator？

可视化评审 AI 生成的计划和代码，精确反馈，团队协作。

可视化计划评审

在浏览器中查看 AI 生成的计划，用删除、替换、插入、评论注释精确反馈。

代码审查可视化

Git diff side-by-side 或 unified 视图，支持行级注释和代码建议。

多平台支持

支持 Claude Code 和 OpenCode，无缝融入现有开发工作流。

URL 无后端分享

压缩到 URL hash 中，无需服务器即可分享评审内容给团队成员。

图像标注工具

附加图像并使用画笔、箭头、圆形工具进行标注，增强反馈表达。

第三方笔记集成

批准的计划自动保存到 Obsidian 或 Bear，包含 frontmatter 和标签。
