On this page

常见问题 ​

本章节收集了 OpenSkills 使用过程中的常见问题、故障排除方案和设计理念解释，帮助你快速定位和解决问题。

本章内容 ​

常见问题解答 ​

解答安装、配置、使用等方面的常见疑问。包括 OpenSkills 与 Claude Code 的区别、技能安装位置、AGENTS.md 同步问题、多项目共享技能等。

故障排除 ​

提供常见错误的诊断和修复方法。涵盖 Git clone 失败、找不到 SKILL.md、技能未找到、权限错误、更新跳过等问题的详细解决步骤。

为什么是 CLI？ ​

解释 OpenSkills 选择 CLI 而非 MCP 的设计理由。深入对比两者的定位差异，理解技能系统为什么更适合静态文件模式。
