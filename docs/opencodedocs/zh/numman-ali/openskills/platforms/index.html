On this page

平台功能 ​

本章节深入讲解 OpenSkills CLI 的核心命令和技能管理功能，帮助你从"会用"到"精通"。

前置条件 ​

开始前请确认

在学习本章节之前，请确保你已完成 快速开始 章节，特别是：

  • 安装 OpenSkills：已成功安装 OpenSkills CLI
  • 安装第一个技能：了解基本的技能安装流程

本章内容 ​

本章节包含 6 个主题，覆盖 OpenSkills 的所有核心功能：

命令详解 ​

完整介绍所有 7 个命令、参数和标志，提供命令速查表。适合需要快速查找命令用法的用户。

安装来源详解 ​

详细讲解三种安装来源：GitHub 仓库、本地路径、私有 Git 仓库。每种方式的适用场景、命令格式和注意事项。

全局安装 vs 项目本地安装 ​

解释 --global 和默认（项目本地）安装的区别，帮助你选择合适的安装位置，理解技能搜索优先级规则。

列出已安装技能 ​

教会你使用 list 命令查看已安装技能，理解 (project) 和 (global) 位置标签的含义。

更新技能 ​

指导你使用 update 命令刷新已安装技能，支持全部更新和指定技能更新，保持技能与源仓库同步。

删除技能 ​

介绍两种删除方式：交互式 manage 命令和脚本化 remove 命令，帮助你高效管理技能库。
