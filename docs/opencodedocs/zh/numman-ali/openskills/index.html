主题

openskills
AI 编码代理的通用技能加载器

让任何 AI 编码代理（Claude Code、Cursor、Windsurf、Aider 等）都能使用标准的 SKILL.md 格式技能。统一标准、渐进式加载、多代理支持。

快速开始 GitHub
为什么选择 OpenSkills？

统一的技能格式，让技能在任何代理中可用，告别平台锁定。

Claude Code 完全兼容

相同的提示格式、相同的 marketplace、相同的文件夹结构，无缝迁移。

多代理支持

支持 Claude Code、Cursor、Windsurf、Aider、Codex 等所有可读取 AGENTS.md 的代理。

渐进式加载

按需加载技能，保持代理上下文精简和聚焦，避免上下文污染。

灵活的安装来源

支持从 GitHub、本地路径、私有 git 仓库安装技能，适应各种使用场景。

Universal 模式

一套技能服务多个代理，避免与 Claude Code 冲突，统一技能管理。

纯本地运行

CLI 工具无需服务器，所有操作在本地文件系统完成，确保隐私安全。
