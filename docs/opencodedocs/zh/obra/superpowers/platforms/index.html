On this page

平台集成 ​

本章节帮助你了解 Superpowers 在三大主流 AI 编码平台上的安装和配置方法，包括 Claude Code、OpenCode 和 Codex 的集成细节。

开始前

本章节假设你已完成 Superpowers 安装。如果还没有安装，请先前往安装章节。

学习路径 ​

建议按以下顺序学习各个平台，先了解你正在使用的平台：

1. Claude Code 平台 ​

专为 Claude Code 设计的官方插件，提供钩子系统和无缝集成。

  • 插件安装方法
  • 技能自动加载机制
  • 会话钩子配置
  • 斜杠命令使用

2. OpenCode 平台 ​

基于插件的集成方案，支持技能发现和热加载。

  • 插件安装步骤
  • 技能目录配置
  • 技能自动发现
  • 开发模式与生产模式

3. Codex 平台 ​

通过 CLI 工具实现的集成，适合喜欢命令行的开发者。

  • CLI 工具安装
  • 技能加载命令
  • 技能管理操作
  • 配置文件说明

4. 平台对比与选择 ​

对比三大平台的功能差异、适用场景，帮你选择最合适的平台。

  • 功能对比表格
  • 适用场景分析
  • 性能对比
  • 迁移指南

前置条件 ​

  • 已完成 Superpowers 安装
  • 了解 如何使用技能 的基本概念
  • 熟悉至少一个 AI 编码平台的基础操作

下一步 ​

掌握平台集成后，建议学习 核心工作流 章节，了解如何在实际开发中应用 Superpowers 的各种技能。

选择你的平台

如果你只需要在特定平台上使用 Superpowers，可以直接跳转到对应的平台章节。大多数开发者从 Claude Code 开始，它是官方支持最完整的平台。
