opencode-supermemory
Cross-Session Persistent Memory

OpenCode plugin that gives coding agents persistent memory using Supermemory. Your agent remembers what you tell it - across sessions, across projects. Features automatic context injection, intelligent compaction, and privacy protection.

Get Started GitHub
Why opencode-supermemory?

Teach your Agent once, remember forever. No more repeating the same rules every session.

Cross-Project Memory

User coding preferences automatically apply across different projects. Teach once, remember forever.

Automatic Context Injection

Seamlessly loads user profiles and project memories at session start. Agent knows your habits from the first message.

Intelligent Compaction

Automatically summarizes and saves context when token usage exceeds 80%, preventing loss of critical information.

Natural Language Triggers

Say 'remember this' or 'save this' and the agent auto-saves to memory. No complex operations required.

Privacy Protection

Content in <private> tags is automatically redacted before upload. Sensitive data never leaves your machine.

Codebase Indexing

Run /supermemory-init to explore and memorize your codebase structure, patterns, and conventions.
