Moltbot
Your Private AI Assistant, Available Anywhere (formerly Clawdbot)

Run your personal AI assistant on your own devices, seamlessly connecting to WhatsApp, Telegram, Slack, and 12+ other channels. Features browser automation, multi-device collaboration, voice interaction, with all data staying local for complete privacy.

Quick Start GitHub
Why Choose Moltbot?

Powerful, Private, and Easy-to-Deploy Personal AI Assistant

🌐 Unified Channels

Manage 12+ messaging platforms like WhatsApp, Telegram, Slack with a single configuration

🔒 Local-First

All data and AI runtime stay on your devices, zero privacy leaks

🛠️ Powerful Toolchain

Browser automation, cron jobs, webhooks, and more extensibility

📱 Multi-Device

Connect macOS/iOS/Android nodes to execute local operations

🎤 Voice Interaction

Supports Voice Wake, Talk Mode, and multiple TTS engines

🧠 Memory System

Vectorized storage and semantic search for AI that remembers every conversation

🔐 Secure Isolation

Sandbox isolation, DM pairing protection, and tool permission control
