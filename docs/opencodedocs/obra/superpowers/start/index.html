Quick Start with Superpowers ​

Welcome to the Superpowers Quick Start Guide. This chapter helps you install and configure Superpowers from scratch, understand core concepts, and experience the powerful capabilities of AI coding agents within 5 minutes.

Learning Path ​

We recommend following this order to gradually master the basics of Superpowers:

1. What is Superpowers ​

Understand the core value, use cases, and working principles of Superpowers. Learn why AI agents need these skills.

2. Quick Start: Get Started in 5 Minutes ​

Quickly experience the core features of Superpowers through hands-on practice. Feel how it transforms the way AI coding agents work.

3. Installation Guide ​

Detailed installation steps covering Claude Code, OpenCode, and Codex platforms. Ensure you can use Superpowers in your own environment.

4. Verify Installation ​

Learn how to verify that Superpowers is correctly installed and loaded, and troubleshoot common installation issues.

5. How to Use Skills ​

Understand the core concepts of the skill system, skill priorities, and invocation rules. Build a solid foundation for subsequent learning.

Prerequisites ​

Before starting this chapter, please ensure:

  • You are familiar with basic command line operations
  • You have installed and configured at least one AI coding agent (Claude Code, OpenCode, or Codex)
  • You have basic programming experience

Next Steps ​

After completing this chapter, you can:

  • 📖 Learn detailed usage for specific platforms → Platforms & Integration
  • 🚀 Learn core development workflows → Core Workflows
