    基于不同编程语言的量化框架与其Github仓库¶

    本列表由LLMQuant社区整理, 个精心整理、极其优秀的量化（定量金融）相关库、包及资源列表, 只供学习交流使用, 版权归原作者所有。

    语言¶

    • Python - 最流行的量化金融语言，拥有丰富的数据分析、机器学习库和金融工具包
    • R - 统计分析的首选语言，在金融建模和风险分析方面有独特优势
    • Matlab - 在金融工程和数值计算领域广泛使用，尤其适合矩阵运算
    • Julia - 专为高性能数值计算设计的新兴语言，在量化金融中逐渐流行
    • Java - 企业级应用的标准选择，适合构建大规模交易系统
    • JavaScript - 主要用于金融数据可视化和Web端交易界面开发
    • Haskell - 函数式编程语言，在复杂金融产品建模方面具有优势
    • Scala - 结合了面向对象和函数式编程，适合构建高并发交易系统
    • Ruby - 以简洁著称，主要用于快速开发原型和小型交易系统
    • Elixir/Erlang - 在构建容错和分布式交易系统方面表现出色
    • Golang - 以高性能和并发特性著称，适合开发高频交易系统
    • CPP - 最适合开发低延迟交易系统的语言，广泛应用于高频交易
    • CSharp - 微软技术栈的首选，在Windows平台上开发量化交易系统的理想选择
    • Rust - 新一代系统编程语言，在保证性能的同时提供内存安全保证
    • Frameworks - 跨语言的量化金融框架，提供完整的交易和分析解决方案
    • 再现研究成果、培训与书籍 - 学习资源和经典研究的代码实现

    选择建议¶

    1. 入门学习
    2. Python：最佳入门选择，学习资源丰富，生态系统完善

    3. R：适合想专注于统计分析和风险建模的学习者

    4. 专业开发

    5. C++：追求极致性能的高频交易系统
    6. Java：企业级交易平台开发

    7. Golang：现代高并发交易系统

    8. 特定场景

    9. Julia：需要高性能数值计算
    10. JavaScript：开发Web交易界面

    11. Rust：需要兼顾性能和安全性

    12. 研究导向

    13. Python/R：数据分析和策略研究
    14. Matlab：金融工程研究
    15. Julia：算法优化研究

    技术栈趋势¶

    • Python生态持续主导量化金融领域
    • Julia在高性能计算领域崭露头角
    • Rust在系统级开发中逐渐替代C++
    • Go语言在微服务架构中应用增多

    Python¶

    数值库与数据结构¶

    • numpy - NumPy 是 Python 科学计算的基础包。
    • scipy - SciPy（发音类似"Sigh Pie"）是基于 Python 的开源数学、科学和工程生态系统。
    • pandas - pandas 是一个开源的、BSD 许可的库，为 Python 提供高性能、易于使用的数据结构和数据分析工具。
    • polars - Polars 是一个用于操作结构化数据的超快 DataFrame 库。
    • quantdsl - 面向金融和交易的定量分析领域专用语言。
    • statistics - Python 内置的基础统计计算库。
    • sympy - SymPy 是一个 Python 符号数学库。
    • pymc3 - Python 中的概率编程：基于 Theano 的贝叶斯建模和概率机器学习。
    • modelx - 将电子表格重新想象为与 pandas 互操作的基于公式的对象。
    • ArcticDB - 面向时间序列和报价数据的高性能数据存储。

    金融工具与定价¶

    • OpenBB Terminal - 人人可用的投资研究终端。
    • Fincept Terminal - 基于高级数据与 AI 的金融资产研究终端。
    • PyQL - QuantLib 的 Python 移植版本。
    • pyfin - Python 中的基础期权定价。存档项目
    • vollib - 计算期权价格、隐含波动率和希腊值的 Python 库。
    • QuantPy - Python 中的定量金融框架。
    • Finance-Python - Python 金融工具集。
    • ffn - 金融函数库。
    • pynance - 轻量级 Python 库，用于装配和分析金融数据。
    • tia - 集成与分析的工具包。
    • hasura/base-python-dash - Hasura 快速启动，用于部署 Dash 框架。Dash 基于 Flask、Plotly.js 和 React.js，非常适用于在纯 Python 中构建数据可视化应用。
    • hasura/base-python-bokeh - Hasura 快速启动，用 bokeh 库可视化数据。
    • pysabr - SABR 模型的 Python 实现。
    • FinancePy - 专注于定价和风险管理金融衍生品（包括固定收益、股票、外汇和信用衍生品）的 Python 库。
    • gs-quant - 高盛提供的定量金融 Python 工具箱。
    • willowtree - 对衍生品定价使用柳树格 lattice 的 Python 实现。
    • financial-engineering - 以 Python 进行金融工程项目的蒙特卡洛方法应用。
    • optlib - 一款期权定价的 Python 库。
    • tf-quant-finance - 基于 TensorFlow 的高性能量化金融库。
    • Q-Fin - 针对数学金融的 Python 库。
    • Quantsbin - 用于定价与绘制香草期权价格、希腊值等分析的工具集。
    • finoptions - 完整实现 R 包 fOptions 以及部分 fExoticOptions 的 Python 库，定价各类期权。
    • pypme - PME（Public Market Equivalent）的计算工具。
    • AbsBox - Python 库，用于模拟资产支持证券（ABS）和抵押贷款支持证券（MBS）等结构化产品的现金流。
    • Intrinsic-Value-Calculator - 基于贴现现金流（DCF）分析的股票内在价值计算器。
    • Kelly-Criterion - 使用 Kelly 公式进行投资组合头寸规模控制的 Python 实现。
    • rateslib - 用于定价债券、国债期货，以及诸如 IRS、跨货币和外汇互换等衍生品的固定收益库。
    • fypy - 用于研究和开发的香草及奇异期权定价库。关注在 Black-Scholes 之外的模型定价，以及市场数据的校准。

    指标¶

    • pandas_talib - 基于 Pandas 实现的技术分析指标。
    • finta - 常见金融技术分析指标的 Pandas 实现。
    • Tulipy - 金融技术分析指标库（为 tulipindicators 提供 Python 封装）。
    • lppls - 针对 Log-Periodic Power Law Singularity (LPPLS) 模型的 Python 拟合模块。
    • talipp - 针对 Python 的增量式技术分析指标库。
    • streaming_indicators - 用于在流数据上计算技术分析指标的 Python 库。

    交易与回测¶

    • skfolio - 基于 scikit-learn 的投资组合优化 Python 库，提供统一接口与兼容 sklearn 的工具来构建、调参和交叉验证投资组合模型。
    • Investing algorithm framework - 用于开发、回测和部署自动化交易算法的框架。
    • QSTrader - QSTrader 回测仿真引擎。
    • Blankly - 完整集成回测、模拟交易及实时部署。
    • TA-Lib - TA-Lib (http://ta-lib.org/) 的 Python 封装。
    • zipline - Pythonic 的算法交易库。
    • zipline-reloaded - Zipline 的衍生版本，一个 Pythonic 算法交易库。
    • QuantSoftware Toolkit - 基于 Python 的开源软件框架，用于支持投资组合构建和管理。
    • quantitative - 定量金融和回测库。
    • analyzer - 面向实时金融和回测交易策略的 Python 框架。
    • bt - 灵活的 Python 回测库。
    • backtrader - Python 交易策略回测库。
    • pythalesians - Python 库，用于回测交易策略、绘制图表、无缝下载市场数据、分析市场模式等。
    • pybacktest - 基于 Python / pandas 的向量化回测框架，使回测更简单。
    • pyalgotrade - Python 算法交易库。
    • basana - 基于 Python 异步与事件驱动的算法交易框架，专注于加密货币。
    • tradingWithPython - 一组用于量化交易的函数和类集合。
    • Pandas TA - 拥有 115+ 指标的简易 Python 3 Pandas 扩展，用于构建自定义策略。
    • ta - 使用 Pandas 的技术分析库 (Python)。
    • algobroker - 一个执行引擎，用于算法交易。
    • pysentosa - sentosa 交易系统的 Python API。
    • finmarketpy - 用于回测交易策略和分析金融市场的 Python 库。
    • binary-martingale - 自动二元期权交易的马丁格尔策略计算机程序。
    • fooltrader - 使用大数据技术统一分析整个市场的项目。
    • zvt - 使用 SQL、pandas 提供统一可扩展的方式来记录数据、计算因子、选股、回测、实时交易，并可实时清晰地可视化所有流程。
    • pylivetrader - 兼容 zipline 的实时交易库。
    • pipeline-live - 使用 IEX 数据进行实时交易的 zipline pipeline 功能。
    • zipline-extensions - Zipline 在 QuantRocket 环境下的扩展与适配。
    • moonshot - 基于 Pandas 的向量化回测与交易引擎，适配 QuantRocket。
    • PyPortfolioOpt - Python 中的投资组合优化，包括传统有效前沿法与高级方法。
    • Eiten - Tradytics 出品的开源工具包，包含各种投资组合算法（如最小方差组合、最大 Sharpe 比率组合、遗传算法组合等）。
    • riskparity.py - 使用 TensorFlow 2.0 的快速、可扩展风险平价投资组合设计。
    • mlfinlab - 实现 Marcos Lopez de Prado 所著《Advances in Financial Machine Learning》中的相关技术（特征工程、金融数据结构、元标签）。
